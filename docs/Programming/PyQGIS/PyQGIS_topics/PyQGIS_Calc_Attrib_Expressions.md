# ­ЪЌ║№ИЈ PyQGIS рЃњрЃћрЃЮрЃАрЃўрЃЋрЃарЃфрЃўрЃЌрЃў рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃћрЃЉрЃўрЃА рЃАрЃЎрЃарЃўрЃърЃбрЃћрЃЉрЃў

рЃћрЃА рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃў рЃерЃћрЃўрЃфрЃљрЃЋрЃА Python рЃАрЃЎрЃарЃўрЃърЃбрЃћрЃЉрЃА QGIS-рЃерЃў рЃњрЃћрЃЮрЃАрЃўрЃЋрЃарЃфрЃўрЃЌрЃў рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃџрЃљрЃЊ, рЃЏрЃљрЃЌ рЃерЃЮрЃарЃўрЃА:

- рЃ«рЃљрЃќрЃЮрЃЋрЃљрЃюрЃў рЃерЃарЃўрЃА рЃАрЃўрЃњрЃарЃФрЃўрЃА рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃљ  
- рЃърЃЮрЃџрЃўрЃњрЃЮрЃюрЃљрЃџрЃБрЃарЃў рЃерЃарЃўрЃА рЃцрЃљрЃарЃЌрЃЮрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃърЃћрЃарЃўрЃЏрЃћрЃбрЃарЃўрЃА рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃљ  
- рЃЏрЃЮрЃАрЃљрЃ«рЃџрЃћрЃЮрЃЉрЃўрЃА рЃАрЃўрЃЏрЃГрЃўрЃЊрЃарЃЮрЃЋрЃўрЃА рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃљ рЃфрЃ«рЃарЃўрЃџрЃерЃў  

рЃДрЃЋрЃћрЃџрЃљ рЃАрЃЎрЃарЃўрЃърЃбрЃў рЃўрЃДрЃћрЃюрЃћрЃЉрЃА QGIS-рЃўрЃА Python API-рЃА (**PyQGIS**) рЃЊрЃљ рЃўрЃЌрЃЋрЃљрЃџрЃўрЃАрЃгрЃўрЃюрЃћрЃЉрЃА рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃў рЃерЃарЃћрЃћрЃЉрЃўрЃА рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃљрЃА QGIS рЃърЃарЃЮрЃћрЃЦрЃбрЃерЃў.

---

## 1№ИЈРЃБ рЃ«рЃљрЃќрЃЮрЃЋрЃљрЃюрЃў рЃерЃарЃўрЃА рЃАрЃўрЃњрЃарЃФрЃўрЃА рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃљ

рЃћрЃА рЃАрЃЎрЃарЃўрЃърЃбрЃў рЃўрЃЌрЃЋрЃџрЃўрЃА рЃ«рЃљрЃќрЃЮрЃЋрЃљрЃюрЃў рЃерЃарЃўрЃА (рЃЏрЃљрЃњ., *Rivers*) рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃћрЃЉрЃўрЃА рЃАрЃўрЃњрЃарЃФрЃћрЃА рЃЊрЃљ рЃўрЃюрЃљрЃ«рЃљрЃЋрЃА рЃерЃћрЃЊрЃћрЃњрЃА рЃљрЃ«рЃљрЃџ рЃЋрЃћрЃџрЃерЃў **`Len_M`**.

### ­ЪЊю рЃЎрЃЮрЃЊрЃў

```python
# рЃерЃарЃўрЃА рЃЏрЃўрЃдрЃћрЃЉрЃљ рЃАрЃљрЃ«рЃћрЃџрЃўрЃЌ 'Rivers'
layers = QgsProject.instance().mapLayersByName('Rivers')
layer = layers[0]

# рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃърЃарЃЮрЃЋрЃљрЃўрЃЊрЃћрЃарЃќрЃћ рЃгрЃЋрЃЊрЃЮрЃЏрЃљ
pv = layer.dataProvider()

# рЃљрЃ«рЃљрЃџрЃў рЃЋрЃћрЃџрЃўрЃА 'Len_M' рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ рЃАрЃўрЃњрЃарЃФрЃўрЃАрЃЌрЃЋрЃўрЃА (double рЃбрЃўрЃърЃў)
pv.addAttributes([QgsField('Len_M', QVariant.Double)])
layer.updateFields()

# рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃўрЃА рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃўрЃА рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљ
expr1 = QgsExpression('length($geometry)')

# рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃўрЃА рЃЎрЃЮрЃюрЃбрЃћрЃЦрЃАрЃбрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ
cntx = QgsExpressionContext()
cntx.appendScopes(QgsExpressionContextUtils.globalProjectLayerScopes(layer))

# рЃерЃарЃўрЃА рЃарЃћрЃЊрЃљрЃЦрЃбрЃўрЃарЃћрЃЉрЃљ рЃЊрЃљ рЃАрЃўрЃњрЃарЃФрЃўрЃА рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃљ
with edit(layer):
    for i in layer.getFeatures():
        cntx.setFeature(i)
        i['Len_M'] = expr1.evaluate(cntx)
        layer.updateFeature(i)
```

### ­ЪњА рЃњрЃљрЃюрЃЏрЃљрЃарЃбрЃћрЃЉрЃљ

- **рЃерЃарЃўрЃА рЃљрЃарЃЕрЃћрЃЋрЃљ:** рЃўрЃдрЃћрЃЉрЃА рЃерЃарЃћрЃА рЃАрЃљрЃ«рЃћрЃџрЃўрЃЌ `'Rivers'` QGIS рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃЊрЃљрЃю.  
- **рЃЋрЃћрЃџрЃў:** рЃЦрЃЏрЃюрЃўрЃА рЃљрЃ«рЃљрЃџ рЃЋрЃћрЃџрЃА `Len_M` рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃўрЃА рЃАрЃўрЃњрЃарЃФрЃўрЃА рЃерЃћрЃАрЃљрЃюрЃљрЃ«рЃљрЃЊ.  
- **рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃљ:** рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ `length($geometry)` рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃўрЃА рЃњрЃћрЃЮрЃЏрЃћрЃбрЃарЃўрЃўрЃА рЃАрЃўрЃњрЃарЃФрЃўрЃА рЃњрЃљрЃЏрЃЮрЃАрЃљрЃЌрЃЋрЃџрЃћрЃџрЃљрЃЊ.  
- **рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃљ:** рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџ рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃќрЃћ рЃ«рЃЊрЃћрЃЉрЃљ рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃўрЃА рЃерЃћрЃцрЃљрЃАрЃћрЃЉрЃљ рЃЊрЃљ `Len_M` рЃЋрЃћрЃџрЃўрЃА рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃљ.

## 2№ИЈРЃБ рЃърЃЮрЃџрЃўрЃњрЃЮрЃюрЃљрЃџрЃБрЃарЃў рЃерЃарЃўрЃА рЃцрЃљрЃарЃЌрЃЮрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃърЃћрЃарЃўрЃЏрЃћрЃбрЃарЃўрЃА рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃљ

рЃћрЃА рЃАрЃЎрЃарЃўрЃърЃбрЃў рЃўрЃЌрЃЋрЃџрЃўрЃА рЃърЃЮрЃџрЃўрЃњрЃЮрЃюрЃљрЃџрЃБрЃарЃў рЃерЃарЃўрЃА (рЃЏрЃљрЃњ., *nakveti*) рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃћрЃЉрЃўрЃА рЃцрЃљрЃарЃЌрЃЮрЃЉрЃАрЃљ рЃЊрЃљ рЃърЃћрЃарЃўрЃЏрЃћрЃбрЃарЃА рЃЊрЃљ рЃерЃћрЃЊрЃћрЃњрЃћрЃЉрЃА рЃўрЃюрЃљрЃ«рЃљрЃЋрЃА рЃљрЃ«рЃљрЃџ рЃЋрЃћрЃџрЃћрЃЉрЃерЃў **`Area_sq_m`** рЃЊрЃљ **`Perimeter`**.

### ­ЪЊю рЃЎрЃЮрЃЊрЃў

```python
# рЃерЃарЃўрЃА рЃЏрЃўрЃдрЃћрЃЉрЃљ рЃАрЃљрЃ«рЃћрЃџрЃўрЃЌ 'nakveti'
layers = QgsProject.instance().mapLayersByName('nakveti')
layer = layers[0]

# рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃърЃарЃЮрЃЋрЃљрЃўрЃЊрЃћрЃарЃќрЃћ рЃгрЃЋрЃЊрЃЮрЃЏрЃљ
pv = layer.dataProvider()

# рЃљрЃ«рЃљрЃџрЃў рЃЋрЃћрЃџрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ рЃцрЃљрЃарЃЌрЃЮрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃърЃћрЃарЃўрЃЏрЃћрЃбрЃарЃўрЃАрЃЌрЃЋрЃўрЃА
pv.addAttributes([QgsField('Area_sq_m', QVariant.Double)])
pv.addAttributes([QgsField('Perimeter', QVariant.Double)])
layer.updateFields()

# рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃўрЃА рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљ
expr1 = QgsExpression('area($geometry)')
expr2 = QgsExpression('perimeter($geometry)')

# рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃўрЃА рЃЎрЃЮрЃюрЃбрЃћрЃЦрЃАрЃбрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ
cntx = QgsExpressionContext()
cntx.appendScopes(QgsExpressionContextUtils.globalProjectLayerScopes(layer))

# рЃерЃарЃўрЃА рЃарЃћрЃЊрЃљрЃЦрЃбрЃўрЃарЃћрЃЉрЃљ рЃЊрЃљ рЃцрЃљрЃарЃЌрЃЮрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃърЃћрЃарЃўрЃЏрЃћрЃбрЃарЃўрЃА рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃљ
with edit(layer):
    for i in layer.getFeatures():
        cntx.setFeature(i)
        i['Area_sq_m'] = expr1.evaluate(cntx)
        i['Perimeter'] = expr2.evaluate(cntx)
        layer.updateFeature(i)
```

### ­ЪњА рЃњрЃљрЃюрЃЏрЃљрЃарЃбрЃћрЃЉрЃљ

- **рЃерЃарЃўрЃА рЃљрЃарЃЕрЃћрЃЋрЃљ:** рЃўрЃдрЃћрЃЉрЃА рЃерЃарЃћрЃА рЃАрЃљрЃ«рЃћрЃџрЃўрЃЌ `'nakveti'`.  
- **рЃЋрЃћрЃџрЃў:** рЃЦрЃЏрЃюрЃўрЃА рЃЮрЃа рЃљрЃ«рЃљрЃџ рЃЋрЃћрЃџрЃА: `Area_sq_m` рЃцрЃљрЃарЃЌрЃЮрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃЊрЃљ `Perimeter` рЃърЃћрЃарЃўрЃЏрЃћрЃбрЃарЃўрЃАрЃЌрЃЋрЃўрЃА.  
- **рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃћрЃЉрЃў:**  
  - `area($geometry)` Рђћ рЃўрЃЌрЃЋрЃџрЃўрЃА рЃцрЃљрЃарЃЌрЃЮрЃЉрЃА  
  - `perimeter($geometry)` Рђћ рЃўрЃЌрЃЋрЃџрЃўрЃА рЃърЃћрЃарЃўрЃЏрЃћрЃбрЃарЃА  
- **рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃљ:** рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџ рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃќрЃћ рЃ«рЃЊрЃћрЃЉрЃљ рЃЮрЃарЃўрЃЋрЃћ рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃўрЃА рЃерЃћрЃцрЃљрЃАрЃћрЃЉрЃљ рЃЊрЃљ рЃЋрЃћрЃџрЃћрЃЉрЃўрЃА рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃљ.

---

## 3№ИЈРЃБ рЃЏрЃЮрЃАрЃљрЃ«рЃџрЃћрЃЮрЃЉрЃўрЃА рЃАрЃўрЃЏрЃГрЃўрЃЊрЃарЃЮрЃЋрЃўрЃА рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃљ рЃфрЃ«рЃарЃўрЃџрЃерЃў

рЃћрЃА рЃАрЃЎрЃарЃўрЃърЃбрЃў рЃўрЃЌрЃЋрЃџрЃўрЃА рЃЏрЃЮрЃАрЃљрЃ«рЃџрЃћрЃЮрЃЉрЃўрЃА рЃАрЃўрЃЏрЃГрЃўрЃЊрЃарЃЮрЃЋрЃћрЃА рЃерЃарЃћрЃерЃў (рЃЏрЃљрЃњ., *Municipalitys*) рЃцрЃЮрЃарЃЏрЃБрЃџрЃўрЃЌ:

\[
D = \frac{P}{A}
\]

рЃАрЃљрЃЊрЃљрЃф:

- **D** Рђћ рЃЏрЃЮрЃАрЃљрЃ«рЃџрЃћрЃЮрЃЉрЃўрЃА рЃАрЃўрЃЏрЃГрЃўрЃЊрЃарЃЮрЃЋрЃћ  
- **P** Рђћ рЃЏрЃЮрЃАрЃљрЃ«рЃџрЃћрЃЮрЃЉрЃўрЃА рЃарЃљрЃЮрЃЊрЃћрЃюрЃЮрЃЉрЃљ (рЃЋрЃћрЃџрЃў `Mosaxl_200`)
