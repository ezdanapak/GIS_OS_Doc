### рЃгрЃћрЃарЃбрЃўрЃџрЃЮрЃЋрЃљрЃюрЃў рЃерЃарЃћ

- рЃњрЃћрЃЮрЃЏрЃћрЃбрЃарЃўрЃљ рЃЊрЃљрЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃћрЃЉрЃБрЃџрЃўрЃљ рЃљрЃЦрЃЋрЃћ рЃарЃљрЃЊрЃњрЃљрЃю рЃАрЃљрЃГрЃўрЃарЃЮрЃљ writer - рЃерЃў рЃЏрЃўрЃАрЃў рЃњрЃљрЃЌрЃЋрЃљрЃџрЃўрЃАрЃгрЃўрЃюрЃћрЃЉрЃљ, QgsWkbTypes.Point рЃўрЃА рЃюрЃљрЃфрЃЋрЃџрЃљрЃЊ рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ Line, Polygon, Unknown, Null - рЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ.

## РюЁ рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃњрЃћрЃЮрЃЏрЃћрЃбрЃарЃўрЃБрЃџрЃў рЃбрЃўрЃърЃћрЃЉрЃў PyQGIS-рЃерЃў:

- рЃљрЃарЃњрЃБрЃЏрЃћрЃюрЃбрЃћрЃЉрЃерЃў рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃњрЃљрЃЊрЃљрЃЋрЃфрЃћрЃЌ рЃБрЃюрЃўрЃЎрЃЮрЃЊрЃўрЃарЃћрЃЉрЃљ, рЃърЃарЃЮрЃћрЃЦрЃфрЃўрЃљ, рЃЊрЃарЃљрЃўрЃЋрЃћрЃарЃў рЃЊрЃљ рЃљ.рЃе
- рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ рЃЊрЃљрЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃА рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃБрЃџрЃўрЃљ рЃарЃЮрЃњрЃЮрЃарЃф writer рЃЎрЃЮрЃЊрЃерЃў, рЃарЃЮрЃЏрЃћрЃџрЃАрЃљрЃф рЃерЃћрЃБрЃФрЃџрЃўрЃљ рЃЊрЃљрЃљрЃЏрЃљрЃбрЃЮрЃА рЃЊрЃљ рЃЕрЃљрЃгрЃћрЃарЃЮрЃА рЃљрЃ«рЃљрЃџрЃў рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃћрЃЉрЃў рЃерЃарЃћрЃерЃў.

XY = [(356671.0049, 4679923.0988),

## Рё╣№ИЈ рЃњрЃљрЃюрЃЏрЃљрЃарЃбрЃћрЃЉрЃћрЃЉрЃў РђЊ PyQGIS рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃћрЃЉрЃў

### ­Ъћ╣ `QgsField`

рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃерЃарЃўрЃА рЃЋрЃћрЃџрЃўрЃА (рЃљрЃбрЃарЃўрЃЉрЃБрЃбрЃўрЃА рЃАрЃЋрЃћрЃбрЃўрЃА) рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃўрЃАрЃЌрЃЋрЃўрЃА. рЃЏрЃЮрЃўрЃфрЃљрЃЋрЃА рЃЋрЃћрЃџрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃАрЃљ рЃЊрЃљ рЃбрЃўрЃърЃА.

**рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў**:

```python
QgsField("name", QVariant.String)
# рЃЦрЃЏрЃюрЃўрЃА "name" рЃАрЃљрЃ«рЃћрЃџрЃўрЃА рЃЏрЃЦрЃЮрЃюрЃћ рЃбрЃћрЃЦрЃАрЃбрЃБрЃа рЃЋрЃћрЃџрЃА
```

### ­Ъћ╣ `QgsFields`

рЃгрЃљрЃарЃЏрЃЮрЃљрЃЊрЃњрЃћрЃюрЃА рЃЋрЃћрЃџрЃћрЃЉрЃўрЃА рЃЎрЃЮрЃџрЃћрЃЦрЃфрЃўрЃљрЃА (рЃљрЃюрЃБ рЃарЃљрЃЏрЃЊрЃћрЃюрЃўрЃЏрЃћ `QgsField` рЃћрЃарЃЌрЃљрЃЊ). рЃАрЃљрЃГрЃўрЃарЃЮрЃљ, рЃарЃЮрЃфрЃљ рЃБрЃюрЃЊрЃљ рЃњрЃљрЃЊрЃЏрЃЮрЃљрЃгрЃЮрЃЊрЃЮ рЃДрЃЋрЃћрЃџрЃљ рЃЋрЃћрЃџрЃў рЃћрЃарЃЌрЃљрЃЊ, рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ рЃерЃарЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃўрЃАрЃљрЃА.

**рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў**:

```python
fields = QgsFields()
fields.append(QgsField("name", QVariant.String))
```

---

### ­Ъћ╣ `QVariant`

Qt-рЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃбрЃўрЃърЃћрЃЉрЃўрЃА рЃЎрЃџрЃљрЃАрЃў, рЃарЃЮрЃЏрЃћрЃџрЃАрЃљрЃф QGIS рЃўрЃДрЃћрЃюрЃћрЃЉрЃА рЃљрЃбрЃарЃўрЃЉрЃБрЃбрЃўрЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃўрЃА рЃбрЃўрЃърЃћрЃЉрЃўрЃА рЃњрЃљрЃюрЃАрЃљрЃАрЃљрЃќрЃдрЃЋрЃарЃљрЃЊ. (рЃЏрЃљрЃњ. `String`, `Int`, `Double` рЃЊрЃљ рЃАрЃ«рЃЋ.)

**рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃў**:

* `QVariant.String` РђЊ рЃбрЃћрЃЦрЃАрЃбрЃБрЃарЃў рЃЋрЃћрЃџрЃў
* `QVariant.Int` РђЊ рЃЏрЃЌрЃћрЃџрЃў рЃарЃўрЃфрЃ«рЃЋрЃў
* `QVariant.Double` РђЊ рЃљрЃЌрЃгрЃўрЃџрЃљрЃЊрЃў рЃарЃўрЃфрЃ«рЃЋрЃў

### ­Ъћ╣ `QgsFeature`

рЃгрЃљрЃарЃЏрЃЮрЃљрЃЊрЃњрЃћрЃюрЃА рЃћрЃарЃЌ рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃА (рЃгрЃћрЃарЃбрЃўрЃџрЃА, рЃ«рЃљрЃќрЃА рЃљрЃю рЃърЃЮрЃџрЃўрЃњрЃЮрЃюрЃА) рЃЌрЃљрЃЋрЃўрЃАрЃў рЃњрЃћрЃЮрЃЏрЃћрЃбрЃарЃўрЃўрЃЌрЃљ рЃЊрЃљ рЃљрЃбрЃарЃўрЃЉрЃБрЃбрЃћрЃЉрЃўрЃЌ.

**рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў**:

```python
feature = QgsFeature()
feature.setGeometry(QgsGeometry.fromPointXY(QgsPointXY(44.8, 41.7)))
feature.setAttributes(["Tbilisi"])
```

### ­Ъћ╣ `iface`

`iface` (Interface) рЃљрЃарЃўрЃА QGIS-рЃўрЃА рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ QGIS-рЃўрЃА рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃЌрЃљрЃю (Layers Panel, Map Canvas рЃЊрЃљ рЃАрЃ«рЃЋ.) рЃБрЃарЃЌрЃўрЃћрЃарЃЌрЃЮрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.

**рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў**:

```python
iface.addVectorLayer("/path/to/file.shp", "My Layer", "ogr")
# рЃљрЃЏрЃљрЃбрЃћрЃЉрЃА рЃцрЃћрЃюрЃљрЃА рЃърЃўрЃарЃЊрЃљрЃърЃўрЃа QGIS-рЃўрЃА рЃърЃарЃЮрЃћрЃЦрЃбрЃерЃў
```

Рџа№ИЈ `iface` рЃ«рЃћрЃџрЃЏрЃўрЃАрЃљрЃгрЃЋрЃЊрЃЮрЃЏрЃўрЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ QGIS-рЃўрЃА **Python Console**-рЃерЃў. рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃБрЃџ Python рЃАрЃЎрЃарЃўрЃърЃбрЃерЃў рЃўрЃА рЃљрЃа рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА.

---

### ­Ъћ╣ рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌрЃў рЃАрЃљрЃАрЃљрЃарЃњрЃћрЃЉрЃџрЃЮ рЃЎрЃџрЃљрЃАрЃћрЃЉрЃў

| `QgsCoordinateReferenceSystem` | рЃАрЃљрЃЎрЃЮрЃЮрЃарЃЊрЃўрЃюрЃљрЃбрЃЮ рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљ EPSG рЃЎрЃЮрЃЊрЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ           |
