# ­ЪїЇ OOP рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃў рЃњрЃћрЃЮрЃњрЃарЃљрЃцрЃўрЃБрЃџ рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃћрЃЉрЃќрЃћ рЃЊрЃљ рЃЏрЃљрЃЌрЃў рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ PyQGIS-рЃерЃў

## рЃерЃћрЃАрЃљрЃЋрЃљрЃџрЃў
Object-Oriented Programming (OOP) Рђћ рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃќрЃћ рЃЮрЃарЃўрЃћрЃюрЃбрЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃљ Рђћ рЃљрЃарЃўрЃА рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃўрЃА рЃърЃљрЃарЃљрЃЊрЃўрЃњрЃЏрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃЋрЃћрЃ«рЃЏрЃљрЃарЃћрЃЉрЃљ рЃЎрЃЮрЃЊрЃўрЃА рЃЮрЃарЃњрЃљрЃюрЃўрЃќрЃћрЃЉрЃљрЃАрЃљ рЃЊрЃљ рЃарЃћрЃљрЃџрЃБрЃа рЃАрЃљрЃЏрЃДрЃљрЃарЃЮрЃАрЃЌрЃљрЃю рЃЏрЃўрЃљрЃ«рЃџрЃЮрЃћрЃЉрЃБрЃџ рЃЏрЃЮрЃЊрЃћрЃџрЃўрЃарЃћрЃЉрЃљрЃерЃў.  
GIS-рЃерЃў (Geographic Information Systems) рЃћрЃА рЃњрЃљрЃюрЃАрЃљрЃЎрЃБрЃЌрЃарЃћрЃЉрЃўрЃЌ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃўрЃљ, рЃарЃљрЃЊрЃњрЃљрЃю рЃњрЃћрЃЮрЃњрЃарЃљрЃцрЃўрЃБрЃџрЃў рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃћрЃЉрЃў рЃЉрЃБрЃюрЃћрЃЉрЃарЃўрЃЋрЃљрЃЊ рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃљрЃдрЃЋрЃгрЃћрЃарЃЮрЃЌ рЃарЃЮрЃњрЃЮрЃарЃф **рЃЎрЃџрЃљрЃАрЃћрЃЉрЃў** рЃЊрЃљ **рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃћрЃЉрЃў**.

---

## ­Ъј» OOP-рЃўрЃА рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃърЃарЃўрЃюрЃфрЃўрЃърЃћрЃЉрЃў

- **рЃЎрЃџрЃљрЃАрЃў (Class)** Рђћ рЃерЃљрЃЉрЃџрЃЮрЃюрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃдрЃгрЃћрЃарЃА рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃА (рЃЏрЃљрЃњ., РђъPointРђю, РђъPolygonРђю).  
- **рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃў (Object)** Рђћ рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃў рЃўрЃюрЃАрЃбрЃљрЃюрЃфрЃўрЃљ рЃЎрЃџрЃљрЃАрЃўрЃА (рЃЏрЃљрЃњ., `Tbilisi = Point(41.7, 44.8)`).  
- **рЃљрЃбрЃарЃўрЃЉрЃБрЃбрЃћрЃЉрЃў (Attributes)** Рђћ рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃўрЃА рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃџрЃћрЃЉрЃў (рЃЏрЃљрЃњ., рЃЎрЃЮрЃЮрЃарЃЊрЃўрЃюрЃљрЃбрЃћрЃЉрЃў, рЃцрЃћрЃарЃў, рЃАрЃљрЃ«рЃћрЃџрЃў).  
- **рЃЏрЃћрЃЌрЃЮрЃЊрЃћрЃЉрЃў (Methods)** Рђћ рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃћрЃЉрЃў, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃАрЃљрЃф рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃў рЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃА (рЃЏрЃљрЃњ., РђърЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃљрЃџрЃћ рЃАрЃўрЃњрЃарЃФрЃћРђю, РђърЃЊрЃљрЃ«рЃљрЃбрЃћ рЃарЃБрЃЎрЃљрЃќрЃћРђю).  
- **рЃЏрЃћрЃЏрЃЎрЃЋрЃўрЃЊрЃарЃћрЃЮрЃЉрЃљ (Inheritance)** Рђћ рЃЎрЃџрЃљрЃАрЃћрЃЉрЃўрЃА рЃњрЃљрЃЊрЃљрЃЏрЃћрЃЏрЃЎрЃЋрЃўрЃЊрЃарЃћрЃЉрЃћрЃџрЃў рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ (рЃЏрЃљрЃњ., `Geometry` Рєњ `Point`, `Line`, `Polygon`).  
- **рЃърЃЮрЃџрЃўрЃЏрЃЮрЃарЃцрЃўрЃќрЃЏрЃў (Polymorphism)** Рђћ рЃћрЃарЃЌрЃў рЃЊрЃљ рЃўрЃњрЃўрЃЋрЃћ рЃЏрЃћрЃЌрЃЮрЃЊрЃўрЃА рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃарЃћрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃљ рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃЎрЃџрЃљрЃАрЃўрЃАрЃЌрЃЋрЃўрЃА.

---

## ­ЪЊї рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў: рЃгрЃћрЃарЃбрЃўрЃџрЃў рЃарЃЮрЃњрЃЮрЃарЃф рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃў

```python
class Point:
    def __init__(self, x, y, name=""):
        self.x = x
        self.y = y
        self.name = name

    def __repr__(self):
        return f"Point({self.name}: {self.x}, {self.y})"
    
    def distance_to(self, other):
        return ((self.x - other.x)**2 + (self.y - other.y)**2)**0.5

# рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃћрЃЉрЃў
tbilisi = Point(44.8, 41.7, "Tbilisi")
batumi = Point(41.6, 41.65, "Batumi")

print(tbilisi)
print("Distance:", tbilisi.distance_to(batumi))
```

­ЪЊЇ рЃерЃћрЃЊрЃћрЃњрЃў: рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃћрЃЉрЃўрЃА рЃљрЃдрЃгрЃћрЃарЃљ рЃЊрЃљ рЃЏрЃљрЃЌ рЃерЃЮрЃарЃўрЃА рЃЏрЃљрЃюрЃФрЃўрЃџрЃўрЃА рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃљ.

---

## ­ЪЊї рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў: рЃ«рЃљрЃќрЃў рЃЊрЃљ рЃАрЃўрЃњрЃарЃФрЃћ

```python
class Line:
    def __init__(self, points):
        self.points = points
    
    def length(self):
        length = 0
        for i in range(len(self.points)-1):
            length += self.points[i].distance_to(self.points[i+1])
        return length

# рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃћрЃЉрЃў
p1 = Point(0, 0)
p2 = Point(3, 4)
p3 = Point(6, 8)

line = Line([p1, p2, p3])
print("Line length:", line.length())
```

­ЪЊЈ рЃерЃћрЃЊрЃћрЃњрЃў: рЃ«рЃљрЃќрЃўрЃА рЃАрЃўрЃњрЃарЃФрЃћ = рЃгрЃћрЃарЃбрЃўрЃџрЃћрЃЉрЃА рЃерЃЮрЃарЃўрЃА рЃЊрЃўрЃАрЃбрЃљрЃюрЃфрЃўрЃћрЃЉрЃўрЃА рЃ»рЃљрЃЏрЃў.

---

## ­ЪЊї рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў: рЃърЃЮрЃџрЃўрЃњрЃЮрЃюрЃў рЃЊрЃљ рЃцрЃљрЃарЃЌрЃЮрЃЉрЃў

```python
class Polygon:
    def __init__(self, points):
        self.points = points
    
    def area(self):
        # Shoelace formula
        n = len(self.points)
        area = 0
        for i in range(n):
            j = (i + 1) % n
            area += self.points[i].x * self.points[j].y
            area -= self.points[j].x * self.points[i].y
        return abs(area) / 2

# рЃАрЃљрЃЏрЃЎрЃБрЃЌрЃ«рЃћрЃЊрЃў
triangle = Polygon([Point(0,0), Point(4,0), Point(4,3)])
print("Area:", triangle.area())
```

­ЪЊљ рЃерЃћрЃЊрЃћрЃњрЃў: рЃърЃЮрЃџрЃўрЃњрЃЮрЃюрЃўрЃА рЃцрЃљрЃарЃЌрЃЮрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃљ.

---

## ­Ъїљ PyQGIS-рЃерЃў рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ

PyQGIS рЃљрЃарЃўрЃА Python API QGIS-рЃўрЃАрЃЌрЃЋрЃўрЃА. рЃљрЃЦ рЃЎрЃџрЃљрЃАрЃћрЃЉрЃў рЃБрЃЎрЃЋрЃћ рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА:  
- `QgsPointXY` Рђћ рЃгрЃћрЃарЃбрЃўрЃџрЃў  
- `QgsGeometry` Рђћ рЃњрЃћрЃЮрЃЏрЃћрЃбрЃарЃўрЃљ  
- `QgsFeature` Рђћ рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃў рЃљрЃбрЃарЃўрЃЉрЃБрЃбрЃћрЃЉрЃўрЃЌ  
- `QgsVectorLayer` Рђћ рЃцрЃћрЃюрЃљ

### ­ЪЊї PyQGIS рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў: рЃгрЃћрЃарЃбрЃўрЃџрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ рЃЊрЃљ рЃарЃБрЃЎрЃљрЃќрЃћ рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ

```python
from qgis.core import QgsProject, QgsPointXY, QgsGeometry, QgsFeature, QgsVectorLayer

# рЃцрЃћрЃюрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ
layer = QgsVectorLayer("Point?crs=EPSG:4326", "Cities", "memory")
prov = layer.dataProvider()

# рЃљрЃбрЃарЃўрЃЉрЃБрЃбрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ
layer.startEditing()
prov.addAttributes([QgsField("Name", QVariant.String)])
layer.updateFields()

# рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ
feat = QgsFeature()
feat.setGeometry(QgsGeometry.fromPointXY(QgsPointXY(44.8, 41.7)))
feat.setAttributes(["Tbilisi"])
prov.addFeatures([feat])
layer.commitChanges()

# рЃцрЃћрЃюрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ рЃърЃарЃЮрЃћрЃЦрЃбрЃерЃў
QgsProject.instance().addMapLayer(layer)
```

­ЪЊЇ рЃерЃћрЃЊрЃћрЃњрЃў: рЃарЃБрЃЎрЃљрЃќрЃћ рЃЊрЃљрЃћрЃЏрЃљрЃбрЃћрЃЉрЃљ рЃгрЃћрЃарЃбрЃўрЃџрЃў (рЃЌрЃЉрЃўрЃџрЃўрЃАрЃў).

---

## ­ЪЊї PyQGIS рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў: рЃ«рЃљрЃќрЃўрЃА рЃАрЃўрЃњрЃарЃФрЃўрЃА рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃљ

```python
from qgis.core import QgsGeometry, QgsPointXY

line = QgsGeometry.fromPolylineXY([
    QgsPointXY(0,0), QgsPointXY(3,4), QgsPointXY(6,8)
])

print("Line length:", line.length())
```

­ЪЊЈ рЃерЃћрЃЊрЃћрЃњрЃў: рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃўрЃА рЃАрЃўрЃњрЃарЃФрЃћрЃА CRS-рЃўрЃА рЃћрЃарЃЌрЃћрЃБрЃџрЃћрЃЉрЃерЃў.

---

## ­ЪЊї PyQGIS рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў: рЃърЃЮрЃџрЃўрЃњрЃЮрЃюрЃўрЃА рЃцрЃљрЃарЃЌрЃЮрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃљ

```python
from qgis.core import QgsGeometry, QgsPointXY

polygon = QgsGeometry.fromPolygonXY([[
    QgsPointXY(0,0),
    QgsPointXY(4,0),
    QgsPointXY(4,3),
    QgsPointXY(0,0)
]])

print("Polygon area:", polygon.area())
```

­ЪЊљ рЃерЃћрЃЊрЃћрЃњрЃў: рЃърЃЮрЃџрЃўрЃњрЃЮрЃюрЃўрЃА рЃцрЃљрЃарЃЌрЃЮрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃљ.

---

## ­Ъџђ рЃАрЃљрЃБрЃЎрЃћрЃЌрЃћрЃАрЃЮ рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃћрЃЉрЃў GIS-рЃерЃў OOP-рЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃАрЃљрЃА

1. **рЃЎрЃџрЃљрЃАрЃћрЃЉрЃўрЃЌ рЃљрЃдрЃгрЃћрЃарЃћрЃЌ рЃњрЃћрЃЮрЃњрЃарЃљрЃцрЃўрЃБрЃџрЃў рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃћрЃЉрЃў** Рђћ рЃљрЃАрЃћ рЃЎрЃЮрЃЊрЃў рЃ«рЃЊрЃћрЃЉрЃљ рЃњрЃљрЃАрЃљрЃњрЃћрЃЉрЃў рЃЊрЃљ рЃ«рЃћрЃџрЃљрЃ«рЃљрЃџрЃў рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљрЃЊрЃў.  
2. **рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ рЃЏрЃћрЃЏрЃЎрЃЋрЃўрЃЊрЃарЃћрЃЮрЃЉрЃљ** Рђћ рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃЏрЃерЃЮрЃЉрЃћрЃџрЃў `Geometry` рЃЎрЃџрЃљрЃАрЃў рЃЊрЃљ рЃерЃЋрЃўрЃџрЃћрЃЉрЃў (`Point`, `Line`, `Polygon`).  
3. **рЃерЃћрЃљрЃћрЃарЃЌрЃћрЃЌ Python рЃЎрЃџрЃљрЃАрЃћрЃЉрЃў PyQGIS API-рЃЌрЃљрЃю** Рђћ рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃўрЃА рЃЏрЃАрЃњрЃљрЃЋрЃАрЃљрЃЊ, рЃЌрЃЦрЃЋрЃћрЃюрЃА рЃЎрЃџрЃљрЃАрЃћрЃЉрЃА рЃерЃћрЃБрЃФрЃџрЃўрЃљ рЃ░рЃЦрЃЮрЃюрЃЊрЃћрЃЌ рЃЏрЃћрЃЌрЃЮрЃЊрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА `QgsGeometry` рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃА.  
4. **рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ рЃљрЃбрЃарЃўрЃЉрЃБрЃбрЃћрЃЉрЃў рЃЎрЃџрЃљрЃАрЃћрЃЉрЃерЃў** Рђћ GIS рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃћрЃЉрЃА рЃ«рЃерЃўрЃарЃљрЃЊ рЃљрЃЦрЃЋрЃЌ рЃАрЃљрЃ«рЃћрЃџрЃў, рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃљ, рЃцрЃћрЃарЃў.  
5. **рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљ** Рђћ рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃФрЃљрЃџрЃўрЃљрЃю рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃўрЃљ рЃљрЃ«рЃАрЃюрЃўрЃџрЃў рЃўрЃДрЃЮрЃА рЃарЃљрЃА рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃЎрЃџрЃљрЃАрЃў рЃЊрЃљ рЃЏрЃћрЃЌрЃЮрЃЊрЃў.

## рЃЊрЃљрЃАрЃЎрЃЋрЃюрЃљ

OOP рЃљрЃарЃўрЃА рЃФрЃџрЃўрЃћрЃарЃў рЃЏрЃўрЃЊрЃњрЃЮрЃЏрЃљ GIS рЃърЃарЃЮрЃћрЃЦрЃбрЃћрЃЉрЃерЃў.  
- рЃњрЃћрЃЮрЃњрЃарЃљрЃцрЃўрЃБрЃџрЃў рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃћрЃЉрЃў рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ рЃљрЃдрЃгрЃћрЃарЃўрЃџрЃў рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃўрЃДрЃЮрЃА рЃЎрЃџрЃљрЃАрЃћрЃЉрЃўрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃўрЃЌ.  
- PyQGIS рЃБрЃЎрЃЋрЃћ рЃњрЃЋрЃЌрЃљрЃЋрЃљрЃќрЃЮрЃЉрЃА рЃЏрЃарЃљрЃЋрЃљрЃџрЃў рЃЎрЃџрЃљрЃАрЃўрЃА рЃарЃћрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃљрЃА.  
- рЃАрЃгрЃЮрЃарЃў рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ, рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃерЃћрЃЦрЃЏрЃюрЃљрЃЌ рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃў, рЃњрЃљрЃАрЃљрЃњрЃћрЃЉрЃў рЃЊрЃљ рЃ«рЃћрЃџрЃљрЃ«рЃљрЃџрЃў рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљрЃЊрЃў рЃЎрЃЮрЃЊрЃў.

РъА№ИЈ **рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃюрЃљрЃЉрЃўрЃ»рЃў рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА:** рЃАрЃфрЃљрЃЊрЃћрЃЌ рЃерЃћрЃЦрЃЏрЃюрЃљрЃЌ рЃАрЃљрЃЎрЃБрЃЌрЃљрЃарЃў `Geometry` рЃЎрЃџрЃљрЃАрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА PyQGIS-рЃЌрЃљрЃю рЃћрЃарЃЌрЃљрЃЊ рЃЊрЃљ рЃљрЃЦрЃЋрЃА рЃЏрЃћрЃЌрЃЮрЃЊрЃћрЃЉрЃў рЃцрЃљрЃарЃЌрЃЮрЃЉрЃўрЃА, рЃАрЃўрЃњрЃарЃФрЃўрЃА рЃЊрЃљ рЃЏрЃљрЃюрЃФрЃўрЃџрЃўрЃА рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃўрЃАрЃЌрЃЋрЃўрЃА.
