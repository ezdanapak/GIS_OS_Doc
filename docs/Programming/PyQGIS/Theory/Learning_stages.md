# ­ЪљЇ PyQGIS рЃАрЃгрЃљрЃЋрЃџрЃћрЃЉрЃўрЃА рЃАрЃљрЃцрЃћрЃ«рЃБрЃарЃћрЃЉрЃў рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА  

## 1№ИЈРЃБ рЃАрЃљрЃцрЃБрЃФрЃЋрЃџрЃћрЃЉрЃў  
- QGIS рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃњрЃљрЃфрЃюрЃЮрЃЉрЃљ (Layers, CRS, Project Structure)  
- Python Console QGIS-рЃерЃў  
- PyQGIS API-рЃўрЃА рЃЏрЃўрЃЏрЃЮрЃ«рЃўрЃџрЃЋрЃљ  
- рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃАрЃЎрЃарЃўрЃърЃбрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ  

­ЪЉЅ рЃерЃћрЃЊрЃћрЃњрЃў: рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃЏрЃљ рЃўрЃфрЃўрЃА рЃАрЃљрЃЊ рЃЊрЃљ рЃарЃЮрЃњрЃЮрЃа рЃгрЃћрЃарЃЮрЃА PyQGIS рЃЎрЃЮрЃЊрЃў.  

---

## 2№ИЈРЃБ QGIS рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃћрЃЉрЃўрЃА рЃАрЃљрЃцрЃБрЃФрЃЋрЃџрЃћрЃЉрЃў  
- QgsProject РђЊ рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ  
- QgsVectorLayer / QgsRasterLayer РђЊ рЃцрЃћрЃюрЃћрЃЉрЃўрЃА рЃЕрЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃљ  
- QgsFeature РђЊ рЃљрЃбрЃарЃўрЃЉрЃБрЃбрЃћрЃЉрЃў рЃЊрЃљ рЃњрЃћрЃЮрЃЏрЃћрЃбрЃарЃўрЃљ  
- QgsGeometry РђЊ рЃгрЃћрЃарЃбрЃўрЃџрЃћрЃЉрЃў, рЃ«рЃљрЃќрЃћрЃЉрЃў, рЃърЃЮрЃџрЃўрЃњрЃЮрЃюрЃћрЃЉрЃў  

­ЪЉЅ рЃерЃћрЃЊрЃћрЃњрЃў: рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃў рЃЊрЃљрЃЏрЃЮрЃБрЃЎрЃўрЃЊрЃћрЃЉрЃџрЃљрЃЊ рЃбрЃЋрЃўрЃарЃЌрЃљрЃЋрЃА рЃцрЃћрЃюрЃћрЃЉрЃА рЃЊрЃљ рЃЎрЃўрЃЌрЃ«рЃБрЃџрЃЮрЃЉрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃА.  

---

## 3№ИЈРЃБ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљ рЃЊрЃљ рЃарЃћрЃЊрЃљрЃЦрЃбрЃўрЃарЃћрЃЉрЃљ  
- рЃцрЃћрЃюрЃўрЃЊрЃљрЃю рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃћрЃЉрЃўрЃА рЃљрЃЏрЃЮрЃдрЃћрЃЉрЃљ (`getFeatures()`)  
- рЃљрЃбрЃарЃўрЃЉрЃБрЃбрЃћрЃЉрЃўрЃА рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљ рЃЊрЃљ рЃерЃћрЃфрЃЋрЃџрЃљ  
- рЃњрЃћрЃЮрЃЏрЃћрЃбрЃарЃўрЃўрЃА рЃарЃћрЃЊрЃљрЃЦрЃбрЃўрЃарЃћрЃЉрЃљ (рЃљрЃ«рЃљрЃџрЃў рЃгрЃћрЃарЃбрЃўрЃџрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ, рЃ«рЃљрЃќрЃўрЃА рЃЊрЃљрЃ«рЃљрЃбрЃЋрЃљ)  
- рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃћрЃЦрЃАрЃърЃЮрЃарЃбрЃў (Shapefile, GeoJSON, GPKG)  

­ЪЉЅ рЃерЃћрЃЊрЃћрЃњрЃў: рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃА рЃерЃћрЃБрЃФрЃџрЃўрЃљ рЃерЃћрЃЦрЃЏрЃюрЃљрЃА рЃЊрЃљ рЃЕрЃљрЃљрЃАрЃгрЃЮрЃарЃЮрЃА рЃАрЃўрЃЋрЃарЃфрЃўрЃЌрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў.  

---

## 4№ИЈРЃБ рЃАрЃўрЃЋрЃарЃфрЃўрЃЌрЃў рЃљрЃюрЃљрЃџрЃўрЃќрЃў  
- рЃЉрЃБрЃцрЃћрЃарЃћрЃЉрЃў (Buffer)  
- рЃњрЃљрЃЊрЃљрЃцрЃљрЃарЃЋрЃљ (Intersection, Union, Difference)  
- рЃЏрЃљрЃюрЃФрЃўрЃџрЃўрЃАрЃљ рЃЊрЃљ рЃцрЃљрЃарЃЌрЃЮрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃљ  
- CRS рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃюрЃћрЃЉрЃў (Coordinate Transformations)  

­ЪЉЅ рЃерЃћрЃЊрЃћрЃњрЃў: рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃў рЃАрЃгрЃљрЃЋрЃџрЃЮрЃЉрЃА рЃАрЃўрЃЋрЃарЃфрЃўрЃЌрЃў рЃљрЃюрЃљрЃџрЃўрЃќрЃўрЃА рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃўрЃќрЃљрЃфрЃўрЃљрЃА.  

---

## 5№ИЈРЃБ рЃЎрЃџрЃљрЃАрЃћрЃЉрЃў рЃЊрЃљ рЃЏрЃЮрЃЊрЃБрЃџрЃљрЃарЃЮрЃЉрЃљ PyQGIS-рЃерЃў  
- рЃЎрЃџрЃљрЃАрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ PyQGIS рЃАрЃЎрЃарЃўрЃърЃбрЃћрЃЉрЃерЃў  
- Manager рЃЎрЃџрЃљрЃАрЃћрЃЉрЃў (LayerManager, GeometryManager)  
- рЃЏрЃћрЃЏрЃЎрЃЋрЃўрЃЊрЃарЃћрЃЮрЃЉрЃљ рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃбрЃўрЃърЃўрЃА рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА  
- рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃљрЃЊ рЃЊрЃљрЃДрЃЮрЃцрЃљ (Utilities.py, Analysis.py)  

­ЪЉЅ рЃерЃћрЃЊрЃћрЃњрЃў: рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃА рЃљрЃЦрЃЋрЃА рЃЮрЃарЃњрЃљрЃюрЃўрЃќрЃћрЃЉрЃБрЃџрЃў рЃЊрЃљ рЃ«рЃћрЃџрЃљрЃ«рЃџрЃљ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљрЃЊрЃў рЃЎрЃЮрЃЊрЃў.  

---

## 6№ИЈРЃБ GUI рЃЊрЃљ рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃўрЃА рЃњрЃљрЃюрЃЋрЃўрЃЌрЃљрЃарЃћрЃЉрЃљ  
- PyQt-рЃўрЃА рЃўрЃюрЃбрЃћрЃњрЃарЃљрЃфрЃўрЃљ PyQGIS-рЃЌрЃљрЃю  
- ToolButton, Dialog, Input widgets  
- QgsMapCanvas РђЊ рЃарЃБрЃЎрЃўрЃА рЃЕрЃЋрЃћрЃюрЃћрЃЉрЃљ рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃерЃў  
- рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃърЃџрЃљрЃњрЃўрЃюрЃўрЃА рЃЕрЃЮрЃюрЃЕрЃ«рЃў (Plugin Builder-рЃўрЃЌ рЃљрЃю рЃ«рЃћрЃџрЃўрЃЌ)  

­ЪЉЅ рЃерЃћрЃЊрЃћрЃњрЃў: рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃЏрЃљ рЃўрЃфрЃўрЃА рЃарЃЮрЃњрЃЮрЃа рЃерЃћрЃЦрЃЏрЃюрЃљрЃА рЃЏрЃљрЃарЃбрЃўрЃЋрЃў QGIS рЃърЃџрЃљрЃњрЃўрЃюрЃў.  

---

## 7№ИЈРЃБ рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃўрЃќрЃљрЃфрЃўрЃљ рЃЊрЃљ рЃАрЃЎрЃарЃўрЃърЃбрЃўрЃюрЃњрЃў  
- Model Рєњ Script рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃюрЃљ  
- Batch рЃърЃарЃЮрЃфрЃћрЃАрЃўрЃюрЃњрЃў PyQGIS-рЃўрЃЌ  
- рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃљ (рЃЏрЃљрЃњ., CSV Рєњ Shapefile Рєњ Buffer)  
- рЃџрЃЮрЃњрЃўрЃарЃћрЃЉрЃљ рЃЊрЃљ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ  

­ЪЉЅ рЃерЃћрЃЊрЃћрЃњрЃў: рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃА рЃерЃћрЃБрЃФрЃџрЃўрЃљ рЃърЃарЃЮрЃфрЃћрЃАрЃћрЃЉрЃўрЃА рЃАрЃарЃБрЃџрЃљрЃЊ рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃўрЃќрЃћрЃЉрЃљ.  

---

## 8№ИЈРЃБ рЃЏрЃЮрЃгрЃўрЃюрЃљрЃЋрЃћ рЃЌрЃћрЃЏрЃћрЃЉрЃў  
- PyQGIS + PostgreSQL/PostGIS рЃЎрЃљрЃЋрЃерЃўрЃарЃў  
- PyQGIS + Processing Toolbox  
- Network Analysis, Routing  
- Raster Analysis (DEM, slope, hillshade)  
- PyQGIS + Web (QGIS Server, PyQGIS API рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ backend-рЃерЃў)  

­ЪЉЅ рЃерЃћрЃЊрЃћрЃњрЃў: рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃў рЃБрЃЎрЃЋрЃћ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА рЃарЃћрЃљрЃџрЃБрЃа GIS рЃљрЃЏрЃЮрЃфрЃљрЃюрЃћрЃЉрЃќрЃћ.  

---

## 9№ИЈРЃБ Capstone Project  
- рЃЏрЃфрЃўрЃарЃћ рЃърЃарЃЮрЃћрЃЦрЃбрЃў, рЃАрЃљрЃЊрЃљрЃф рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃў рЃЌрЃЋрЃўрЃЌрЃЮрЃю рЃЦрЃЏрЃюрЃўрЃА PyQGIS рЃАрЃЎрЃарЃўрЃърЃбрЃА рЃљрЃю рЃърЃџрЃљрЃњрЃўрЃюрЃА  
- рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЕрЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃљ Рєњ рЃљрЃюрЃљрЃџрЃўрЃќрЃў Рєњ рЃерЃћрЃЊрЃћрЃњрЃўрЃА рЃЋрЃўрЃќрЃБрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃљ  
- рЃЎрЃЮрЃЊрЃўрЃА рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃўрЃарЃћрЃЉрЃљ рЃЊрЃљ рЃњрЃљрЃќрЃўрЃљрЃарЃћрЃЉрЃљ GitHub-рЃќрЃћ  

­ЪЉЅ рЃерЃћрЃЊрЃћрЃњрЃў: рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃА рЃљрЃЦрЃЋрЃА рЃАрЃљрЃЎрЃБрЃЌрЃљрЃарЃў **рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃБрЃџрЃў рЃърЃарЃЮрЃћрЃЦрЃбрЃў**, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф CV-рЃерЃў рЃЕрЃљрЃАрЃљрЃгрЃћрЃарЃљрЃЊ рЃњрЃљрЃЏрЃЮрЃЊрЃњрЃћрЃЉрЃљ.  

---

# ­Ъћ╣ рЃерЃћрЃ»рЃљрЃЏрЃћрЃЉрЃљ  
PyQGIS рЃАрЃгрЃљрЃЋрЃџрЃўрЃА рЃЮрЃърЃбрЃўрЃЏрЃљрЃџрЃБрЃарЃў рЃњрЃќрЃљ:  
1. рЃАрЃљрЃцрЃБрЃФрЃЋрЃџрЃћрЃЉрЃў (QGIS + Python Console)  
2. рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃћрЃЉрЃўрЃА рЃАрЃљрЃцрЃБрЃФрЃЋрЃџрЃћрЃЉрЃў (Layers, Features, Geometry)  
3. рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ  
4. рЃАрЃўрЃЋрЃарЃфрЃўрЃЌрЃў рЃљрЃюрЃљрЃџрЃўрЃќрЃў  
5. рЃЎрЃџрЃљрЃАрЃћрЃЉрЃў рЃЊрЃљ рЃЏрЃЮрЃЊрЃБрЃџрЃљрЃарЃЮрЃЉрЃљ  
6. GUI рЃЊрЃљ рЃърЃџрЃљрЃњрЃўрЃюрЃћрЃЉрЃў  
7. рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃўрЃќрЃљрЃфрЃўрЃљ  
8. рЃЏрЃЮрЃгрЃўрЃюрЃљрЃЋрЃћ рЃЌрЃћрЃЏрЃћрЃЉрЃў  
9. рЃАрЃљрЃЉрЃЮрЃџрЃЮрЃЮ рЃърЃарЃЮрЃћрЃЦрЃбрЃў  

­ЪЉЅ рЃљрЃЏ рЃњрЃќрЃўрЃА рЃњрЃљрЃЋрЃџрЃўрЃЌ рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃў рЃњрЃљрЃ«рЃЊрЃћрЃЉрЃљ рЃљрЃарЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ QGIS рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃў, рЃљрЃарЃљрЃЏрЃћрЃЊ **GIS рЃЊрЃћрЃЋрЃћрЃџрЃЮрЃърЃћрЃарЃў**.
