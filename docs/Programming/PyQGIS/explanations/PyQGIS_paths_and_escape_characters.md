# ­ЪЊѓ Python-рЃерЃў рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃћрЃЉрЃўрЃА рЃАрЃгрЃЮрЃарЃљрЃЊ рЃЕрЃљрЃгрЃћрЃарЃљ

Python-рЃерЃў рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃћрЃЉрЃўрЃА рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃћрЃЉрЃўрЃА рЃЕрЃљрЃгрЃћрЃарЃўрЃАрЃљрЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃўрЃљ рЃАрЃгрЃЮрЃарЃљрЃЊ рЃњрЃљрЃюрЃЋрЃљрЃАрЃ«рЃЋрЃљрЃЋрЃЮрЃЌ **рЃБрЃЎрЃБрЃЦрЃфрЃћрЃБрЃџрЃў рЃ«рЃљрЃќрЃў (\\)** рЃЊрЃљ **рЃЏрЃўрЃЏрЃљрЃарЃЌрЃБрЃџрЃў рЃ«рЃљрЃќрЃў (/)**.  

---

## РюЁ рЃЋрЃљрЃџрЃўрЃЊрЃБрЃарЃў рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃћрЃЉрЃўрЃА рЃЕрЃљрЃюрЃљрЃгрЃћрЃарЃћрЃЉрЃў

Python-рЃерЃў рЃћрЃарЃЌрЃќрЃћ рЃЏрЃћрЃбрЃў рЃњрЃќрЃљрЃљ рЃЉрЃўрЃџрЃўрЃЎрЃћрЃЉрЃўрЃА (path) рЃАрЃгрЃЮрЃарЃљрЃЊ рЃЕрЃљрЃАрЃљрЃгрЃћрЃарЃљрЃЊ:

- рЃЮрЃарЃЏрЃљрЃњрЃў рЃБрЃЎрЃБрЃЦрЃфрЃћрЃБрЃџрЃў рЃ«рЃљрЃќрЃў (**\\**)  
- рЃЏрЃўрЃЏрЃљрЃарЃЌрЃБрЃџрЃў рЃ«рЃљрЃќрЃў (**/**)  
- **raw string** рЃАрЃўрЃюрЃбрЃљрЃЦрЃАрЃў (r"...")  

> рЃЌрЃБ рЃўрЃДрЃћрЃюрЃћрЃЉ `/` рЃљрЃю `\\` Рђћ **r рЃЊрЃљрЃгрЃћрЃарЃљ рЃАрЃљрЃЋрЃљрЃџрЃЊрЃћрЃЉрЃБрЃџрЃЮ рЃљрЃдрЃљрЃарЃљрЃљ**.

### рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃў:
```py
r"C:\Lessons\NotebookStart\Toronto.gdb\ambulances"
"C:/Lessons/NotebookStart/Toronto.gdb/ambulances"
"C:\\Lessons\\NotebookStart\\Toronto.gdb\\ambulances"
```

рЃДрЃЋрЃћрЃџрЃљ рЃћрЃА рЃЕрЃљрЃюрЃљрЃгрЃћрЃарЃў рЃАрЃгрЃЮрЃарЃўрЃљ рЃЊрЃљ **рЃБрЃарЃЌрЃўрЃћрЃарЃЌрЃерЃћрЃюрЃљрЃфрЃЋрЃџрЃћрЃЉрЃљрЃЊрЃўрЃљ**.

---

## ­Ъћ╣ рЃЎрЃЮрЃЊрЃерЃў рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃћрЃЉрЃўрЃА рЃАрЃљрЃЎрЃўрЃЌрЃ«рЃў

Python-рЃерЃў рЃћрЃарЃЌрЃў рЃБрЃЎрЃБрЃЦрЃфрЃћрЃБрЃџрЃў рЃ«рЃљрЃќрЃў **(\)** рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃарЃЮрЃњрЃЮрЃарЃф **Escape Character**.  
рЃљрЃЏрЃўрЃА рЃњрЃљрЃЏрЃЮ, рЃЉрЃўрЃџрЃўрЃЎрЃћрЃЉрЃерЃў рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃћрЃЉрЃБрЃџрЃўрЃљ рЃЮрЃарЃЏрЃљрЃњрЃў рЃБрЃЎрЃБрЃЦрЃфрЃћрЃБрЃџрЃў рЃ«рЃљрЃќрЃў (\\), raw рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃў (r"...") рЃљрЃю рЃЏрЃўрЃЏрЃљрЃарЃЌрЃБрЃџрЃў рЃ«рЃљрЃќрЃў (/).

рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃў:

```py
r"C:\QGIS\PyQGIS\database.gpkg"
"C:/QGIS/PyQGIS/database.gpkg"
"C:\\QGIS\\PyQGIS\\database.gpkg"
```

# ­ЪћЉ Escape Character Python-рЃерЃў

Escape Character рЃљрЃарЃўрЃА **рЃБрЃЎрЃБрЃЦрЃфрЃћрЃБрЃџрЃў рЃ«рЃљрЃќрЃў ( \ )**, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃАрЃърЃћрЃфрЃўрЃљрЃџрЃБрЃарЃў рЃАрЃўрЃЏрЃЉрЃЮрЃџрЃЮрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃАрЃљрЃбрЃљрЃюрЃљрЃЊ.  

---

## ­ЪЊЇ рЃАрЃърЃћрЃфрЃўрЃљрЃџрЃБрЃарЃў рЃАрЃўрЃЏрЃЉрЃЮрЃџрЃЮрЃћрЃЉрЃўрЃА рЃЕрЃљрЃАрЃЏрЃљ

### рЃЉрЃарЃГрЃДрЃљрЃџрЃћрЃЉрЃў рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃерЃў
```py title="Special characters" linenums="1"
City = "My city is \"Chiatura!\""
print(City)
# рЃњрЃљрЃЏрЃЮрЃАрЃљрЃЋрЃљрЃџрЃў: My city is "Chiatura!"
```

### рЃљрЃ«рЃљрЃџрЃў рЃ«рЃљрЃќрЃў `\n`
```py title="New line" linenums="1"
City = "My city is \nChiatura"
print(City)
# рЃњрЃљрЃЏрЃЮрЃАрЃљрЃЋрЃљрЃџрЃў:
# My city is
# Chiatura
```

### рЃбрЃљрЃЉрЃБрЃџрЃљрЃфрЃўрЃљ `\t`
```py title="Tab character" linenums="1"
City = "My city is \tChiatura"
print(City)
# рЃњрЃљрЃЏрЃЮрЃАрЃљрЃЋрЃљрЃџрЃў: My city is   Chiatura
```

### рЃБрЃЎрЃБрЃЦрЃфрЃћрЃБрЃџрЃў рЃ«рЃљрЃќрЃў `\\`
рЃЌрЃБ рЃњрЃЋрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ рЃЌрЃљрЃЋрЃљрЃЊ рЃ«рЃљрЃќрЃўрЃА рЃњрЃљрЃЏрЃЮрЃЕрЃћрЃюрЃљ, рЃЏрЃљрЃА рЃЮрЃарЃ»рЃћрЃа рЃЋрЃгрЃћрЃарЃЌ:

```py title="Backslash" linenums="1"
path = "C:\\QGIS\\PyQGIS\\database.gpkg"
print(path)
# рЃњрЃљрЃЏрЃЮрЃАрЃљрЃЋрЃљрЃџрЃў: C:\QGIS\PyQGIS\database.gpkg
```

---

## ­Ъќ╝№ИЈ Path рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃўрЃА рЃЋрЃўрЃќрЃБрЃљрЃџрЃБрЃарЃў рЃЊрЃўрЃљрЃњрЃарЃљрЃЏрЃљ

Windows рЃЊрЃљ Linux рЃЉрЃўрЃџрЃўрЃЎрЃћрЃЉрЃўрЃА рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃў:

```
Windows:   C:\Users\Student\Documents\project\data.csv
Linux:     /home/student/project/data.csv
Raw string: r"C:\Users\Student\Documents\project\data.csv"
```


---

## ­ЪЊї рЃерЃћрЃ»рЃљрЃЏрЃћрЃЉрЃљ
- **\** Рєњ Escape Character (рЃАрЃърЃћрЃфрЃўрЃљрЃџрЃБрЃарЃў рЃАрЃўрЃЏрЃЉрЃЮрЃџрЃЮрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА).  
- **\\** Рєњ рЃарЃћрЃљрЃџрЃБрЃарЃў рЃБрЃЎрЃБрЃЦрЃфрЃћрЃБрЃџрЃў рЃ«рЃљрЃќрЃў рЃЉрЃўрЃџрЃўрЃЎрЃерЃў.  
- **/** Рєњ рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮ Windows-рЃќрЃћрЃф.  
- **r"..."** Рєњ raw string, рЃњрЃљрЃЏрЃЮрЃарЃўрЃфрЃ«рЃљрЃЋрЃА Escape Character-рЃўрЃА рЃАрЃљрЃГрЃўрЃарЃЮрЃћрЃЉрЃљрЃА.  

­ЪЉЅ рЃћрЃА рЃДрЃЋрЃћрЃџрЃљрЃцрЃћрЃарЃў Python-рЃерЃў рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃАрЃгрЃЮрЃарЃў рЃЉрЃўрЃџрЃўрЃЎрЃћрЃЉрЃўрЃА рЃЕрЃљрЃАрЃљрЃгрЃћрЃарЃљрЃЊ рЃЊрЃљ рЃбрЃћрЃЦрЃАрЃбрЃЌрЃљрЃю рЃЏрЃБрЃерЃљрЃЮрЃЉрЃўрЃАрЃљрЃА рЃАрЃўрЃќрЃБрЃАрЃбрЃўрЃА рЃерЃћрЃАрЃљрЃюрЃљрЃарЃЕрЃБрЃюрЃћрЃЉрЃџрЃљрЃЊ.
