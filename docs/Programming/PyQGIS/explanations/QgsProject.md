# QgsProject

`QgsProject` рЃљрЃарЃўрЃА PyQGIS-рЃўрЃА рЃћрЃарЃЌ-рЃћрЃарЃЌрЃў рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў рЃЎрЃџрЃљрЃАрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ QGIS рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА (`.qgz` рЃљрЃю `.qgs`) рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃАрЃљрЃЏрЃљрЃарЃЌрЃљрЃЋрЃљрЃЊ.  
рЃљрЃЏ рЃЎрЃџрЃљрЃАрЃўрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃўрЃЌ рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ:

- рЃљрЃарЃАрЃћрЃЉрЃБрЃџрЃў рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃњрЃљрЃ«рЃАрЃюрЃљ  
- рЃљрЃ«рЃљрЃџрЃў рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ  
- рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃерЃћрЃюрЃљрЃ«рЃЋрЃљ  
- рЃцрЃћрЃюрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ/рЃљрЃЏрЃЮрЃдрЃћрЃЉрЃљ рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃЊрЃљрЃю  
- рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃЌрЃЋрЃўрЃАрЃћрЃЉрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ (CRS, рЃњрЃќрЃћрЃЉрЃў, рЃЕрЃљрЃюрЃљрЃгрЃћрЃарЃћрЃЉрЃў)

---

## ­ЪЊї рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃЏрЃћрЃЌрЃЮрЃЊрЃћрЃЉрЃў рЃЊрЃљ рЃљрЃбрЃарЃўрЃЉрЃБрЃбрЃћрЃЉрЃў

### 1. `QgsProject.instance()`
QGIS-рЃерЃў рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃћрЃарЃЌрЃў рЃљрЃЦрЃбрЃўрЃБрЃарЃў рЃърЃарЃЮрЃћрЃЦрЃбрЃў. рЃЏрЃўрЃАрЃў рЃгрЃЋрЃЊрЃЮрЃЏрЃљ рЃ«рЃЊрЃћрЃЉрЃљ `instance()` рЃЏрЃћрЃЌрЃЮрЃЊрЃўрЃЌ.

```python
project = QgsProject.instance()
```

---

### 2. `setFileName(path)`
рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃцрЃљрЃўрЃџрЃўрЃА рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃўрЃА рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃљ.

```python
project.setFileName("C:/Users/Public/Documents/GIS/my_project.qgz")
```

---

### 3. `read(path)` РђЊ рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃњрЃљрЃ«рЃАрЃюрЃљ
рЃљрЃарЃАрЃћрЃЉрЃБрЃџрЃў `.qgz` рЃљрЃю `.qgs` рЃцрЃљрЃўрЃџрЃўрЃА рЃЕрЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃљ.

```python
project.read("C:/Users/Public/Documents/GIS/existing_project.qgz")
```

---

### 4. `write()`
рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃерЃћрЃюрЃљрЃ«рЃЋрЃљ рЃўрЃЏрЃљрЃЋрЃћ рЃцрЃљрЃўрЃџрЃерЃў, рЃарЃЮрЃЏрЃћрЃџрЃАрЃљрЃф рЃЏрЃўрЃћрЃюрЃўрЃГрЃљ `setFileName()`.

```python
project.write()
```

рЃљрЃю рЃљрЃ«рЃљрЃџрЃў рЃАрЃљрЃ«рЃћрЃџрЃўрЃЌ:

```python
project.write("C:/Users/Public/Documents/GIS/new_project.qgz")
```

---

### 5. рЃцрЃћрЃюрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ рЃърЃарЃЮрЃћрЃЦрЃбрЃерЃў
рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃЊрЃљрЃЋрЃљрЃЏрЃљрЃбрЃЮрЃЌ рЃарЃЮрЃњрЃЮрЃарЃф рЃЋрЃћрЃЦрЃбрЃЮрЃарЃБрЃџрЃў, рЃўрЃАрЃћ рЃарЃљрЃАрЃбрЃарЃБрЃџрЃў рЃцрЃћрЃюрЃћрЃЉрЃў.

```python
from qgis.core import QgsVectorLayer, QgsRasterLayer

# рЃЋрЃћрЃЦрЃбрЃЮрЃарЃБрЃџрЃў рЃцрЃћрЃюрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ
vlayer = QgsVectorLayer("C:/Users/Public/Documents/GIS/roads.shp", "Roads", "ogr")
project.addMapLayer(vlayer)

# рЃарЃљрЃАрЃбрЃарЃБрЃџрЃў рЃцрЃћрЃюрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ
rlayer = QgsRasterLayer("C:/Users/Public/Documents/GIS/dem.tif", "DEM")
project.addMapLayer(rlayer)
```

---

### 6. `removeMapLayer(layer)`
рЃцрЃћрЃюрЃўрЃА рЃгрЃљрЃерЃџрЃљ рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃЊрЃљрЃю.

```python
project.removeMapLayer(vlayer.id())
```

---

### 7. `mapLayers()`
рЃърЃарЃЮрЃћрЃЦрЃбрЃерЃў рЃљрЃарЃАрЃћрЃЉрЃБрЃџрЃў рЃДрЃЋрЃћрЃџрЃљ рЃцрЃћрЃюрЃўрЃА dictionary-рЃА рЃЊрЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃљ (`layer_id: layer_object`).

```python
layers = project.mapLayers()
for layer_id, layer in layers.items():
    print(layer.name(), layer.type())
```

---

### 8. CRS (Coordinate Reference System) рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃАрЃЌрЃЋрЃўрЃА
рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў CRS-рЃўрЃА рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃљ.

```python
from qgis.core import QgsCoordinateReferenceSystem

crs = QgsCoordinateReferenceSystem("EPSG:4326")  # WGS84
project.setCrs(crs)
```

## ­ЪЊї рЃАрЃарЃБрЃџрЃў рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў

```python
from qgis.core import QgsProject, QgsVectorLayer, QgsRasterLayer, QgsCoordinateReferenceSystem

# рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃўрЃюрЃАрЃбрЃљрЃюрЃАрЃўрЃА рЃЏрЃўрЃдрЃћрЃЉрЃљ
project = QgsProject.instance()

# рЃцрЃљрЃўрЃџрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃўрЃА рЃЏрЃўрЃюрЃўрЃГрЃћрЃЉрЃљ
project.setFileName("C:/Users/Public/Documents/GIS/my_project.qgz")

# рЃЋрЃћрЃЦрЃбрЃЮрЃарЃБрЃџрЃў рЃцрЃћрЃюрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ
roads = QgsVectorLayer("C:/Users/Public/Documents/GIS/roads.shp", "Roads", "ogr")
project.addMapLayer(roads)

# рЃарЃљрЃАрЃбрЃарЃБрЃџрЃў рЃцрЃћрЃюрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ
dem = QgsRasterLayer("C:/Users/Public/Documents/GIS/dem.tif", "DEM")
project.addMapLayer(dem)

# CRS рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃљ
crs = QgsCoordinateReferenceSystem("EPSG:32638")  # UTM Zone 38N
project.setCrs(crs)

# рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃерЃћрЃюрЃљрЃ«рЃЋрЃљ
project.write()
```

## ­ЪЊќ рЃЊрЃљрЃАрЃЎрЃЋрЃюрЃљ
`QgsProject` рЃњрЃЋрЃљрЃФрЃџрЃћрЃЋрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃЋрЃЏрЃљрЃарЃЌрЃЮрЃЌ рЃЏрЃЌрЃћрЃџрЃў QGIS рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃБрЃџрЃљрЃЊ.  
рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃћрЃА рЃљрЃарЃўрЃА рЃћрЃарЃЌ-рЃћрЃарЃЌрЃў рЃАрЃљрЃЉрЃљрЃќрЃўрЃАрЃЮ рЃЌрЃћрЃЏрЃљ, рЃарЃљрЃЊрЃњрЃљрЃю рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃБрЃџрЃљрЃЊ рЃДрЃЋрЃћрЃџрЃљ PyQGIS рЃАрЃЎрЃарЃўрЃърЃбрЃерЃў рЃАрЃљрЃГрЃўрЃарЃЮрЃљ рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃцрЃћрЃюрЃћрЃЉрЃЌрЃљрЃю рЃЏрЃБрЃерЃљрЃЮрЃЉрЃљ.
