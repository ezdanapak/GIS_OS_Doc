# QgsPointXY PyQGIS-рЃерЃў

## Рё╣№ИЈ рЃќрЃЮрЃњрЃљрЃЊрЃў рЃљрЃдрЃгрЃћрЃарЃљ
**QgsPointXY** рЃљрЃарЃўрЃА рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃгрЃљрЃарЃЏрЃЮрЃљрЃЊрЃњрЃћрЃюрЃА л┤л▓ЛЃл╝рЃћрЃбрЃарЃўрЃљрЃю рЃгрЃћрЃарЃбрЃўрЃџрЃА (X,Y рЃЎрЃЮрЃЮрЃарЃЊрЃўрЃюрЃљрЃбрЃћрЃЉрЃўрЃЌ).  
рЃўрЃњрЃў рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃарЃЮрЃњрЃЮрЃарЃф рЃњрЃћрЃЮрЃЏрЃћрЃбрЃарЃўрЃБрЃџрЃў рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃћрЃЉрЃўрЃА (рЃЏрЃљрЃњ. `QgsGeometry`) рЃерЃћрЃАрЃљрЃЦрЃЏрЃюрЃћрЃџрЃљрЃЊ рЃЊрЃљ рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃњрЃћрЃЮрЃњрЃарЃљрЃцрЃўрЃБрЃџ рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃерЃў.

---

## ­ЪЏа№ИЈ рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃЎрЃЮрЃюрЃАрЃбрЃарЃБрЃЦрЃбрЃЮрЃарЃћрЃЉрЃў
```python
# рЃфрЃљрЃарЃўрЃћрЃџрЃў рЃгрЃћрЃарЃбрЃўрЃџрЃў
pt = QgsPointXY()

# X рЃЊрЃљ Y рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃўрЃЌ
pt = QgsPointXY(357965.61, 4683353.56)

# рЃАрЃ«рЃЋрЃљ QgsPointXY рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃўрЃА рЃЎрЃЮрЃърЃўрЃарЃћрЃЉрЃљ
pt2 = QgsPointXY(pt)
```

## ­ЪЊї рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃЏрЃћрЃЌрЃЮрЃЊрЃћрЃЉрЃў

### рЃЎрЃЮрЃЮрЃарЃЊрЃўрЃюрЃљрЃбрЃћрЃЉрЃќрЃћ рЃгрЃЋрЃЊрЃЮрЃЏрЃљ
```python
x = pt.x()   # рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА X рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљрЃА
y = pt.y()   # рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА Y рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљрЃА
```

### рЃЎрЃЮрЃЮрЃарЃЊрЃўрЃюрЃљрЃбрЃћрЃЉрЃўрЃА рЃерЃћрЃфрЃЋрЃџрЃљ
```python
pt.setX(358000.0)
pt.setY(4683400.0)
```

### рЃгрЃћрЃарЃбрЃўрЃџрЃўрЃА рЃерЃћрЃЊрЃљрЃарЃћрЃЉрЃљ
```python
pt1 = QgsPointXY(10, 20)
pt2 = QgsPointXY(10, 20)
pt3 = QgsPointXY(15, 25)

print(pt1 == pt2)  # True
print(pt1 == pt3)  # False
```

## ­ЪДГ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ QgsGeometry-рЃЌрЃљрЃю
рЃ«рЃерЃўрЃарЃљрЃЊ `QgsPointXY` рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃњрЃћрЃЮрЃЏрЃћрЃбрЃарЃўрЃўрЃА рЃерЃћрЃАрЃљрЃЦрЃЏрЃюрЃћрЃџрЃљрЃЊ:
```python
pt = QgsPointXY(357965.61, 4683353.56)
geom = QgsGeometry.fromPointXY(pt)
```

## ­ЪїЇ рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃБрЃџрЃў рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў Рђћ рЃгрЃћрЃарЃбрЃўрЃџрЃЮрЃЋрЃљрЃюрЃў рЃерЃарЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ
```python
fn = r'C:/Users/Public/Documents/GIS/points.shp'

fields = QgsFields()
fields.append(QgsField('ID', QVariant.Int))
fields.append(QgsField('Name', QVariant.String))

writer = QgsVectorFileWriter(fn, 'UTF-8', fields, QgsWkbTypes.Point,
                             QgsCoordinateReferenceSystem('EPSG:32638'), 'ESRI Shapefile')

# рЃгрЃћрЃарЃбрЃўрЃџрЃћрЃЉрЃў
coords = [(357960, 4683350), (357970, 4683360), (357980, 4683370)]

for i, (x, y) in enumerate(coords):
    pt = QgsPointXY(x, y)
    geom = QgsGeometry.fromPointXY(pt)
    feat = QgsFeature()
    feat.setGeometry(geom)
    feat.setAttributes([i, f"point_{i}"])
    writer.addFeature(feat)

del(writer)

layer = iface.addVectorLayer(fn, '', 'ogr')
```

## ­ЪЊќ рЃарЃЮрЃЊрЃўрЃА рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃЮрЃЌ
- рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃгрЃћрЃарЃбрЃўрЃџрЃўрЃА рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА (рЃЏрЃљрЃњ. X,Y рЃЎрЃЮрЃЮрЃарЃЊрЃўрЃюрЃљрЃбрЃћрЃЉрЃў).
- рЃњрЃћрЃЮрЃЏрЃћрЃбрЃарЃўрЃўрЃА (`QgsGeometry`) рЃерЃћрЃАрЃљрЃЦрЃЏрЃюрЃћрЃџрЃљрЃЊ.
- рЃЊрЃарЃЮрЃћрЃЉрЃўрЃЌрЃў рЃгрЃћрЃарЃбрЃўрЃџрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃюрЃљрЃ«рЃљрЃЊ рЃЊрЃљ рЃљрЃюрЃљрЃџрЃўрЃќрЃўрЃАрЃЌрЃЋрЃўрЃА.
- рЃљрЃџрЃњрЃЮрЃарЃўрЃЌрЃЏрЃћрЃЉрЃўрЃА рЃњрЃљрЃАрЃљрЃЏрЃљрЃарЃбрЃўрЃЋрЃћрЃЉрЃџрЃљрЃЊ, рЃАрЃљрЃЊрЃљрЃф рЃАрЃарЃБрЃџрЃў рЃњрЃћрЃЮрЃЏрЃћрЃбрЃарЃўрЃБрЃџрЃў рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃў рЃАрЃљрЃГрЃўрЃарЃЮ рЃљрЃа рЃљрЃарЃўрЃА.

---

РюЁ **QgsPointXY** PyQGIS-рЃерЃў рЃћрЃарЃЌ-рЃћрЃарЃЌрЃў рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃ«рЃерЃўрЃарЃљрЃЊ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљрЃЊрЃў рЃЎрЃџрЃљрЃАрЃў рЃгрЃћрЃарЃбрЃўрЃџрЃЮрЃЋрЃљрЃю рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃЌрЃљрЃю рЃЏрЃБрЃерЃљрЃЮрЃЉрЃўрЃАрЃљрЃА.
