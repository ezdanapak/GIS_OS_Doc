# ­ЪљЇ рЃърЃўрЃЌрЃЮрЃюрЃўрЃА рЃърЃарЃЮрЃћрЃЦрЃбрЃћрЃЉрЃўрЃА рЃћрЃцрЃћрЃЦрЃбрЃБрЃарЃў рЃЏрЃљрЃарЃЌрЃЋрЃљ рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃарЃў рЃњрЃљрЃарЃћрЃЏрЃЮрЃћрЃЉрЃўрЃЌ  

## ­Ъћ╣ рЃерЃћрЃАрЃљрЃЋрЃљрЃџрЃў  
рЃарЃЮрЃЊрЃћрЃАрЃљрЃф рЃЏрЃБрЃерЃљрЃЮрЃЉрЃЌ рЃърЃўрЃЌрЃЮрЃюрЃўрЃА рЃърЃарЃЮрЃћрЃЦрЃбрЃћрЃЉрЃќрЃћ, рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃерЃћрЃљрЃЏрЃЕрЃюрЃўрЃЮрЃЌ, рЃарЃЮрЃЏ рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃърЃарЃЮрЃћрЃЦрЃбрЃА рЃАрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ рЃърЃўрЃЌрЃЮрЃюрЃўрЃА рЃљрЃю рЃърЃљрЃЎрЃћрЃбрЃћрЃЉрЃўрЃА рЃњрЃљрЃюрЃАрЃ«рЃЋрЃљрЃЋрЃћрЃЉрЃБрЃџрЃў рЃЋрЃћрЃарЃАрЃўрЃћрЃЉрЃў.  
рЃДрЃЋрЃћрЃџрЃљ рЃърЃљрЃЎрЃћрЃбрЃўрЃА рЃњрЃџрЃЮрЃЉрЃљрЃџрЃБрЃарЃљрЃЊ рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃљрЃЏ рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃгрЃЋрЃўрЃЮрЃА рЃЎрЃЮрЃюрЃцрЃџрЃўрЃЦрЃбрЃћрЃЉрЃў.  

­ЪЉЅ рЃљрЃЦ рЃЕрЃюрЃЊрЃћрЃЉрЃљ **рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃарЃў рЃњрЃљрЃарЃћрЃЏрЃЮ** Рђћ рЃўрЃюрЃАрЃбрЃарЃБрЃЏрЃћрЃюрЃбрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЦрЃЏрЃюрЃўрЃА рЃўрЃќрЃЮрЃџрЃўрЃарЃћрЃЉрЃБрЃџ рЃАрЃўрЃЋрЃарЃфрЃћрЃА рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃАрЃЌрЃЋрЃўрЃА рЃЊрЃљ рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА рЃАрЃБрЃцрЃЌрЃљ рЃЊрЃљ рЃћрЃцрЃћрЃЦрЃбрЃБрЃа рЃЏрЃљрЃарЃЌрЃЋрЃљрЃА.  

---

## ­Ъћ╣ рЃарЃљ рЃљрЃарЃўрЃА рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃарЃў рЃњрЃљрЃарЃћрЃЏрЃЮ?  
рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃарЃў рЃњрЃљрЃарЃћрЃЏрЃЮ рЃљрЃарЃўрЃА рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃърЃўрЃЌрЃЮрЃюрЃўрЃА рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃАрЃЌрЃЋрЃўрЃА рЃњрЃљрЃюрЃЎрЃБрЃЌрЃЋрЃюрЃўрЃџрЃў **РђърЃЦрЃЋрЃўрЃерЃўрЃА рЃДрЃБрЃЌрЃўРђю**.  
рЃћрЃА рЃљрЃарЃўрЃА рЃфрЃљрЃџрЃЎрЃћ рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃерЃћрЃўрЃфрЃљрЃЋрЃА:  
- рЃърЃўрЃЌрЃЮрЃюрЃўрЃА рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџ рЃЋрЃћрЃарЃАрЃўрЃљрЃА  
- рЃДрЃЋрЃћрЃџрЃљ рЃўрЃЏ рЃърЃљрЃЎрЃћрЃбрЃА, рЃарЃљрЃф рЃЌрЃЦрЃЋрЃћрЃюрЃА рЃърЃарЃЮрЃћрЃЦрЃбрЃА рЃАрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ  

­ЪЊќ рЃљрЃюрЃљрЃџрЃЮрЃњрЃўрЃљ: рЃўрЃАрЃћрЃЋрЃћ, рЃарЃЮрЃњрЃЮрЃарЃф рЃАрЃЎрЃЮрЃџрЃљрЃерЃў рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃАрЃљрЃњрЃюрЃўрЃАрЃЌрЃЋрЃўрЃА рЃфрЃљрЃџрЃЎрЃћ рЃарЃЋрЃћрЃБрЃџрЃў рЃњрЃљрЃЦрЃЋрЃЌ, рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃарЃў рЃњрЃљрЃарЃћрЃЏрЃЮ рЃўрЃюрЃљрЃарЃЕрЃБрЃюрЃћрЃЉрЃА рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃћрЃЉрЃћрЃЉрЃА рЃњрЃљрЃюрЃфрЃљрЃџрЃЎрЃћрЃЋрЃћрЃЉрЃБрЃџрЃљрЃЊ.  

---

## ­Ъћ╣ рЃарЃљрЃбрЃЮрЃЏ рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃЮрЃЌ рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃарЃў рЃњрЃљрЃарЃћрЃЏрЃЮрЃћрЃЉрЃў?  

- РюЁ **рЃЎрЃЮрЃюрЃцрЃџрЃўрЃЦрЃбрЃћрЃЉрЃўрЃА рЃЌрЃљрЃЋрЃўрЃЊрЃљрЃю рЃљрЃфрЃўрЃџрЃћрЃЉрЃљ** РђЊ рЃћрЃарЃЌрЃў рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃўрЃДрЃћрЃюрЃћрЃЉрЃА `requests v2.25`, рЃ«рЃЮрЃџрЃЮ рЃЏрЃћрЃЮрЃарЃћ `requests v2.28`-рЃА.  
- РюЁ **рЃАрЃБрЃцрЃЌрЃљ рЃњрЃљрЃюрЃЋрЃўрЃЌрЃљрЃарЃћрЃЉрЃљ** РђЊ рЃњрЃџрЃЮрЃЉрЃљрЃџрЃБрЃарЃў Python рЃарЃЕрЃћрЃЉрЃљ рЃАрЃБрЃцрЃЌрЃљ.  
- РюЁ **рЃњрЃљрЃюрЃЏрЃћрЃЮрЃарЃћрЃЉрЃљрЃЊрЃЮрЃЉрЃљ** РђЊ `requirements.txt` рЃцрЃљрЃўрЃџрЃўрЃЌ рЃАрЃ«рЃЋрЃљрЃАрЃљрЃф рЃерЃћрЃБрЃФрЃџрЃўрЃљ рЃќрЃБрЃАрЃбрЃљрЃЊ рЃўрЃњрЃўрЃЋрЃћ рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ.  
- РюЁ **рЃћрЃЦрЃАрЃърЃћрЃарЃўрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў** РђЊ рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃЊ рЃњрЃљрЃЏрЃЮрЃАрЃфрЃљрЃЊрЃћрЃЌ рЃљрЃ«рЃљрЃџрЃў рЃърЃљрЃЎрЃћрЃбрЃћрЃЉрЃў.  
- РюЁ **рЃърЃЮрЃарЃбрЃљрЃЉрЃћрЃџрЃБрЃарЃЮрЃЉрЃљ** РђЊ рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃћрЃарЃЌрЃюрЃљрЃўрЃарЃљрЃЊ рЃўрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃА рЃАрЃ«рЃЋрЃљ рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃќрЃћрЃф.  

---

## ­Ъћ╣ рЃарЃЮрЃњрЃЮрЃа рЃерЃћрЃЋрЃЦрЃЏрЃюрЃљрЃЌ рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃарЃў рЃњрЃљрЃарЃћрЃЏрЃЮ  

### 1. рЃерЃћрЃЦрЃЏрЃюрЃљ  
```bash
python -m venv venv
```

### 2. рЃњрЃљрЃљрЃЦрЃбрЃўрЃБрЃарЃћрЃЉрЃљ  
- **Windows-рЃќрЃћ**  
```bash
.env\Scriptsctivate
```
- **macOS/Linux-рЃќрЃћ**  
```bash
source venv/bin/activate
```

­ЪЉЅ рЃбрЃћрЃарЃЏрЃўрЃюрЃљрЃџрЃерЃў рЃњрЃљрЃЏрЃЮрЃЕрЃюрЃЊрЃћрЃЉрЃљ `(venv)` Рђћ рЃћрЃА рЃюрЃўрЃерЃюрЃљрЃЋрЃА, рЃарЃЮрЃЏ рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃарЃў рЃњрЃљрЃарЃћрЃЏрЃЮ рЃљрЃЦрЃбрЃўрЃБрЃарЃўрЃљ.  

### 3. рЃърЃљрЃЎрЃћрЃбрЃћрЃЉрЃўрЃА рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃљ  
```bash
pip install requests
pip list
```

### 4. рЃЊрЃћрЃљрЃЦрЃбрЃўрЃЋрЃљрЃфрЃўрЃљ  
```bash
deactivate
```

- **рЃЕрЃљрЃЏрЃЮрЃюрЃљрЃЌрЃЋрЃљрЃџрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ**

```bash
pip freeze > requirements.txt
```

- **рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃљ рЃАрЃ«рЃЋрЃљ рЃњрЃљрЃарЃћрЃЏрЃЮрЃерЃў**

```bash
pip install -r requirements.txt
```

## ­Ъћ╣ рЃАрЃљрЃБрЃЎрЃћрЃЌрЃћрЃАрЃЮ рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃљ

- рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ **рЃћрЃарЃЌрЃў рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃарЃў рЃњрЃљрЃарЃћрЃЏрЃЮ рЃЌрЃўрЃЌрЃЮ рЃърЃарЃЮрЃћрЃЦрЃбрЃќрЃћ**  
- рЃЊрЃљрЃљрЃарЃЦрЃЋрЃўрЃЌ рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃБрЃџрЃў рЃАрЃљрЃ«рЃћрЃџрЃў: `venv` рЃљрЃю `.venv`  
- рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃА рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ **requirements.txt**  
- рЃгрЃљрЃерЃљрЃџрЃћрЃЌ рЃњрЃљрЃарЃћрЃЏрЃЮ, рЃЌрЃБ рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃљрЃдрЃљрЃа рЃњрЃГрЃўрЃарЃЊрЃћрЃЉрЃљрЃЌ  
- рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ Git рЃЎрЃЮрЃЊрЃўрЃА рЃЋрЃћрЃарЃАрЃўрЃћрЃЉрЃўрЃА рЃЊрЃљ рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃЏрЃљрЃарЃЌрЃЋрЃўрЃАрЃЌрЃЋрЃўрЃА

## ­Ъћ╣ venv-рЃўрЃА рЃљрЃџрЃбрЃћрЃарЃюрЃљрЃбрЃўрЃЋрЃћрЃЉрЃў  

- **virtualenv** РђЊ рЃФрЃЋрЃћрЃџрЃў рЃЋрЃћрЃарЃАрЃўрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА  
- **poetry / pipenv** РђЊ рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃћрЃЉрЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ рЃћрЃарЃЌрЃерЃў  
- **conda** РђЊ рЃърЃЮрЃърЃБрЃџрЃљрЃарЃБрЃџрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЏрЃћрЃфрЃюрЃўрЃћрЃарЃћрЃЉрЃљрЃерЃў

## ­Ъћ╣ рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃћрЃЉрЃўрЃА рЃњрЃљрЃЊрЃљрЃГрЃарЃљ  

- РЮї `command not found` Рєњ рЃЊрЃљрЃарЃгрЃЏрЃБрЃюрЃЊрЃўрЃЌ, рЃарЃЮрЃЏ Python PATH-рЃерЃўрЃљ  
- РЮї рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃў рЃЋрЃћрЃарЃАрЃўрЃљ Рєњ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ `python3 -m venv venv`  
- РЮї рЃърЃљрЃЎрЃћрЃбрЃў рЃЋрЃћрЃа рЃЏрЃЮрЃўрЃФрЃћрЃЉрЃюрЃљ Рєњ рЃЊрЃљрЃарЃгрЃЏрЃБрЃюрЃЊрЃўрЃЌ, рЃарЃЮрЃЏ рЃњрЃљрЃарЃћрЃЏрЃЮ рЃњрЃљрЃљрЃЦрЃбрЃўрЃБрЃарЃћрЃЉрЃБрЃџрЃўрЃљ

## ­Ъћ╣ рЃЊрЃљрЃАрЃЎрЃЋрЃюрЃљ  

рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃарЃў рЃњрЃљрЃарЃћрЃЏрЃЮрЃћрЃЉрЃў:
