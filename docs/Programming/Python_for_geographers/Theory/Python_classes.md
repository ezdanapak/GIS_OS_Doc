# ­ЪљЇ рЃърЃўрЃЌрЃЮрЃюрЃўрЃА рЃЎрЃџрЃљрЃАрЃћрЃЉрЃў: рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ рЃњрЃћрЃЮрЃњрЃарЃљрЃцрЃўрЃБрЃџ рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃБрЃџ рЃАрЃўрЃАрЃбрЃћрЃЏрЃћрЃЉрЃерЃў (GIS)

## ­Ъћ╣ рЃерЃћрЃАрЃљрЃЋрЃљрЃџрЃў  
рЃърЃўрЃЌрЃЮрЃюрЃўрЃА рЃЎрЃџрЃљрЃАрЃћрЃЉрЃў рЃљрЃарЃўрЃА рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃќрЃћ рЃЮрЃарЃўрЃћрЃюрЃбрЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃўрЃА (OOP) рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃњрЃЋрЃљрЃФрЃџрЃћрЃЋрЃА рЃерЃћрЃЋрЃЦрЃЏрЃюрЃљрЃЌ рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃў, рЃ«рЃћрЃџрЃљрЃ«рЃџрЃљ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљрЃЊрЃў рЃЊрЃљ рЃЮрЃарЃњрЃљрЃюрЃўрЃќрЃћрЃЉрЃБрЃџрЃў рЃЎрЃЮрЃЊрЃў.  
GIS-рЃерЃў, рЃАрЃљрЃЊрЃљрЃф рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў рЃ«рЃерЃўрЃарЃљрЃЊ рЃЏрЃЮрЃўрЃфрЃљрЃЋрЃА рЃарЃЌрЃБрЃџ рЃАрЃўрЃЋрЃарЃфрЃўрЃЌ рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃћрЃЉрЃА (рЃгрЃћрЃарЃбрЃўрЃџрЃћрЃЉрЃў, рЃ«рЃљрЃќрЃћрЃЉрЃў, рЃърЃЮрЃџрЃўрЃњрЃЮрЃюрЃћрЃЉрЃў), рЃЎрЃџрЃљрЃАрЃћрЃЉрЃў рЃњрЃЋрЃћрЃ«рЃЏрЃљрЃарЃћрЃЉрЃљ рЃљрЃЏ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃћрЃцрЃћрЃЦрЃбрЃБрЃарЃљрЃЊ рЃЏрЃЮрЃЊрЃћрЃџрЃўрЃарЃћрЃЉрЃљрЃерЃў.  

­ЪЉЅ рЃћрЃА рЃАрЃљрЃ«рЃћрЃџрЃЏрЃФрЃдрЃЋрЃљрЃюрЃћрЃџрЃЮ рЃњрЃљрЃюрЃЎрЃБрЃЌрЃЋрЃюрЃўрЃџрЃўрЃљ рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃАрЃгрЃљрЃЋрЃџрЃЮрЃЉрЃћрЃю рЃърЃўрЃЌрЃЮрЃюрЃА рЃЊрЃљ рЃљрЃўрЃюрЃбрЃћрЃарЃћрЃАрЃћрЃЉрЃЌ GIS.  

---

## ­Ъћ╣ рЃарЃљ рЃљрЃарЃўрЃА рЃЎрЃџрЃљрЃАрЃў?  
рЃЎрЃџрЃљрЃАрЃў рЃљрЃарЃўрЃА **РђърЃерЃљрЃЉрЃџрЃЮрЃюрЃўРђю** рЃљрЃю **РђърЃюрЃљрЃ«рЃљрЃќрЃўРђю**, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљрЃЋрЃА:  
- **рЃљрЃбрЃарЃўрЃЉрЃБрЃбрЃћрЃЉрЃА** (рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў, лйл░л┐Лђ. рЃгрЃћрЃарЃбрЃўрЃџрЃўрЃА рЃЎрЃЮрЃЮрЃарЃЊрЃўрЃюрЃљрЃбрЃћрЃЉрЃў)  
- **рЃЏрЃћрЃЌрЃЮрЃЊрЃћрЃЉрЃА** (рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў, лйл░л┐Лђ. рЃЏрЃљрЃюрЃФрЃўрЃџрЃўрЃА рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃљ)  

­ЪЊќ GIS-рЃўрЃА рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:  
- рЃЎрЃџрЃљрЃАрЃў рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃгрЃљрЃарЃЏрЃЮрЃљрЃЊрЃњрЃћрЃюрЃЊрЃћрЃА рЃгрЃћрЃарЃбрЃўрЃџрЃА рЃарЃБрЃЎрЃљрЃќрЃћ  
- рЃЏрЃЊрЃўрЃюрЃљрЃарЃўрЃА рЃ«рЃљрЃќрЃА  
- рЃЦрЃљрЃџрЃљрЃЦрЃўрЃА рЃАрЃљрЃќрЃдрЃЋрЃарЃўрЃА рЃърЃЮрЃџрЃўрЃњрЃЮрЃюрЃА  

---

## ­Ъћ╣ рЃарЃљрЃбрЃЮрЃЏ рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃЮрЃЌ рЃЎрЃџрЃљрЃАрЃћрЃЉрЃў GIS-рЃерЃў?  

- РюЁ **рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЮрЃарЃњрЃљрЃюрЃўрЃќрЃћрЃЉрЃљ** РђЊ рЃарЃЌрЃБрЃџрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃўрЃарЃћрЃЉрЃљ  
- РюЁ **рЃ«рЃћрЃџрЃљрЃ«рЃџрЃљ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ** РђЊ рЃћрЃарЃЌрЃ«рЃћрЃџ рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃў рЃЎрЃџрЃљрЃАрЃў рЃЉрЃћрЃЋрЃа рЃърЃарЃЮрЃћрЃЦрЃбрЃерЃў рЃњрЃљрЃЏрЃЮрЃњрЃљрЃЊрЃњрЃћрЃЉрЃљ  
- РюЁ **рЃАрЃўрЃфрЃ«рЃљрЃЊрЃћ** РђЊ рЃЎрЃЮрЃЊрЃў рЃБрЃцрЃарЃЮ рЃњрЃљрЃАрЃљрЃњрЃћрЃЉрЃў рЃ«рЃЊрЃћрЃЉрЃљ  
- РюЁ **рЃЏрЃЮрЃЊрЃћрЃџрЃўрЃарЃћрЃЉрЃљ** РђЊ рЃарЃћрЃљрЃџрЃБрЃарЃў рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃћрЃЉрЃўрЃА (рЃњрЃќрЃћрЃЉрЃў, рЃерЃћрЃюрЃЮрЃЉрЃћрЃЉрЃў) рЃЏрЃЮрЃЊрЃћрЃџрЃўрЃарЃћрЃЉрЃљ  

---

## ­Ъћ╣ рЃЎрЃџрЃљрЃАрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃўрЃА рЃАрЃљрЃцрЃБрЃФрЃЋрЃџрЃћрЃЉрЃў  

```python
class Point:
    def __init__(self, x, y):
        self.x = x  # x-рЃЎрЃЮрЃЮрЃарЃЊрЃўрЃюрЃљрЃбрЃў
        self.y = y  # y-рЃЎрЃЮрЃЮрЃарЃЊрЃўрЃюрЃљрЃбрЃў
    
    def describe(self):
        return f"рЃгрЃћрЃарЃбрЃўрЃџрЃў рЃЎрЃЮрЃЮрЃарЃЊрЃўрЃюрЃљрЃбрЃћрЃЉрЃўрЃЌ ({self.x}, {self.y})"
```

- `class Point:` Рєњ рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљрЃЋрЃА рЃЎрЃџрЃљрЃАрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃА  
- `__init__` Рєњ рЃЎрЃЮрЃюрЃАрЃбрЃарЃБрЃЦрЃбрЃЮрЃарЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃўрЃАрЃљрЃА рЃўрЃгрЃДрЃћрЃЉрЃА рЃЏрЃўрЃА рЃЌрЃЋрЃўрЃАрЃћрЃЉрЃћрЃЉрЃА  
- `self` Рєњ рЃЏрЃўрЃБрЃЌрЃўрЃЌрЃћрЃЉрЃА рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџ рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃќрЃћ  
- `describe()` Рєњ рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА рЃгрЃћрЃарЃбрЃўрЃџрЃўрЃА рЃљрЃдрЃгрЃћрЃарЃљрЃА  

**рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ:**  
```python
tbilisi = Point(44.8, 41.7)
print(tbilisi.describe())
# рЃерЃћрЃЊрЃћрЃњрЃў: рЃгрЃћрЃарЃбрЃўрЃџрЃў рЃЎрЃЮрЃЮрЃарЃЊрЃўрЃюрЃљрЃбрЃћрЃЉрЃўрЃЌ (44.8, 41.7)
```

---

## ­Ъћ╣ GIS-рЃўрЃА рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў: рЃЏрЃљрЃюрЃФрЃўрЃџрЃўрЃА рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃљ  

```python
import math

class Point:
    def __init__(self, x, y):
        self.x = x
        self.y = y
    
    def describe(self):
        return f"рЃгрЃћрЃарЃбрЃўрЃџрЃў рЃЎрЃЮрЃЮрЃарЃЊрЃўрЃюрЃљрЃбрЃћрЃЉрЃўрЃЌ ({self.x}, {self.y})"
    
    def distance_to(self, other_point):
        dx = self.x - other_point.x
        dy = self.y - other_point.y
        return math.sqrt(dx**2 + dy**2)
```

**рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ:**  
```python
tbilisi = Point(44.8, 41.7)
batumi = Point(41.6, 41.6)

distance = tbilisi.distance_to(batumi)
print(f"рЃЏрЃљрЃюрЃФрЃўрЃџрЃў рЃЌрЃЉрЃўрЃџрЃўрЃАрЃАрЃљ рЃЊрЃљ рЃЉрЃљрЃЌрЃБрЃЏрЃА рЃерЃЮрЃарЃўрЃА: {distance:.2f} рЃћрЃарЃЌрЃћрЃБрЃџрЃў")
```

­ЪЉЅ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ **рЃћрЃЋрЃЎрЃџрЃўрЃЊрЃБрЃарЃў рЃЏрЃљрЃюрЃФрЃўрЃџрЃў**. рЃарЃћрЃљрЃџрЃБрЃа GIS-рЃерЃў рЃерЃћрЃњрЃЋрЃўрЃФрЃџрЃўрЃљ рЃ░рЃљрЃЋрЃћрЃарЃАрЃўрЃюрЃўрЃА рЃцрЃЮрЃарЃЏрЃБрЃџрЃљрЃф рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃЮрЃЌ.  

---

## ­Ъћ╣ GIS-рЃўрЃА рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў: рЃърЃЮрЃџрЃўрЃњрЃЮрЃюрЃўрЃА рЃцрЃљрЃарЃЌрЃЮрЃЉрЃў  

```python
class Polygon:
    def __init__(self, points):
        self.points = points  # рЃгрЃћрЃарЃбрЃўрЃџрЃћрЃЉрЃўрЃА рЃАрЃўрЃљ [(x1, y1), (x2, y2), ...]
    
    def describe(self):
        return f"рЃърЃЮрЃџрЃўрЃњрЃЮрЃюрЃў {len(self.points)} рЃгрЃЋрЃћрЃарЃЮрЃЌрЃў"
    
    def area(self):
        # Shoelace formula
        n = len(self.points)
        area = 0.0
        for i in range(n):
            j = (i + 1) % n
            area += self.points[i][0] * self.points[j][1]
            area -= self.points[j][0] * self.points[i][1]
        return abs(area) / 2.0
```

**рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ:**  
```python
triangle = Polygon([(0, 0), (4, 0), (0, 3)])
print(triangle.describe())
print(f"рЃцрЃљрЃарЃЌрЃЮрЃЉрЃў: {triangle.area()} рЃЎрЃЋ. рЃћрЃарЃЌрЃћрЃБрЃџрЃў")
```

­ЪЉЅ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ **Shoelace formula**, рЃарЃљрЃф GIS-рЃерЃў рЃцрЃљрЃарЃЌрЃЮрЃЊ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ.  

---

## ­Ъћ╣ GIS-рЃўрЃА рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў: рЃЏрЃћрЃЏрЃЎрЃЋрЃўрЃЊрЃарЃћрЃЮрЃЉрЃљ  

```python
class Geometry:
    def __init__(self, name):
        self.name = name
    
    def describe(self):
        return f"рЃњрЃћрЃЮрЃЏрЃћрЃбрЃарЃўрЃБрЃџрЃў рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃў: {self.name}"

class Point(Geometry):
    def __init__(self, x, y, name):
        super().__init__(name)
        self.x = x
        self.y = y
    
    def describe(self):
        return f"рЃгрЃћрЃарЃбрЃўрЃџрЃў {self.name} рЃЎрЃЮрЃЮрЃарЃЊрЃўрЃюрЃљрЃбрЃћрЃЉрЃўрЃЌ ({self.x}, {self.y})"

class Line(Geometry):
    def __init__(self, start_point, end_point, name):
        super().__init__(name)
        self.start = start_point
        self.end = end_point
    
    def length(self):
        return math.sqrt((self.end.x - self.start.x)**2 + (self.end.y - self.start.y)**2)
```

**рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ:**  
```python
tbilisi = Point(44.8, 41.7, "рЃЌрЃЉрЃўрЃџрЃўрЃАрЃў")
batumi = Point(41.6, 41.6, "рЃЉрЃљрЃЌрЃБрЃЏрЃў")
road = Line(tbilisi, batumi, "рЃЌрЃЉрЃўрЃџрЃўрЃАрЃў-рЃЉрЃљрЃЌрЃБрЃЏрЃў")

print(tbilisi.describe())
print(f"рЃ«рЃљрЃќрЃўрЃА {road.name} рЃАрЃўрЃњрЃарЃФрЃћ: {road.length():.2f} рЃћрЃарЃЌрЃћрЃБрЃџрЃў")
```

­ЪЉЅ `Point` рЃЊрЃљ `Line` рЃЏрЃћрЃЏрЃЎрЃЋрЃўрЃЊрЃарЃћрЃЮрЃЉрЃўрЃЌ рЃўрЃдрЃћрЃЉрЃћрЃю Geometry-рЃА, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃљрЃЏрЃљрЃбрЃћрЃЉрЃћрЃю рЃАрЃљрЃЎрЃБрЃЌрЃљрЃа рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃА.

## ­Ъћ╣ рЃАрЃљрЃБрЃЎрЃћрЃЌрЃћрЃАрЃЮ рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃљ 

- рЃЊрЃљрЃўрЃгрЃДрЃћрЃЌ **рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃЎрЃџрЃљрЃАрЃћрЃЉрЃўрЃЌ** (рЃЏрЃљрЃњ., Point)  
- рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ **рЃљрЃдрЃгрЃћрЃарЃўрЃЌрЃў рЃАрЃљрЃ«рЃћрЃџрЃћрЃЉрЃў**  
- **рЃЏрЃћрЃЏрЃЎрЃЋрЃўрЃЊрЃарЃћрЃЮрЃЉрЃљ** рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ рЃњрЃЮрЃюрЃўрЃЋрЃарЃБрЃџрЃљрЃЊ  
- рЃЊрЃљрЃљрЃЏрЃљрЃбрЃћрЃЌ рЃЎрЃЮрЃЏрЃћрЃюрЃбрЃљрЃарЃћрЃЉрЃў рЃЎрЃЮрЃЊрЃерЃў  
- GIS-рЃерЃў рЃ«рЃерЃўрЃарЃљрЃЊ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃћрЃЉрЃў: **geopandas, shapely**  

---

## ­Ъћ╣ рЃЊрЃљрЃАрЃЎрЃЋрЃюрЃљ  

рЃърЃўрЃЌрЃЮрЃюрЃўрЃА рЃЎрЃџрЃљрЃАрЃћрЃЉрЃў GIS-рЃерЃў:  
- рЃўрЃюрЃљрЃарЃЕрЃБрЃюрЃћрЃЉрЃА рЃЎрЃЮрЃЊрЃА рЃЮрЃарЃњрЃљрЃюрЃўрЃќрЃћрЃЉрЃБрЃџрЃљрЃЊ  
- рЃљрЃљрЃЊрЃЋрЃўрЃџрЃћрЃЉрЃА рЃАрЃўрЃЋрЃарЃфрЃўрЃЌрЃў рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃћрЃЉрЃўрЃА рЃЏрЃЮрЃЊрЃћрЃџрЃўрЃарЃћрЃЉрЃљрЃА  
- рЃљрЃЏрЃљрЃарЃбрЃўрЃЋрЃћрЃЉрЃА рЃљрЃюрЃљрЃџрЃўрЃќрЃА (рЃЏрЃљрЃњ., рЃЏрЃљрЃюрЃФрЃўрЃџрЃў, рЃцрЃљрЃарЃЌрЃЮрЃЉрЃў)
