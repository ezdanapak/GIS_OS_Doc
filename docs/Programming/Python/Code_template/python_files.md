# ­ЪЊџ Python: рЃЏрЃБрЃерЃљрЃЮрЃЉрЃљ рЃцрЃљрЃўрЃџрЃћрЃЉрЃЌрЃљрЃю (File I/O)

рЃцрЃљрЃўрЃџрЃћрЃЉрЃЌрЃљрЃю рЃЏрЃБрЃерЃљрЃЮрЃЉрЃљ рЃљрЃарЃўрЃА рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃўрЃА рЃцрЃБрЃюрЃЊрЃљрЃЏрЃћрЃюрЃбрЃБрЃарЃў рЃюрЃљрЃгрЃўрЃџрЃў Рђћ рЃўрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃњрЃЋрЃљрЃФрЃџрЃћрЃЋрЃА рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљ, рЃЕрЃљрЃгрЃћрЃарЃљ, рЃгрЃљрЃерЃџрЃљрЃќрЃћ рЃЊрЃљ рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃљрЃќрЃћ.

## 1. рЃАрЃљрЃЉрЃљрЃќрЃўрЃАрЃЮ рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃў рЃцрЃљрЃўрЃџрЃћрЃЉрЃЌрЃљрЃю

### 1.1. рЃцрЃљрЃўрЃџрЃўрЃА рЃњрЃљрЃ«рЃАрЃюрЃљ рЃЊрЃљ рЃЊрЃљрЃ«рЃБрЃарЃЋрЃљ: `open()` рЃЊрЃљ `close()`

рЃцрЃљрЃўрЃџрЃќрЃћ рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃарЃў рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃўрЃА рЃЊрЃљрЃгрЃДрЃћрЃЉрЃљрЃЏрЃЊрЃћ рЃўрЃА рЃБрЃюрЃЊрЃљ рЃњрЃљрЃўрЃ«рЃАрЃюрЃљрЃА рЃЕрЃљрЃерЃћрЃюрЃћрЃЉрЃБрЃџрЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ `open()`-рЃўрЃЌ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА рЃцрЃљрЃўрЃџрЃўрЃА рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃА.

рЃАрЃўрЃюрЃбрЃљрЃЦрЃАрЃў:
```python
file_object = open(filename, mode)
```

рЃЏрЃўрЃЏрЃЮрЃ«рЃўрЃџрЃЋрЃљ:

| рЃарЃћрЃЪрЃўрЃЏрЃў | рЃљрЃдрЃгрЃћрЃарЃљ | рЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃљ |
|---|---:|---|
| `'r'` | рЃЎрЃўрЃЌрЃ«рЃЋрЃљ (read) | рЃюрЃљрЃњрЃБрЃџрЃўрЃАрЃ«рЃЏрЃћрЃЋрЃў; рЃцрЃљрЃўрЃџрЃў рЃБрЃюрЃЊрЃљ рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃЊрЃћрЃА |
| `'w'` | рЃЕрЃљрЃгрЃћрЃарЃљ (write) | рЃЦрЃЏрЃюрЃўрЃА рЃљрЃ«рЃљрЃџ рЃцрЃљрЃўрЃџрЃА рЃљрЃю рЃгрЃљрЃерЃџрЃўрЃА рЃљрЃарЃАрЃћрЃЉрЃБрЃџ рЃерЃўрЃњрЃЌрЃљрЃЋрЃАрЃА |
| `'a'` | рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ (append) | рЃЕрЃљрЃгрЃћрЃарЃљ рЃцрЃљрЃўрЃџрЃўрЃА рЃЉрЃЮрЃџрЃЮрЃА; рЃЦрЃЏрЃюрЃўрЃА рЃцрЃљрЃўрЃџрЃА, рЃЌрЃБ рЃљрЃа рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА |
| `'r+'` | рЃЎрЃўрЃЌрЃ«рЃЋрЃљ рЃЊрЃљ рЃЕрЃљрЃгрЃћрЃарЃљ | рЃцрЃљрЃўрЃџрЃў рЃБрЃюрЃЊрЃљ рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃЊрЃћрЃА; рЃърЃЮрЃќрЃўрЃфрЃўрЃљ рЃўрЃгрЃДрЃћрЃЉрЃљ рЃЌрЃљрЃЋрЃўрЃЊрЃљрЃю |
| `'w+'` | рЃЕрЃљрЃгрЃћрЃарЃљ рЃЊрЃљ рЃЎрЃўрЃЌрЃ«рЃЋрЃљ | рЃЦрЃЏрЃюрЃўрЃА рЃљрЃю рЃерЃџрЃўрЃА рЃљрЃарЃАрЃћрЃЉрЃБрЃџрЃў рЃерЃўрЃњрЃЌрЃљрЃЋрЃАрЃА |
| `'a+'` | рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ рЃЊрЃљ рЃЎрЃўрЃЌрЃ«рЃЋрЃљ | рЃЎрЃўрЃЌрЃ«рЃЋрЃљ рЃўрЃгрЃДрЃћрЃЉрЃљ рЃЌрЃљрЃЋрЃўрЃЊрЃљрЃю, рЃЕрЃљрЃгрЃћрЃарЃљ Рђћ рЃЉрЃЮрЃџрЃЮрЃЊрЃљрЃю |

!!! warning "рЃДрЃБрЃарЃљрЃЊрЃдрЃћрЃЉрЃљ"
    рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃўрЃА рЃЊрЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ рЃАрЃљрЃГрЃўрЃарЃЮрЃљ рЃцрЃљрЃўрЃџрЃўрЃА рЃЊрЃљрЃ«рЃБрЃарЃЋрЃљ `close()`-рЃўрЃЌ, рЃарЃљрЃЌрЃљ рЃњрЃљрЃюрЃЌрЃљрЃЋрЃўрЃАрЃБрЃцрЃџрЃЊрЃћрЃА рЃарЃћрЃАрЃБрЃарЃАрЃћрЃЉрЃў рЃЊрЃљ рЃЊрЃљрЃўрЃгрЃДрЃЮрЃА рЃДрЃЋрЃћрЃџрЃљ рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃўрЃА рЃерЃћрЃюрЃљрЃ«рЃЋрЃљ.

!!! tip "рЃЊрЃљрЃБрЃЎрЃЋрЃўрЃарЃЊрЃў"
    рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ encoding="utf-8" рЃњрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ рЃЌрЃБ рЃцрЃљрЃўрЃџрЃерЃў рЃЕрЃљрЃгрЃћрЃа рЃљрЃю рЃгрЃљрЃўрЃЎрЃўрЃЌрЃ«рЃљрЃЋ рЃБрЃюрЃўрЃЎрЃЮрЃЊрЃўрЃарЃћрЃЉрЃљ рЃњрЃљрЃюрЃАрЃ«рЃЋрЃљрЃЋрЃћрЃЉрЃБрЃџ рЃбрЃћрЃЦрЃАрЃбрЃА.

    
#### рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў 1: рЃцрЃљрЃўрЃџрЃўрЃА рЃњрЃљрЃ«рЃАрЃюрЃљ рЃЊрЃљ рЃЊрЃљрЃ«рЃБрЃарЃЋрЃљ
```python
# рЃцрЃљрЃўрЃџрЃўрЃА рЃњрЃљрЃ«рЃАрЃюрЃљ рЃЕрЃљрЃгрЃћрЃарЃўрЃА рЃарЃћрЃЪрЃўрЃЏрЃерЃў
file = open("test_file.txt", "w")
# ... рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃў ...
file.close()
print("рЃцрЃљрЃўрЃџрЃў 'test_file.txt' рЃерЃћрЃўрЃЦрЃЏрЃюрЃљ рЃЊрЃљ рЃЊрЃљрЃўрЃ«рЃБрЃарЃљ.")
```

### 1.2. рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃљ: `with` рЃЮрЃърЃћрЃарЃљрЃбрЃЮрЃарЃў

`with` рЃЎрЃЮрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃљ рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃа рЃЊрЃљрЃ«рЃБрЃарЃЋрЃљрЃА, рЃЌрЃБрЃюрЃЊрЃљрЃф рЃЉрЃџрЃЮрЃЎрЃерЃў рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃЊрЃћрЃА рЃњрЃљрЃЏрЃЮрЃюрЃљрЃЎрЃџрЃўрЃАрЃў. рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃћрЃЉрЃБрЃџрЃўрЃљ Python-рЃерЃў.

#### рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў 2: `with` рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ
```python
with open("safe_file.txt", "w", encoding="utf-8") as f:
    f.write("рЃљрЃЏ рЃбрЃћрЃЦрЃАрЃбрЃА рЃЕрЃљрЃЋрЃгрЃћрЃарЃЌ рЃцрЃљрЃўрЃџрЃерЃў.\n")
# рЃцрЃљрЃўрЃџрЃў рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃЊрЃљрЃўрЃ«рЃБрЃарЃљ
print("рЃцрЃљрЃўрЃџрЃў рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃЊрЃљрЃўрЃ«рЃБрЃарЃљ.")
```

## 2. рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЕрЃљрЃгрЃћрЃарЃљ рЃцрЃљрЃўрЃџрЃерЃў

рЃАрЃљрЃБрЃЎрЃћрЃЌрЃћрЃАрЃЮ рЃЏрЃћрЃЌрЃЮрЃЊрЃћрЃЉрЃў:
- `write(string)` Рђћ рЃгрЃћрЃарЃА рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃА рЃЊрЃљ рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА рЃЕрЃљрЃгрЃћрЃарЃўрЃџрЃў рЃАрЃўрЃЏрЃЉрЃЮрЃџрЃЮрЃћрЃЉрЃўрЃА рЃарЃљрЃЮрЃЊрЃћрЃюрЃЮрЃЉрЃљрЃА.
- `writelines(list_of_strings)` Рђћ рЃгрЃћрЃарЃА рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃћрЃЉрЃўрЃА рЃАрЃўрЃљрЃА (рЃљрЃа рЃАрЃФрЃћрЃюрЃА `\n` рЃЌрЃљрЃЋрЃўрЃАрЃўрЃЌ).

#### рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў 3: `write()` рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ
```python
with open("data_write.txt", "w", encoding="utf-8") as f:
    f.write("рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃ«рЃљрЃќрЃў.\n")
    f.write("рЃЏрЃћрЃЮрЃарЃћ рЃ«рЃљрЃќрЃў.\n")
    print("рЃЏрЃћрЃАрЃљрЃЏрЃћ рЃ«рЃљрЃќрЃў (рЃЉрЃћрЃГрЃЊрЃЋрЃљ рЃцрЃљрЃўрЃџрЃерЃў)", file=f)
print("рЃЕрЃљрЃгрЃћрЃарЃљ рЃЊрЃљрЃАрЃарЃБрЃџрЃЊрЃљ.")
```
!!! note
    1. with open("data_write.txt", "w") as f: <br>
    open(...) РђЊ рЃ«рЃАрЃюрЃўрЃА рЃцрЃљрЃўрЃџрЃА рЃАрЃљрЃ«рЃћрЃџрЃўрЃЌ data_write.txt. <br>
    "w" РђЊ рЃюрЃўрЃерЃюрЃљрЃЋрЃА write modeРђЊрЃА (рЃЕрЃљрЃгрЃћрЃарЃљ). <br>
    рЃЌрЃБ рЃцрЃљрЃўрЃџрЃў рЃљрЃа рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА Рђћ рЃерЃћрЃЦрЃЏрЃюрЃўрЃА, рЃЌрЃБ рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА Рђћ рЃљрЃюрЃБрЃџрЃћрЃЉрЃА рЃљрЃарЃАрЃћрЃЉрЃБрЃџ рЃЎрЃЮрЃюрЃбрЃћрЃюрЃбрЃА. <br>
    with ... as f РђЊ рЃЦрЃЏрЃюрЃўрЃА рЃЎрЃЮрЃюрЃбрЃћрЃЦрЃАрЃбрЃўрЃА рЃЏрЃћрЃюрЃћрЃ»рЃћрЃарЃА: <br>
    рЃљрЃЏ рЃЉрЃџрЃЮрЃЎрЃўрЃА рЃЊрЃљрЃ«рЃБрЃарЃЋрЃўрЃАрЃљрЃА рЃцрЃљрЃўрЃџрЃў рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃЊрЃљрЃўрЃ«рЃБрЃарЃћрЃЉрЃљ. <br>
    рЃљрЃа рЃњрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ f.close(). <br>
    рЃљрЃЏ рЃ«рЃљрЃќрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ рЃњрЃЋрЃљрЃЦрЃЋрЃА рЃцрЃљрЃўрЃџрЃўрЃА рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃў f, рЃарЃЮрЃЏрЃћрЃџрЃерЃўрЃф рЃерЃћрЃњрЃЋрЃўрЃФрЃџрЃўрЃљ рЃЕрЃљрЃгрЃћрЃарЃљ. <br>

    2. f.write("рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃ«рЃљрЃќрЃў.\n")  <br>
    f.write() рЃгрЃћрЃарЃА рЃбрЃћрЃЦрЃАрЃбрЃА рЃцрЃљрЃўрЃџрЃерЃў. <br>
    "рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃ«рЃљрЃќрЃў.\n" <br>
    "рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃ«рЃљрЃќрЃў." Рђћ рЃЕрЃљрЃгрЃћрЃарЃўрЃџрЃў рЃбрЃћрЃЦрЃАрЃбрЃў <br>
    \n Рђћ рЃљрЃ«рЃљрЃџрЃў рЃ«рЃљрЃќрЃўрЃА рЃАрЃўрЃЏрЃЉрЃЮрЃџрЃЮ (Enter). <br>

    3. f.write("рЃЏрЃћрЃЮрЃарЃћ рЃ«рЃљрЃќрЃў.\n") <br>
    рЃљрЃюрЃљрЃџрЃЮрЃњрЃўрЃБрЃарЃљрЃЊ: рЃЏрЃћрЃЮрЃарЃћ рЃ«рЃљрЃќрЃў рЃўрЃгрЃћрЃарЃћрЃЉрЃљ рЃцрЃљрЃўрЃџрЃерЃў, рЃерЃћрЃЏрЃЊрЃћрЃњ Рђћ рЃљрЃ«рЃљрЃџрЃў рЃ«рЃљрЃќрЃў. <br>

    4. print("рЃЏрЃћрЃАрЃљрЃЏрЃћ рЃ«рЃљрЃќрЃў (рЃЉрЃћрЃГрЃЊрЃЋрЃљ рЃцрЃљрЃўрЃџрЃерЃў)", file=f) <br>
    рЃЕрЃЋрЃћрЃБрЃџрЃћрЃЉрЃарЃўрЃЋ, print() рЃЉрЃћрЃГрЃЊрЃљрЃЋрЃА рЃћрЃЎрЃарЃљрЃюрЃќрЃћ. <br>
    рЃЏрЃљрЃњрЃарЃљрЃЏ рЃарЃЮрЃфрЃљ рЃЋрЃБрЃЌрЃўрЃЌрЃћрЃЉрЃЌ file=f, рЃћрЃА рЃюрЃўрЃерЃюрЃљрЃЋрЃА: <br>
    рЃЉрЃћрЃГрЃЊрЃЋрЃљ рЃћрЃЎрЃарЃљрЃюрЃўрЃА рЃюрЃљрЃфрЃЋрЃџрЃљрЃЊ рЃЏрЃЮрЃ«рЃЊрЃћрЃЉрЃљ рЃцрЃљрЃўрЃџрЃерЃў. <br>
    рЃљрЃЏ рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў "рЃЏрЃћрЃАрЃљрЃЏрЃћ рЃ«рЃљрЃќрЃў (рЃЉрЃћрЃГрЃЊрЃЋрЃљ рЃцрЃљрЃўрЃџрЃерЃў)" рЃЕрЃљрЃўрЃгрЃћрЃарЃћрЃЉрЃљ рЃцрЃљрЃўрЃџрЃерЃў рЃарЃЮрЃњрЃЮрЃарЃф рЃбрЃћрЃЦрЃАрЃбрЃў рЃЊрЃљ рЃљрЃАрЃћрЃЋрЃћ рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃЊрЃљрЃћрЃЏрЃљрЃбрЃћрЃЉрЃљ рЃљрЃ«рЃљрЃџрЃў рЃ«рЃљрЃќрЃўрЃф. <br>

    5. print("рЃЕрЃљрЃгрЃћрЃарЃљ рЃЊрЃљрЃАрЃарЃБрЃџрЃЊрЃљ.") <br>
    рЃћрЃА print рЃћрЃЎрЃарЃљрЃюрЃќрЃћ рЃЉрЃћрЃГрЃЊрЃљрЃЋрЃА рЃбрЃћрЃЦрЃАрЃбрЃА, рЃарЃљрЃЊрЃњрЃљрЃю file=f рЃљрЃа рЃљрЃарЃўрЃА рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃБрЃџрЃў. <br>
    рЃцрЃљрЃўрЃџрЃА рЃљрЃарЃљрЃюрЃљрЃўрЃа рЃЎрЃљрЃЋрЃерЃўрЃарЃерЃў рЃљрЃа рЃљрЃарЃўрЃА. <br>
    ­Ъћџ рЃерЃћрЃЊрЃћрЃњрЃљрЃЊ рЃцрЃљрЃўрЃџрЃерЃў data_write.txt рЃЕрЃљрЃўрЃгрЃћрЃарЃћрЃЉрЃљ рЃљрЃАрЃћрЃЌрЃў рЃерЃўрЃюрЃљрЃљрЃарЃАрЃў: <br>
    рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃ«рЃљрЃќрЃў. <br>
    рЃЏрЃћрЃЮрЃарЃћ рЃ«рЃљрЃќрЃў. <br>
    рЃЏрЃћрЃАрЃљрЃЏрЃћ рЃ«рЃљрЃќрЃў (рЃЉрЃћрЃГрЃЊрЃЋрЃљ рЃцрЃљрЃўрЃџрЃерЃў) <br>
    рЃћрЃЎрЃарЃљрЃюрЃќрЃћ рЃЎрЃў рЃЊрЃљрЃўрЃЉрЃћрЃГрЃЊрЃћрЃЉрЃљ: <br>
    рЃЕрЃљрЃгрЃћрЃарЃљ рЃЊрЃљрЃАрЃарЃБрЃџрЃЊрЃљ. <br>
    

#### рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў 4: `writelines()` рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ
```python
lines = ["рЃ«рЃљрЃќрЃў 1\n", "рЃ«рЃљрЃќрЃў 2\n", "рЃ«рЃљрЃќрЃў 3\n"]
with open("data_writelines.txt", "w") as f:
    f.writelines(lines)
print("writelines рЃЊрЃљрЃАрЃарЃБрЃџрЃЊрЃљ.")
```

#### рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў 5: рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ (`'a'`)
```python
with open("data_append.txt", "w", encoding="utf-8") as f:
    f.write("рЃћрЃА рЃўрЃДрЃЮ рЃАрЃљрЃгрЃДрЃўрЃАрЃў рЃ«рЃљрЃќрЃў.\n")

with open("data_append.txt", "a", encoding="utf-8") as f:
    f.write("рЃћрЃА рЃЊрЃљрЃћрЃЏрЃљрЃбрЃљ рЃЏрЃЮрЃњрЃЋрЃўрЃљрЃюрЃћрЃЉрЃўрЃЌ.\n")

print("рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ рЃЊрЃљрЃАрЃарЃБрЃџрЃЊрЃљ.")
```

## 3. рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЎрЃўрЃЌрЃ«рЃЋрЃљ рЃцрЃљрЃўрЃџрЃўрЃЊрЃљрЃю

рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃЏрЃћрЃЌрЃЮрЃЊрЃћрЃЉрЃў:
- `read(size)` Рђћ рЃЎрЃўрЃЌрЃ«рЃБрЃџрЃЮрЃЉрЃА size рЃАрЃўрЃЏрЃЉрЃЮрЃџрЃЮрЃА; рЃЌрЃБ size рЃљрЃа рЃљрЃарЃўрЃА, рЃЎрЃўрЃЌрЃ«рЃБрЃџрЃЮрЃЉрЃА рЃЏрЃЌрЃћрЃџ рЃцрЃљрЃўрЃџрЃА. <br>
- `readline()` Рђћ рЃЎрЃўрЃЌрЃ«рЃБрЃџрЃЮрЃЉрЃА рЃћрЃарЃЌ рЃ«рЃљрЃќрЃА. <br>
- `readlines()` Рђћ рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА рЃДрЃЋрЃћрЃџрЃљ рЃ«рЃљрЃќрЃА рЃАрЃўрЃљрЃерЃў. <br>

#### рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў 6: `read()` Рђћ рЃЏрЃЌрЃћрЃџрЃў рЃерЃўрЃњрЃЌрЃљрЃЋрЃАрЃў
```python
with open("read_example.txt", "w") as f:
    f.write("рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃ«рЃљрЃќрЃў.\nрЃЏрЃћрЃЮрЃарЃћ рЃ«рЃљрЃќрЃў.\nрЃЏрЃћрЃАрЃљрЃЏрЃћ рЃ«рЃљрЃќрЃў.")

with open("read_example.txt", "r") as f:
    content = f.read()
    print("--- рЃЏрЃЌрЃћрЃџрЃў рЃцрЃљрЃўрЃџрЃў ---\n", content)
```

#### рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў 7: `readline()` Рђћ рЃЌрЃўрЃЌрЃЮ рЃ«рЃљрЃќрЃќрЃћ
```python
with open("read_example.txt", "r") as f:
    first_line = f.readline().strip()
    second_line = f.readline().strip()
    print(f"рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃ«рЃљрЃќрЃў: {first_line}")
    print(f"рЃЏрЃћрЃЮрЃарЃћ рЃ«рЃљрЃќрЃў: {second_line}")
```

#### рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў 8: `readlines()` Рђћ рЃАрЃўрЃўрЃА рЃАрЃљрЃ«рЃўрЃЌ
```python
with open("read_example.txt", "r") as f:
    lines_list = f.readlines()
    for line in lines_list:
        print(line.strip())
```

### 3.1. рЃцрЃљрЃўрЃџрЃўрЃА рЃўрЃбрЃћрЃарЃљрЃфрЃўрЃљ

рЃЊрЃўрЃЊрЃў рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃћрЃцрЃћрЃЦрЃбрЃБрЃарЃў рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ рЃцрЃљрЃўрЃџрЃўрЃА рЃърЃўрЃарЃЊрЃљрЃърЃўрЃарЃў рЃўрЃбрЃћрЃарЃљрЃфрЃўрЃљ Рђћ рЃЌрЃўрЃЌрЃЮ рЃ«рЃљрЃќрЃќрЃћ рЃЎрЃўрЃЌрЃ«рЃЋрЃљ.

#### рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў 9: рЃўрЃбрЃћрЃарЃљрЃфрЃўрЃљ
```python
with open("read_example.txt", "r") as f:
    for i, line in enumerate(f):
        print(f"рЃ«рЃљрЃќрЃў {i+1}: {line.strip()}")
```

## 4. рЃцрЃљрЃўрЃџрЃБрЃарЃў рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃЏрЃљрЃюрЃўрЃърЃБрЃџрЃљрЃфрЃўрЃљ: `os` рЃЊрЃљ `shutil`

рЃќрЃЮрЃњрЃљрЃЊрЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў:

| рЃЏрЃЮрЃЊрЃБрЃџрЃў | рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ | рЃљрЃдрЃгрЃћрЃарЃљ |
|---|---|---|
| os | `os.rename(old, new)` | рЃњрЃљрЃЊрЃљрЃарЃЦрЃЏрЃћрЃЋрЃљ |
| os | `os.remove(path)` | рЃцрЃљрЃўрЃџрЃўрЃА рЃгрЃљрЃерЃџрЃљ |
| os | `os.mkdir(path)` | рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ |
| os.path | `os.path.exists(path)` | рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА рЃЌрЃБ рЃљрЃарЃљ |
| shutil | `shutil.copy(src, dst)` | рЃЎрЃЮрЃърЃўрЃарЃћрЃЉрЃљ |
| shutil | `shutil.move(src, dst)` | рЃњрЃљрЃЊрЃљрЃбрЃљрЃюрЃљ / рЃњрЃљрЃЊрЃљрЃарЃЦрЃЏрЃћрЃЋрЃљ |
| shutil | `shutil.rmtree(path)` | рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃўрЃА рЃЊрЃљ рЃЏрЃўрЃА рЃерЃўрЃњрЃЌрЃљрЃЋрЃАрЃўрЃА рЃгрЃљрЃерЃџрЃљ |

#### рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў 10: рЃњрЃљрЃЊрЃљрЃарЃЦрЃЏрЃћрЃЋрЃљ
```python
import os

with open("old_name.txt", "w") as f:
    f.write("Hello")

os.rename("old_name.txt", "new_name.txt")
print("рЃцрЃљрЃўрЃџрЃА рЃЊрЃљрЃћрЃарЃЦрЃЋрЃљ рЃАрЃљрЃ«рЃћрЃџрЃў: new_name.txt")
```

#### рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў 11: рЃгрЃљрЃерЃџрЃљ
```python
import os

if os.path.exists("new_name.txt"):
    os.remove("new_name.txt")
    print("рЃцрЃљрЃўрЃџрЃў 'new_name.txt' рЃгрЃљрЃўрЃерЃљрЃџрЃљ.")
else:
    print("рЃцрЃљрЃўрЃџрЃў рЃљрЃа рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃЊрЃљ.")
```

## 5. рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃБрЃџрЃў рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃћрЃЉрЃў 


рЃерЃћрЃЦрЃЏрЃћрЃюрЃў рЃбрЃћрЃЦрЃАрЃбрЃБрЃарЃў рЃцрЃљрЃўрЃџрЃў рЃЊрЃљ рЃЕрЃљрЃгрЃћрЃарЃћ рЃерЃћрЃюрЃў рЃАрЃљрЃ«рЃћрЃџрЃў.

рЃгрЃљрЃўрЃЎрЃўрЃЌрЃ«рЃћ рЃцрЃљрЃўрЃџрЃўрЃА рЃерЃўрЃњрЃЌрЃљрЃЋрЃАрЃў рЃЊрЃљ рЃЊрЃљрЃЉрЃћрЃГрЃЊрЃћ рЃћрЃЎрЃарЃљрЃюрЃќрЃћ.

рЃЕрЃљрЃгрЃћрЃарЃћ 3 рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃў рЃЊрЃљ рЃерЃћрЃЏрЃЊрЃћрЃњ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљрЃерЃў рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃњрЃљрЃЏрЃЮрЃДрЃљрЃЋрЃў рЃфрЃљрЃџ-рЃфрЃљрЃџрЃЎрЃћ.

рЃЊрЃљрЃгрЃћрЃарЃћ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃЌрЃЋрЃџрЃўрЃА рЃцрЃљрЃўрЃџрЃерЃў рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃћрЃЉрЃўрЃА рЃарЃљрЃЮрЃЊрЃћрЃюрЃЮрЃЉрЃљрЃА.

рЃЊрЃљрЃгрЃћрЃарЃћ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃЌрЃЋрЃџрЃўрЃА рЃАрЃўрЃЏрЃЉрЃЮрЃџрЃЮрЃћрЃЉрЃўрЃА рЃарЃљрЃЮрЃЊрЃћрЃюрЃЮрЃЉрЃљрЃА.

рЃЊрЃљрЃљрЃЏрЃљрЃбрЃћ рЃбрЃћрЃЦрЃАрЃбрЃў рЃцрЃљрЃўрЃџрЃерЃў рЃўрЃАрЃћ, рЃарЃЮрЃЏ рЃљрЃарЃАрЃћрЃЉрЃБрЃџрЃў рЃљрЃа рЃгрЃљрЃўрЃерЃљрЃџрЃЮрЃА.

рЃЕрЃљрЃгрЃћрЃарЃћ list рЃцрЃљрЃўрЃџрЃерЃў writelines()-рЃўрЃЌ.

рЃгрЃљрЃўрЃЎрЃўрЃЌрЃ«рЃћ рЃцрЃљрЃўрЃџрЃў рЃЊрЃљ рЃЊрЃљрЃЉрЃћрЃГрЃЊрЃћ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃћрЃЉрЃў, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃерЃћрЃўрЃфрЃљрЃЋрЃА рЃАрЃўрЃбрЃДрЃЋрЃљрЃА РђъPythonРђю.

рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћ рЃцрЃљрЃўрЃџрЃўрЃА рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃљ рЃЊрЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃљрЃЏрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ рЃгрЃљрЃўрЃЎрЃўрЃЌрЃ«рЃћ.

рЃгрЃљрЃўрЃЎрЃўрЃЌрЃ«рЃћ рЃцрЃљрЃўрЃџрЃў рЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃДрЃљрЃЋрЃў рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃърЃўрЃарЃЋрЃћрЃџрЃў 5 рЃАрЃўрЃЏрЃЉрЃЮрЃџрЃЮ.


рЃЕрЃљрЃгрЃћрЃарЃћ рЃарЃўрЃфрЃ«рЃЋрЃћрЃЉрЃўрЃА рЃАрЃўрЃљ рЃцрЃљрЃўрЃџрЃерЃў, рЃерЃћрЃЏрЃЊрЃћрЃњ рЃгрЃљрЃўрЃЎрЃўрЃЌрЃ«рЃћ рЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃљрЃџрЃћ рЃ»рЃљрЃЏрЃў.

рЃцрЃљрЃўрЃџрЃерЃў рЃЕрЃљрЃгрЃћрЃарЃћ 10 рЃарЃўрЃфрЃ«рЃЋрЃў, рЃерЃћрЃЏрЃЊрЃћрЃњ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљрЃЏ рЃБрЃюрЃЊрЃљ рЃўрЃърЃЮрЃЋрЃЮрЃА рЃЏрЃљрЃЦрЃАрЃўрЃЏрЃБрЃЏрЃў.

рЃерЃћрЃЦрЃЏрЃћрЃюрЃў рЃЮрЃарЃў рЃцрЃљрЃўрЃџрЃў рЃЊрЃљ рЃЏрЃћрЃАрЃљрЃЏрЃћ рЃцрЃљрЃўрЃџрЃерЃў рЃЕрЃљрЃгрЃћрЃарЃћ рЃЏрЃљрЃЌрЃў рЃерЃўрЃњрЃЌрЃљрЃЋрЃАрЃўрЃА рЃњрЃљрЃћрЃарЃЌрЃўрЃљрЃюрЃћрЃЉрЃљ.

рЃњрЃљрЃЊрЃљрЃЎрЃЮрЃърЃўрЃарЃћ рЃцрЃљрЃўрЃџрЃў shutil.copy()-рЃўрЃЌ.

рЃЊрЃљрЃгрЃћрЃарЃћ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃцрЃљрЃўрЃџрЃерЃў рЃфрЃЋрЃџрЃўрЃА рЃАрЃўрЃбрЃДрЃЋрЃљ РђъcatРђю-рЃА рЃАрЃўрЃбрЃДрЃЋрЃўрЃЌ РђъdogРђю.

рЃгрЃљрЃўрЃЎрЃўрЃЌрЃ«рЃћ CSV рЃЊрЃљ рЃЊрЃљрЃЉрЃћрЃГрЃЊрЃћ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃљрЃАрЃљрЃЎрЃў.

CSV-рЃерЃў рЃЊрЃљрЃљрЃЏрЃљрЃбрЃћ рЃљрЃ«рЃљрЃџрЃў рЃЏрЃгрЃЎрЃарЃўрЃЋрЃў.

JSON рЃцрЃљрЃўрЃџрЃерЃў рЃерЃћрЃфрЃЋрЃљрЃџрЃћ рЃћрЃарЃЌрЃў value рЃЊрЃљ рЃЎрЃЋрЃџрЃљрЃЋ рЃЕрЃљрЃгрЃћрЃарЃћ.

рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљрЃЏ рЃцрЃљрЃўрЃџрЃерЃў рЃЕрЃљрЃгрЃћрЃарЃЮрЃА 100 рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃўрЃЌрЃў рЃарЃўрЃфрЃ«рЃЋрЃў.

рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљрЃЏ рЃгрЃљрЃўрЃЎрЃўрЃЌрЃ«рЃЮрЃА рЃћрЃА рЃарЃўрЃфрЃ«рЃЋрЃћрЃЉрЃў рЃЊрЃљ рЃЊрЃљрЃ«рЃљрЃарЃўрЃАрЃ«рЃћрЃЉрЃБрЃџрЃў рЃњрЃљрЃЏрЃЮрЃДрЃЮрЃА.


рЃАрЃљрЃгрЃДрЃўрЃАрЃў рЃюрЃљрЃгрЃўрЃџрЃў I Рђћ рЃАрЃљрЃЉрЃљрЃќрЃўрЃАрЃЮ рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃў (open, write, read, close, with) <br>
1. рЃерЃћрЃўрЃЦрЃЏрЃюрЃљрЃА `homework.txt` рЃЊрЃљ рЃЕрЃљрЃгрЃћрЃарЃЮрЃА рЃАрЃљрЃ«рЃћрЃџрЃў (`w`) Рђћ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ `with` рЃЊрЃљ `f.write()`. <br>
2. рЃгрЃљрЃўрЃЎрЃўрЃЌрЃ«рЃЮрЃА рЃЏрЃЌрЃћрЃџрЃў рЃерЃўрЃњрЃЌрЃљрЃЋрЃАрЃў рЃЊрЃљ рЃЊрЃљрЃЉрЃћрЃГрЃЊрЃЮрЃА (`f.read()`). <br>
3. рЃЊрЃљрЃљрЃЏрЃљрЃбрЃЮрЃА рЃњрЃЋрЃљрЃарЃў рЃљрЃ«рЃљрЃџ рЃ«рЃљрЃќрЃќрЃћ (`a`). <br>
4. рЃЕрЃљрЃЏрЃЮрЃљрЃДрЃљрЃџрЃўрЃЉрЃЮрЃА 5 рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃўрЃљрЃюрЃў рЃАрЃўрЃљ рЃЊрЃљ рЃЕрЃљрЃгрЃћрЃарЃЮрЃА `list_data.txt` `writelines()`-рЃўрЃЌ (рЃ«рЃћрЃЊрЃўрЃЌ `\n`). <br>
5. рЃгрЃљрЃўрЃЎрЃўрЃЌрЃ«рЃЮрЃА `list_data.txt` readline()-рЃўрЃЌ 3-рЃ»рЃћрЃа рЃЊрЃљ рЃЊрЃљрЃЉрЃћрЃГрЃЊрЃЮрЃА. <br>
6. рЃгрЃљрЃўрЃЎрЃўрЃЌрЃ«рЃЮрЃА `list_data.txt` `readlines()`-рЃўрЃЌ рЃЊрЃљ рЃЊрЃљрЃЉрЃћрЃГрЃЊрЃЮрЃА рЃАрЃўрЃљ (`.strip()`). <br>
7. рЃерЃћрЃЦрЃЏрЃюрЃљрЃА `numbers.txt` рЃЊрЃљ рЃЕрЃљрЃгрЃћрЃарЃЮрЃА 1РђЊ10, рЃЌрЃўрЃЌрЃЮ рЃарЃўрЃфрЃ«рЃЋрЃў рЃљрЃ«рЃљрЃџ рЃ«рЃљрЃќрЃќрЃћ. <br>
8. рЃгрЃљрЃўрЃЎрЃўрЃЌрЃ«рЃЮрЃА `numbers.txt` рЃЊрЃљ рЃЊрЃљрЃЉрЃћрЃГрЃЊрЃЮрЃА рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃџрЃБрЃгрЃў рЃарЃўрЃфрЃ«рЃЋрЃћрЃЉрЃў (рЃцрЃљрЃўрЃџрЃўрЃА рЃўрЃбрЃћрЃарЃљрЃфрЃўрЃљ). <br>
9. рЃЊрЃљрЃгрЃћрЃарЃЮрЃА рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ `safe_write(filename, content)` рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃДрЃћрЃюрЃћрЃЉрЃА `with open(..., 'w')`. <br>
10. рЃЊрЃљрЃгрЃћрЃарЃЮрЃА рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ `read_and_print(filename)` Рђћ рЃљрЃфрЃюрЃЮрЃЉрЃЮрЃА `FileNotFoundError`. <br>

рЃЎрЃўрЃЌрЃ«рЃЋрЃљ/рЃЕрЃљрЃгрЃћрЃарЃљ, рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃљ <br>
11. рЃерЃћрЃЦрЃЏрЃюрЃљрЃА `scores.txt` (рЃАрЃљрЃ«рЃћрЃџрЃў,рЃЦрЃБрЃџрЃљ) Рђћ 5 рЃЕрЃљрЃюрЃљрЃгрЃћрЃарЃў. <br>
12. рЃгрЃљрЃўрЃЎрЃўрЃЌрЃ«рЃЮрЃА `scores.txt` рЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃЌрЃЋрЃљрЃџрЃЮрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃЮ рЃЦрЃБрЃџрЃљ (`.split(',')`, `int()`). <br>
13. рЃерЃћрЃЦрЃЏрЃюрЃљрЃА `passed.txt` Рђћ рЃЕрЃљрЃўрЃгрЃћрЃарЃЮрЃА рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃЦрЃБрЃџрЃљ > 70. <br>
14. рЃЕрЃљрЃгрЃћрЃарЃЮрЃА `log.txt` рЃЏрЃўрЃЏрЃЊрЃўрЃюрЃљрЃарЃћ рЃЊрЃарЃЮ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃўрЃАрЃљрЃА (`datetime.datetime.now()`). <br>
15. рЃЊрЃљрЃгрЃћрЃарЃЮрЃА `replace_text(filename, old_word, new_word)` (рЃЌрЃћрЃЏрЃърЃЮрЃарЃљрЃџрЃБрЃарЃў рЃцрЃљрЃўрЃџрЃў рЃљрЃю рЃЏрЃЌрЃџрЃўрЃљрЃюрЃљрЃЊ рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃљрЃерЃў). <br>
16. рЃерЃћрЃЦрЃЏрЃюрЃљрЃА `settings.ini` рЃЊрЃљ рЃЕрЃљрЃгрЃћрЃарЃЮрЃА рЃЎрЃЮрЃюрЃцрЃўрЃњрЃў (рЃЏрЃљрЃњ. `username=admin\nmax_attempts=5`) рЃЊрЃљ рЃгрЃљрЃўрЃЎрЃўрЃЌрЃ«рЃЮрЃА рЃџрЃћрЃЦрЃАрЃўрЃЎрЃЮрЃюрЃерЃў. <br>
17. рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ: рЃўрЃюрЃАрЃбрЃарЃБрЃЏрЃћрЃюрЃбрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ рЃЏрЃўрЃўрЃдрЃЮрЃА рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃћрЃЉрЃў рЃЊрЃљ рЃерЃћрЃўрЃюрЃљрЃ«рЃЮрЃА `user_input.txt` рЃЋрЃўрЃЊрЃарЃћ рЃљрЃа рЃерЃћрЃўрЃДрЃЋрЃљрЃюрЃћрЃЉрЃА "рЃЊрЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃљ". <br>
18. рЃгрЃљрЃўрЃЎрЃўрЃЌрЃ«рЃЮрЃА `user_input.txt` рЃЊрЃљ рЃЊрЃљрЃЌрЃЋрЃљрЃџрЃЮрЃА рЃ«рЃљрЃќрЃћрЃЉрЃўрЃА рЃарЃљрЃЮрЃЊрЃћрЃюрЃЮрЃЉрЃљ. <br>
19. рЃерЃћрЃЦрЃЏрЃюрЃљрЃА рЃфрЃљрЃарЃўрЃћрЃџрЃў рЃцрЃљрЃўрЃџрЃў `empty.txt`. <br>
20. рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃЮрЃА, рЃарЃЮрЃЏ `f.read()` рЃЉрЃарЃБрЃюрЃћрЃЉрЃА '' (рЃфрЃљрЃарЃўрЃћрЃџрЃў рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃў). <br>

рЃцрЃљрЃўрЃџрЃБрЃарЃў рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃЏрЃљрЃюрЃўрЃърЃБрЃџрЃљрЃфрЃўрЃљ (`os`, `shutil`) <br>
21. рЃерЃћрЃЦрЃЏрЃюрЃљрЃА рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљ `temp_files` (`os.mkdir()`). <br>
22. рЃерЃћрЃЦрЃЏрЃюрЃљрЃА `temp_files/draft.txt`. <br>
23. рЃњрЃљрЃЊрЃљрЃарЃЦрЃЋрЃљрЃА `draft.txt` Рєњ `final.txt` (`os.rename()`). <br>
24. рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃЮрЃА `os.path.exists('temp_files/final.txt')`. <br>
25. рЃЊрЃљрЃљрЃЎрЃЮрЃърЃўрЃарЃЮрЃА `final.txt` рЃарЃЮрЃњрЃЮрЃарЃф `backup.txt` (`shutil.copy()`). <br>
26. рЃњрЃљрЃЊрЃљрЃўрЃюрЃљрЃфрЃЋрЃџрЃЮрЃА `backup.txt` рЃћрЃарЃЌрЃў рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃўрЃЌ рЃБрЃЎрЃљрЃю (`shutil.move()`). <br>
27. рЃгрЃљрЃерЃљрЃџрЃЮрЃА `backup.txt` (`os.remove()`). <br>
28. рЃгрЃљрЃерЃљрЃџрЃЮрЃА `temp_files/final.txt`. <br>

## 6. рЃњрЃљрЃюрЃ«рЃўрЃџрЃЋрЃљ / рЃЊрЃўрЃАрЃЎрЃБрЃАрЃўрЃљ

- рЃарЃљрЃбрЃЮрЃЏ рЃљрЃарЃўрЃА `with open(...)` рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃћрЃЉрЃБрЃџрЃў?
  - `with` рЃЦрЃЏрЃюрЃўрЃА рЃЎрЃЮрЃюрЃбрЃћрЃЦрЃАрЃбрЃўрЃА рЃЏрЃћрЃюрЃћрЃ»рЃћрЃарЃА, рЃарЃЮрЃЏрЃџрЃўрЃА `__exit__` рЃЏрЃћрЃЌрЃЮрЃЊрЃў рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃА рЃўрЃарЃЕрЃћрЃЋрЃљ рЃЉрЃџрЃЮрЃЎрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ рЃЊрЃљ рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃ«рЃБрЃарЃљрЃЋрЃА рЃцрЃљрЃўрЃџрЃА Рђћ рЃњрЃљрЃарЃљрЃюрЃбрЃўрЃарЃћрЃЉрЃА рЃарЃћрЃАрЃБрЃарЃАрЃћрЃЉрЃўрЃА рЃњрЃљрЃЌрЃљрЃЋрЃўрЃАрЃБрЃцрЃџрЃћрЃЉрЃљрЃАрЃљ рЃЊрЃљ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЏрЃЌрЃџрЃўрЃљрЃюрЃЮрЃЉрЃљрЃА, рЃЌрЃБрЃюрЃЊрЃљрЃф рЃгрЃљрЃарЃБрЃЏрЃљрЃбрЃћрЃЉрЃџрЃЮрЃЉрЃўрЃА рЃЊрЃарЃЮрЃА.

- рЃарЃљ рЃАрЃљрЃцрЃарЃЌрЃ«рЃћрЃћрЃЉрЃўрЃЌ рЃљрЃдрЃўрЃЦрЃЏрЃћрЃЉрЃљ `'w'` (Write) рЃарЃћрЃЪрЃўрЃЏрЃў?
  - `'w'` рЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃА рЃљрЃю рЃЦрЃЏрЃюрЃўрЃА рЃцрЃљрЃўрЃџрЃА рЃЊрЃљ *рЃерЃџрЃўрЃА* рЃљрЃарЃАрЃћрЃЉрЃБрЃџ рЃерЃўрЃњрЃЌрЃљрЃЋрЃАрЃА Рђћ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ рЃБрЃЎрЃўрЃЊрЃБрЃарЃћрЃАрЃљрЃЊ рЃцрЃарЃЌрЃ«рЃўрЃџрЃљрЃЊ; рЃЌрЃБ рЃАрЃљрЃГрЃўрЃарЃЮрЃљ рЃерЃћрЃюрЃљрЃарЃЕрЃБрЃюрЃћрЃЉрЃљ, рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ `'a'` рЃљрЃю рЃгрЃљрЃўрЃЎрЃўрЃЌрЃ«рЃћрЃЌ рЃЊрЃљ рЃерЃћрЃЏрЃЊрЃћрЃњ рЃЕрЃљрЃўрЃгрЃћрЃарЃћрЃЌ рЃ«рЃћрЃџрЃљрЃ«рЃџрЃљ.

- рЃарЃљ рЃњрЃљрЃюрЃАрЃ«рЃЋрЃљрЃЋрЃћрЃЉрЃљрЃљ `read()`, `readline()` рЃЊрЃљ `readlines()` рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃЌрЃЋрЃљрЃџрЃАрЃљрЃќрЃарЃўрЃАрЃўрЃЌ?
  - `read()` рЃЊрЃљ `readlines()` рЃўрЃбрЃЋрЃўрЃарЃЌрЃўрЃљрЃю рЃЏрЃЌрЃћрЃџ рЃцрЃљрЃўрЃџрЃА рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃљрЃерЃў (рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃўрЃА рЃљрЃю рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃћрЃЉрЃўрЃА рЃАрЃљрЃ«рЃўрЃЌ) Рђћ рЃљрЃарЃљрЃћрЃцрЃћрЃЦрЃбрЃБрЃарЃўрЃљ рЃЊрЃўрЃЊрЃў рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.
  - рЃўрЃбрЃћрЃарЃљрЃфрЃўрЃљ рЃцрЃљрЃўрЃџрЃќрЃћ рЃљрЃю `readline()` рЃЎрЃўрЃЌрЃ«рЃБрЃџрЃЮрЃЉрЃА рЃ«рЃљрЃќ-рЃ«рЃљрЃќрЃќрЃћ рЃЊрЃљ рЃўрЃюрЃљрЃ«рЃљрЃЋрЃА рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃљрЃерЃў рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃЌрЃљрЃюрЃЏрЃўрЃЏрЃЊрЃћрЃЋрЃарЃБрЃџрЃљрЃЊ рЃАрЃљрЃГрЃўрЃарЃЮ рЃюрЃљрЃгрЃўрЃџрЃА Рђћ рЃАрЃљрЃБрЃЎрЃћрЃЌрЃћрЃАрЃЮ рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃљ рЃЊрЃўрЃЊ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃЌрЃљрЃю.
