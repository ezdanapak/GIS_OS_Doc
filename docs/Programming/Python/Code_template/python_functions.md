# ­ЪДЕ рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў Python-рЃерЃў

рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ рЃљрЃарЃўрЃА рЃЎрЃЮрЃЊрЃўрЃА рЃЉрЃџрЃЮрЃЎрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃА рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџ рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљрЃА.  
рЃўрЃА рЃўрЃФрЃљрЃ«рЃћрЃЉрЃљ рЃАрЃљрЃГрЃўрЃарЃЮрЃћрЃЉрЃўрЃАрЃљрЃЏрЃћрЃЉрЃа рЃЊрЃљ рЃљрЃЏрЃфрЃўрЃарЃћрЃЉрЃА рЃЎрЃЮрЃЊрЃўрЃА рЃњрЃљрЃЏрЃћрЃЮрЃарЃћрЃЉрЃљрЃА.

---

## ­Ъћ╣ рЃарЃљрЃбрЃЮрЃЏ рЃЋрЃўрЃДрЃћрЃюрЃћрЃЉрЃЌ рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃА?
- рЃЎрЃЮрЃЊрЃў рЃ«рЃЊрЃћрЃЉрЃљ рЃБрЃцрЃарЃЮ рЃЏрЃЮрЃЎрЃџрЃћ рЃЊрЃљ рЃњрЃљрЃАрЃљрЃњрЃћрЃЉрЃў  
- рЃљрЃЊрЃЋрЃўрЃџрЃўрЃљ рЃерЃћрЃфрЃЋрЃџрЃљ рЃЊрЃљ рЃњрЃљрЃЏрЃљрЃарЃЌрЃЋрЃљ  
- рЃерЃћрЃњрЃЋрЃўрЃФрЃџрЃўрЃљ рЃћрЃарЃЌрЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ рЃЏрЃарЃљрЃЋрЃљрЃџрЃ»рЃћрЃа рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃЮрЃЌ рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃќрЃћ  

---

## ­Ъћ╣ рЃцрЃБрЃюрЃЦрЃфрЃўрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ (`def` рЃАрЃљрЃЎрЃЋрЃљрЃюрЃФрЃЮ рЃАрЃўрЃбрЃДрЃЋрЃўрЃЌ)

рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ рЃўрЃЦрЃЏрЃюрЃћрЃЉрЃљ рЃАрЃљрЃЎрЃЋрЃљрЃюрЃФрЃЮ рЃАрЃўрЃбрЃДрЃЋрЃўрЃЌ `def` рЃЊрЃљ рЃЏрЃўрЃАрЃў рЃАрЃљрЃ«рЃћрЃџрЃўрЃЌ.  
рЃерЃўрЃњрЃюрЃўрЃЌ рЃгрЃћрЃарЃўрЃљ рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃљ, рЃ«рЃЮрЃџрЃЮ рЃАрЃБрЃарЃЋрЃўрЃџрЃўрЃАрЃљрЃЏрЃћрЃЉрЃа Рђћ рЃЊрЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃўрЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ `return`.

## ­Ъћ╣ рЃБрЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃЮ рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ | рЃцрЃБрЃюрЃЦрЃфрЃўрЃўрЃА рЃњрЃљрЃЏрЃЮрЃФрЃљрЃ«рЃћрЃЉрЃљ

```python
def greet():
    print("рЃњрЃљрЃЏрЃљрЃарЃ»рЃЮрЃЉрЃљ!")
greet()
```
­ЪЊў рЃерЃћрЃЊрЃћрЃњрЃў:
рЃњрЃљрЃЏрЃљрЃарЃ»рЃЮрЃЉрЃљ!


## ­Ъћ╣ рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃўрЃљрЃюрЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ | рЃцрЃБрЃюрЃЦрЃфрЃўрЃўрЃА рЃњрЃљрЃЏрЃЮрЃФрЃљрЃ«рЃћрЃЉрЃљ
рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў Рђћ рЃћрЃА рЃљрЃарЃўрЃА рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃў, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃАрЃљрЃф рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ рЃўрЃдрЃћрЃЉрЃА, рЃарЃЮрЃфрЃљ рЃЏрЃљрЃА рЃЋрЃўрЃФрЃљрЃ«рЃћрЃЉрЃЌ.

```python

def greet(name):
    print("рЃњрЃљрЃЏрЃљрЃарЃ»рЃЮрЃЉрЃљ,", name, "!")
    
greet("рЃњрЃўрЃЮрЃарЃњрЃў")
```
­ЪЊў рЃерЃћрЃЊрЃћрЃњрЃў:
рЃњрЃљрЃЏрЃљрЃарЃ»рЃЮрЃЉрЃљ, рЃњрЃўрЃЮрЃарЃњрЃў !

## ­Ъћ╣ рЃњрЃљрЃЕрЃБрЃЏрЃћрЃЉрЃўрЃЌ (рЃюрЃљрЃњрЃБрЃџрЃўрЃАрЃ«рЃЏрЃћрЃЋрЃў) рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃў
рЃЌрЃБ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃў рЃљрЃа рЃњрЃљрЃЊрЃљрЃАрЃфрЃћрЃЏрЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљрЃА, рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃА рЃюрЃљрЃњрЃБрЃџрЃўрЃАрЃ«рЃЏрЃћрЃЋрЃА.

```python
def greet(name="рЃњрЃўрЃЮрЃарЃњрЃў"):
    print("рЃњрЃљрЃЏрЃљрЃарЃ»рЃЮрЃЉрЃљ,", name, "!")

greet()
greet("рЃюрЃўрЃюрЃЮ")
```
­ЪЊў рЃерЃћрЃЊрЃћрЃњрЃў:

рЃњрЃљрЃЏрЃљрЃарЃ»рЃЮрЃЉрЃљ, рЃњрЃўрЃЮрЃарЃњрЃў ! <br>
рЃњрЃљрЃЏрЃљрЃарЃ»рЃЮрЃЉрЃљ, рЃюрЃўрЃюрЃЮ !

## ­Ъћ╣ рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљрЃА (return)
```python

def square(x):
    return x * x

result = square(5)
print("рЃарЃўрЃфрЃ«рЃЋрЃўрЃА рЃЎрЃЋрЃљрЃЊрЃарЃљрЃбрЃў рЃљрЃарЃўрЃА:", result)
```
­ЪЊў рЃерЃћрЃЊрЃћрЃњрЃў:
рЃарЃўрЃфрЃ«рЃЋрЃўрЃА рЃЎрЃЋрЃљрЃЊрЃарЃљрЃбрЃў рЃљрЃарЃўрЃА: 25

## ­Ъћ╣ рЃљрЃюрЃЮрЃюрЃўрЃЏрЃБрЃарЃў (lambda) рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ
рЃЌрЃБ рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ рЃФрЃљрЃџрЃўрЃљрЃю рЃЏрЃЮрЃЎрЃџрЃћрЃљ, рЃерЃћрЃњрЃЋрЃўрЃФрЃџрЃўрЃљ рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃЮрЃЌ lambda.

```python
square = lambda x: x * x
print("4 рЃўрЃА рЃЎрЃЋрЃљрЃЊрЃарЃљрЃбрЃўрЃљ:", square(4))
```
­ЪЊў рЃерЃћрЃЊрЃћрЃњрЃў:
4 рЃўрЃА рЃЎрЃЋрЃљрЃЊрЃарЃљрЃбрЃўрЃљ: 16

## ­ЪДа рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃБрЃџрЃў рЃљрЃЏрЃЮрЃфрЃљрЃюрЃћрЃЉрЃў

- рЃЊрЃљрЃгрЃћрЃарЃћ рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ hello_world(), рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЊрЃљрЃЉрЃћрЃГрЃЊрЃљрЃЋрЃА рЃбрЃћрЃЦрЃАрЃбрЃА "Hello, World!" <br>

- рЃерЃћрЃЦрЃЏрЃћрЃюрЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ add(a, b), рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЊрЃљрЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА рЃЮрЃарЃў рЃарЃўрЃфрЃ«рЃЋрЃўрЃА рЃ»рЃљрЃЏрЃА. <br>

- рЃерЃћрЃфрЃЋрЃљрЃџрЃћ рЃгрЃўрЃюрЃљ рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ рЃўрЃАрЃћ, рЃарЃЮрЃЏ рЃЌрЃБ рЃЏрЃћрЃЮрЃарЃћ рЃарЃўрЃфрЃ«рЃЋрЃў рЃљрЃа рЃљрЃарЃўрЃА рЃњрЃљрЃЊрЃљрЃфрЃћрЃЏрЃБрЃџрЃў, рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ рЃўрЃДрЃЮрЃА 0. <br>

- рЃЊрЃљрЃгрЃћрЃарЃћ рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ is_even(n), рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА True, рЃЌрЃБ рЃарЃўрЃфрЃ«рЃЋрЃў рЃџрЃБрЃгрЃўрЃљ. <br>

- рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћ lambda, рЃарЃљрЃЌрЃљ рЃерЃћрЃЦрЃЏрЃюрЃљ рЃЏрЃЮрЃЎрЃџрЃћ рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА рЃарЃўрЃфрЃ«рЃЋрЃўрЃА рЃАрЃљрЃЏрЃЏрЃљрЃњрЃА. <br>

- рЃерЃћрЃЦрЃЏрЃћрЃюрЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ greet_user(name, lang="ka"), рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЊрЃљрЃЉрЃћрЃГрЃЊрЃљрЃЋрЃА: <br>

"рЃњрЃљрЃЏрЃљрЃарЃ»рЃЮрЃЉрЃљ" рЃЌрЃБ lang="ka" <br>

"Hello" рЃЌрЃБ lang="en" <br>

- рЃЊрЃљрЃгрЃћрЃарЃћ рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЏрЃўрЃўрЃдрЃћрЃЉрЃА рЃАрЃўрЃљрЃА рЃЊрЃљ рЃЊрЃљрЃЉрЃћрЃГрЃЊрЃљрЃЋрЃА рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃџрЃБрЃг рЃарЃўрЃфрЃ«рЃЋрЃћрЃЉрЃА. <br>


---


- рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ 1: рЃцрЃљрЃЦрЃбрЃЮрЃарЃўрЃљрЃџрЃў

рЃЊрЃљрЃгрЃћрЃарЃћрЃЌ рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ factorial(n), рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃЌрЃЋрЃџрЃўрЃА n-рЃўрЃА рЃцрЃљрЃЦрЃбрЃЮрЃарЃўрЃљрЃџрЃА (n! = 1 * 2 * ... * n).
рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў: factorial(5) Рєњ 120
рЃцрЃБрЃюрЃЦрЃфрЃўрЃљрЃЏ рЃБрЃюрЃЊрЃљ рЃЏрЃўрЃўрЃдрЃЮрЃА рЃћрЃарЃЌрЃў рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃў n.


- рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ 2: рЃЏрЃўрЃАрЃљрЃџрЃЏрЃћрЃЉрЃљ рЃњрЃљрЃЕрЃБрЃЏрЃћрЃЉрЃўрЃЌ рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃўрЃЌ

рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ say_hello(name="Guest"), рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЉрЃћрЃГрЃЊрЃљрЃЋрЃА РђъHello, [name]!Рђю. рЃЌрЃБ рЃАрЃљрЃ«рЃћрЃџрЃў рЃљрЃа рЃњрЃљрЃЊрЃљрЃћрЃфрЃћрЃЏрЃљ, рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮрЃА РђъGuestРђю.


- рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ 3: рЃЎрЃЋрЃљрЃЊрЃарЃљрЃбрЃћрЃЉрЃўрЃА рЃ»рЃљрЃЏрЃў

рЃЊрЃљрЃгрЃћрЃарЃћрЃЌ рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ sum_of_squares(n), рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃЌрЃЋрЃџрЃўрЃА 1-рЃЊрЃљрЃю n-рЃЏрЃЊрЃћ рЃарЃўрЃфрЃ«рЃЋрЃћрЃЉрЃўрЃА рЃЎрЃЋрЃљрЃЊрЃарЃљрЃбрЃћрЃЉрЃўрЃА рЃ»рЃљрЃЏрЃА (1┬▓ + 2┬▓ + ... + n┬▓).
рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў: sum_of_squares(3) Рєњ 14 (1┬▓ + 2┬▓ + 3┬▓ = 1 + 4 + 9 = 14).


- рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ 4: Lambda рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ

рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ lambda рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃЌрЃЋрЃџрЃўрЃА рЃЏрЃЮрЃфрЃћрЃЏрЃБрЃџрЃў рЃарЃўрЃфрЃ«рЃЋрЃўрЃА рЃЎрЃБрЃЉрЃА (x┬│).
рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў: (lambda x: x**3)(4) Рєњ 64.

!!! tip "рЃарЃЕрЃћрЃЋрЃљ"
    рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃўрЃА рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃўрЃАрЃљрЃА рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ def рЃАрЃљрЃЎрЃЋрЃљрЃюрЃФрЃЮ рЃАрЃўрЃбрЃДрЃЋрЃљ. <br>

    Lambda рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА. <br>
    рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃА рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ рЃерЃћрЃДрЃЋрЃљрЃюрЃўрЃџрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЋрЃљрЃџрЃўрЃЊрЃБрЃарЃЮрЃЉрЃљ (рЃЏрЃљрЃњ., рЃБрЃљрЃарЃДрЃЮрЃцрЃўрЃЌрЃў рЃарЃўрЃфрЃ«рЃЋрЃћрЃЉрЃў factorial-рЃўрЃАрЃЌрЃЋрЃўрЃА). <br>
