# ­ЪљЇ Python String Formatting  
рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃћрЃЉрЃўрЃА рЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃарЃћрЃЉрЃљ Python-рЃерЃў

рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃћрЃЉрЃўрЃА рЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃарЃћрЃЉрЃљ рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃњрЃЋрЃљрЃФрЃџрЃћрЃЋрЃА **рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў рЃЏрЃЮрЃЋрЃљрЃЌрЃљрЃЋрЃАрЃЮрЃЌ рЃбрЃћрЃЦрЃАрЃбрЃерЃў рЃЏрЃЮрЃЦрЃюрЃўрЃџрЃљрЃЊ рЃЊрЃљ рЃЮрЃарЃњрЃљрЃюрЃўрЃќрЃћрЃЉрЃБрЃџрЃљрЃЊ**.  
рЃљрЃЏ рЃњрЃќрЃўрЃЌ рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ рЃЋрЃЉрЃћрЃГрЃЊрЃљрЃЋрЃЌ рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃА, рЃарЃўрЃфрЃ«рЃЋрЃћрЃЉрЃА рЃљрЃю рЃАрЃ«рЃЋрЃљ рЃбрЃўрЃърЃўрЃА рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљрЃА рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃерЃў.

---

!!! warning
    рЃАрЃљрЃџрЃћрЃЦрЃфрЃўрЃЮ рЃЎрЃБрЃарЃАрЃўрЃА рЃцрЃљрЃарЃњрЃџрЃћрЃЉрЃерЃў рЃљрЃа рЃљрЃарЃўрЃА рЃЏрЃўрЃАрЃў рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ рЃАрЃљрЃЋрЃљрЃџрЃЊрЃћрЃЉрЃБрЃџрЃЮ!

## ­Ъћ╣ 1. рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃўрЃА рЃерЃћрЃћрЃарЃЌрЃћрЃЉрЃљ (Concatenation)
рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃњрЃќрЃљ рЃљрЃарЃўрЃА `+` рЃЮрЃърЃћрЃарЃљрЃбрЃЮрЃарЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ.

```python
name = "Giorgi"
age = 25
print("My name is " + name + " and I am " + str(age) + " years old.")
```

­ЪДЕ **рЃерЃћрЃЊрЃћрЃњрЃў:**
```
My name is Giorgi and I am 25 years old.
```

> РЮЌрЃерЃћрЃюрЃўрЃерЃЋрЃюрЃљ: рЃарЃўрЃфрЃ«рЃЋрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў рЃАрЃљрЃГрЃўрЃарЃЮрЃљ `str()` рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ, рЃарЃљрЃЊрЃњрЃљрЃю рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃбрЃўрЃърЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃА рЃЋрЃћрЃа рЃЋрЃљрЃћрЃарЃЌрЃћрЃЉрЃЌ рЃърЃўрЃарЃЊрЃљрЃърЃўрЃа.

---

## ­Ъћ╣ 2. %-рЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃарЃћрЃЉрЃљ (Old Style Formatting)
рЃћрЃА рЃљрЃарЃўрЃА рЃФрЃЋрЃћрЃџрЃў, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃ»рЃћрЃа рЃЎрЃўрЃЊрЃћрЃЋ рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃў рЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃарЃћрЃЉрЃўрЃА рЃЏрЃћрЃЌрЃЮрЃЊрЃў.

```python
name = "Giorgi"
age = 25
print("My name is %s and I am %d years old." % (name, age))
```

­ЪДЕ **рЃерЃћрЃЊрЃћрЃњрЃў:**
```
My name is Giorgi and I am 25 years old.
```

­ЪЊў `%s` Рђћ рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃў  
­ЪЊў `%d` Рђћ рЃЏрЃЌрЃћрЃџрЃў рЃарЃўрЃфрЃ«рЃЋрЃў  
­ЪЊў `%f` Рђћ рЃљрЃЌрЃгрЃўрЃџрЃљрЃЊрЃў рЃарЃўрЃфрЃ«рЃЋрЃў  

---

## ­Ъћ╣ 3. `str.format()` рЃЏрЃћрЃЌрЃЮрЃЊрЃў
рЃЌрЃљрЃюрЃљрЃЏрЃћрЃЊрЃарЃЮрЃЋрЃћ рЃЊрЃљ рЃБрЃюрЃўрЃЋрЃћрЃарЃАрЃљрЃџрЃБрЃарЃў рЃњрЃќрЃљ.

```python
name = "Giorgi"
age = 25
print("My name is {} and I am {} years old.".format(name, age))
```

­ЪДЕ **рЃерЃћрЃЊрЃћрЃњрЃў:**
```
My name is Giorgi and I am 25 years old.
```

рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљ рЃўрЃюрЃЊрЃћрЃЦрЃАрЃћрЃЉрЃўрЃА рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃљрЃф:

```python
print("Name: {0}, Age: {1}".format("Giorgi", 25))
```

рЃљрЃю рЃАрЃљрЃ«рЃћрЃџрЃћрЃЉрЃўрЃЌ:

```python
print("Name: {n}, Age: {a}".format(n="Giorgi", a=25))
```

## ­Ъћ╣ 4. f-рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃћрЃЉрЃў (f-strings) Рђћ ­ЪЈє рЃЌрЃљрЃюрЃљрЃЏрЃћрЃЊрЃарЃЮрЃЋрЃћ рЃњрЃќрЃљ
Python 3.6-рЃЊрЃљрЃю рЃЏрЃЮрЃДрЃЮрЃџрЃћрЃЉрЃБрЃџрЃў рЃАрЃљрЃБрЃЎрЃћрЃЌрЃћрЃАрЃЮ рЃЊрЃљ рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃарЃћрЃЉрЃўрЃА рЃЏрЃћрЃЌрЃЮрЃЊрЃў.

```python
name = "Giorgi"
age = 25
print(f"My name is {name} and I am {age} years old.")
```

­ЪДЕ **рЃерЃћрЃЊрЃћрЃњрЃў:**
```
My name is Giorgi and I am 25 years old.
```

­ЪЉЅ рЃерЃћрЃњрЃЋрЃўрЃФрЃџрЃўрЃљ рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃћрЃЉрЃўрЃф:
```python
x = 10
y = 5
print(f"Sum: {x + y}, Product: {x * y}")
```

## ­Ъћ╣ 5. рЃарЃўрЃфрЃ«рЃЋрЃћрЃЉрЃўрЃА рЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃарЃћрЃЉрЃљ
f-рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃћрЃЉрЃерЃў рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ рЃЋрЃљрЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃћрЃЉрЃЌ рЃарЃўрЃфрЃ«рЃЋрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃЕрЃћрЃюрЃўрЃА рЃцрЃЮрЃарЃЏрЃљрЃА:

```python
pi = 3.14159265
print(f"¤ђ = {pi:.2f}")   # рЃЮрЃарЃў рЃљрЃЌрЃгрЃўрЃџрЃљрЃЊрЃў
```
­ЪДЕ рЃерЃћрЃЊрЃћрЃњрЃў:
```
¤ђ = 3.14
```

рЃарЃўрЃфрЃ«рЃЋрЃћрЃЉрЃўрЃА рЃњрЃљрЃАрЃгрЃЮрЃарЃћрЃЉрЃљ рЃАрЃўрЃњрЃарЃФрЃўрЃЌ:
```python
num = 7
print(f"Number: {num:>5}")   # рЃЏрЃљрЃарЃ»рЃЋрЃюрЃўрЃЋ рЃњрЃљрЃАрЃгрЃЮрЃарЃћрЃЉрЃљ
print(f"Number: {num:<5}")   # рЃЏрЃљрЃарЃфрЃ«рЃюрЃўрЃЋ рЃњрЃљрЃАрЃгрЃЮрЃарЃћрЃЉрЃљ
print(f"Number: {num:^5}")   # рЃфрЃћрЃюрЃбрЃарЃерЃў рЃњрЃљрЃАрЃгрЃЮрЃарЃћрЃЉрЃљ
```

## ­Ъћ╣ 6. рЃЏрЃарЃљрЃЋрЃљрЃџрЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃўрЃљрЃюрЃў рЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃарЃћрЃЉрЃљ
f-рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃћрЃЉрЃў рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА рЃЏрЃарЃљрЃЋрЃљрЃџ рЃ«рЃљрЃќрЃќрЃћрЃф:

```python
name = "Giorgi"
job = "Developer"
print(f\"\"\"
Hello, {name}!
Your role: {job}.
Welcome aboard!
\"\"\")
```

## ­Ъћ╣ 7. рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў Рђћ рЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃфрЃ«рЃарЃўрЃџрЃў

```python
data = [
    ("Alice", 24, "Engineer"),
    ("Bob", 30, "Designer"),
    ("Giorgi", 25, "Developer")
]

for name, age, job in data:
    print(f"{name:<10} | {age:^5} | {job:>10}")
```

­ЪДЕ рЃерЃћрЃЊрЃћрЃњрЃў:
```
Alice      |  24   |   Engineer
Bob        |  30   |    Designer
Giorgi     |  25   |   Developer
```

## РюЁ рЃЊрЃљрЃАрЃЎрЃЋрЃюрЃљ

| рЃЏрЃћрЃЌрЃЮрЃЊрЃў | рЃљрЃдрЃгрЃћрЃарЃљ | рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃљрЃфрЃўрЃљ |
|:--|:--|:--|
| `+` | рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃерЃћрЃћрЃарЃЌрЃћрЃЉрЃљ | рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА |
| `%` | рЃФрЃЋрЃћрЃџрЃў рЃАрЃбрЃўрЃџрЃў | рЃљрЃдрЃљрЃа рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃљрЃ«рЃљрЃџ рЃЎрЃЮрЃЊрЃерЃў |
| `.format()` | рЃЏрЃЮрЃЦрЃюрЃўрЃџрЃў рЃЊрЃљ рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮ | рЃЎрЃљрЃарЃњрЃўрЃљ рЃДрЃЋрЃћрЃџрЃљ рЃЋрЃћрЃарЃАрЃўрЃљрЃерЃў |
| `f""` | рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃЊрЃљ рЃАрЃгрЃарЃљрЃцрЃў | ­ЪћЦ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћ рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃА, рЃарЃЮрЃфрЃљ рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљ |
