## ­Ъћ░ рЃерЃћрЃАрЃљрЃЋрЃљрЃџрЃў

**Exception Handling** (рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃљ) рЃљрЃарЃўрЃА рЃЏрЃћрЃЦрЃљрЃюрЃўрЃќрЃЏрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃњрЃЋрЃљрЃФрЃџрЃћрЃЋрЃА рЃњрЃљрЃЋрЃБрЃЏрЃЎрЃџрЃљрЃЋрЃЊрЃћрЃЌ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃА рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃўрЃА рЃЊрЃарЃЮрЃА рЃгрЃљрЃарЃЏрЃЮрЃЦрЃЏрЃюрЃўрЃџ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃА рЃўрЃАрЃћ, рЃарЃЮрЃЏ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ рЃљрЃа рЃњрЃљрЃЕрЃћрЃарЃЊрЃћрЃА рЃБрЃћрЃфрЃарЃљрЃЊ.

### ­Ъј» рЃарЃљрЃбрЃЮрЃЏ рЃљрЃарЃўрЃА рЃћрЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў?

РюЁ **рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃА рЃАрЃбрЃљрЃЉрЃўрЃџрЃБрЃарЃЮрЃЉрЃљ** - рЃЌрЃљрЃЋрЃўрЃЊрЃљрЃю рЃљрЃЋрЃўрЃфрЃўрЃџрЃЮрЃЌ рЃБрЃћрЃфрЃљрЃарЃў рЃњрЃљрЃЕрЃћрЃарЃћрЃЉрЃљ  
РюЁ **рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃњрЃљрЃЏрЃЮрЃфрЃЊрЃўрЃџрЃћрЃЉрЃљ** - рЃЏрЃўрЃЋрЃфрЃћрЃЌ рЃњрЃљрЃАрЃљрЃњрЃћрЃЉрЃў рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃћрЃЉрЃў  
РюЁ **рЃЊрЃћрЃЉрЃљрЃњрЃўрЃюрЃњрЃў** - рЃЋрЃўрЃърЃЮрЃЋрЃЮрЃЌ рЃЊрЃљ рЃњрЃљрЃЋрЃљрЃАрЃгрЃЮрЃарЃЮрЃЌ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃў  
РюЁ **рЃарЃћрЃАрЃБрЃарЃАрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ** - рЃцрЃљрЃўрЃџрЃћрЃЉрЃў, рЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃў рЃАрЃгрЃЮрЃарЃљрЃЊ рЃЊрЃљрЃўрЃ«рЃБрЃарЃЮрЃА

---

## Рџа№ИЈ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃўрЃА рЃбрЃўрЃърЃћрЃЉрЃў

Python-рЃерЃў рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃў рЃўрЃДрЃЮрЃцрЃљ рЃАрЃљрЃЏ рЃФрЃўрЃарЃўрЃЌрЃљрЃЊ рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃљрЃЊ:

### 1№ИЈРЃБ Syntax Errors (рЃАрЃўрЃюрЃбрЃљрЃЦрЃАрЃўрЃА рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃў)

рЃЎрЃЮрЃЊрЃў рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃљрЃЊ рЃљрЃарЃўрЃА рЃЊрЃљрЃгрЃћрЃарЃўрЃџрЃў рЃЊрЃљ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ рЃАрЃљрЃћрЃарЃЌрЃЮрЃЊ рЃљрЃа рЃњрЃљрЃћрЃерЃЋрЃћрЃЉрЃљ.

```python
# РЮї рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃў рЃАрЃўрЃюрЃбрЃљрЃЦрЃАрЃў
if x > 5
    print("x рЃЊрЃўрЃЊрЃўрЃљ")  # рЃЊрЃљрЃЎрЃљрЃарЃњрЃБрЃџрЃўрЃљ :

# РЮї рЃЊрЃљрЃЎрЃљрЃарЃњрЃБрЃџрЃў рЃЉрЃарЃГрЃДрЃљрЃџрЃћрЃЉрЃў
print(Hello World)

# РЮї рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃў рЃўрЃюрЃЊрЃћрЃюрЃбрЃљрЃфрЃўрЃљ
def my_func():
print("test")
```

### 2№ИЈРЃБ Runtime Errors / Exceptions (рЃњрЃљрЃерЃЋрЃћрЃЉрЃўрЃА рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃў)

рЃЎрЃЮрЃЊрЃў рЃАрЃгрЃЮрЃарЃљрЃЊ рЃљрЃарЃўрЃА рЃЊрЃљрЃгрЃћрЃарЃўрЃџрЃў, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃўрЃАрЃљрЃА рЃ«рЃЊрЃћрЃЉрЃљ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ.

```python
# РЮї ZeroDivisionError
result = 10 / 0

# РЮї TypeError
number = "10" + 5

# РЮї IndexError
my_list = [1, 2, 3]
print(my_list[10])

# РЮї KeyError
my_dict = {"name": "Ana"}
print(my_dict["age"])

# РЮї FileNotFoundError
file = open("рЃљрЃарЃАрЃћрЃЉрЃБрЃџрЃў_рЃцрЃљрЃўрЃџрЃў.txt")
```

### 3№ИЈРЃБ Logical Errors (рЃџрЃЮрЃњрЃўрЃЎрЃБрЃарЃў рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃў)

рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃљрЃарЃљрЃАрЃгрЃЮрЃа рЃерЃћрЃЊрЃћрЃњрЃА рЃўрЃФрЃџрЃћрЃЋрЃљ.

```python
# рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃџрЃЮрЃњрЃўрЃЎрЃљ рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃўрЃљ
def calculate_average(numbers):
    return sum(numbers) / len(numbers) + 1  # РЮї +1 рЃќрЃћрЃЊрЃЏрЃћрЃбрЃўрЃљ
```

## ­ЪћД рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў Exception-рЃћрЃЉрЃўрЃА рЃбрЃўрЃърЃћрЃЉрЃў

| Exception | рЃљрЃдрЃгрЃћрЃарЃљ | рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў |
|-----------|--------|----------|
| **SyntaxError** | рЃАрЃўрЃюрЃбрЃљрЃЦрЃАрЃўрЃА рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ | `if x > 5` (: рЃЊрЃљрЃЎрЃљрЃарЃњрЃБрЃџрЃў) |
| **ZeroDivisionError** | рЃюрЃБрЃџрЃќрЃћ рЃњрЃљрЃДрЃЮрЃцрЃљ | `10 / 0` |
| **TypeError** | рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃў рЃбрЃўрЃърЃў | `"hello" + 5` |
| **ValueError** | рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ | `int("abc")` |
| **IndexError** | рЃўрЃюрЃЊрЃћрЃЦрЃАрЃў рЃЊрЃўрЃљрЃърЃљрЃќрЃЮрЃюрЃА рЃњрЃљрЃарЃћрЃЌ | `list[100]` |
| **KeyError** | key рЃљрЃа рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА | `dict["missing_key"]` |
| **NameError** | рЃфрЃЋрЃџрЃљрЃЊрЃў рЃљрЃа рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА | `print(x)` (x рЃљрЃа рЃљрЃарЃўрЃА рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃБрЃџрЃў) |
| **FileNotFoundError** | рЃцрЃљрЃўрЃџрЃў рЃЋрЃћрЃа рЃЏрЃЮрЃўрЃФрЃћрЃЉрЃюрЃљ | `open("nonexistent.txt")` |
| **AttributeError** | рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃА рЃљрЃа рЃљрЃЦрЃЋрЃА рЃљрЃбрЃарЃўрЃЉрЃБрЃбрЃў | `"hello".append()` |
| **ImportError** | рЃЏрЃЮрЃЊрЃБрЃџрЃў рЃЋрЃћрЃа рЃўрЃЏрЃърЃЮрЃарЃбрЃўрЃарЃЊрЃљ | `import nonexistent_module` |

## ­ЪЏа№ИЈ Try-Except

### ­ЪЊї рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ

```python
try:
    # рЃЎрЃЮрЃЊрЃў рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃгрЃЋрЃўрЃЮрЃА рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ
    risky_code()
except:
    # рЃарЃљ рЃЏрЃЮрЃ«рЃЊрЃћрЃА рЃерЃћрЃфрЃЊрЃЮрЃЏрЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў
    print("рЃЏрЃЮрЃ«рЃЊрЃљ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ!")
```

### ­ЪњА рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў

```python
try:
    number = int(input("рЃерЃћрЃўрЃДрЃЋрЃљрЃюрЃћ рЃарЃўрЃфрЃ«рЃЋрЃў: "))
    result = 100 / number
    print(f"рЃерЃћрЃЊрЃћрЃњрЃў: {result}")
except:
    print("рЃарЃљрЃдрЃљрЃф рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃўрЃљ!")
```

---

### ­Ъј» рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃў Exception-рЃћрЃЉрЃўрЃА рЃЊрЃљрЃГрЃћрЃарЃљ

```python
try:
    number = int(input("рЃерЃћрЃўрЃДрЃЋрЃљрЃюрЃћ рЃарЃўрЃфрЃ«рЃЋрЃў: "))
    result = 100 / number
    print(f"рЃерЃћрЃЊрЃћрЃњрЃў: {result}")
    
except ValueError:
    print("РЮї рЃерЃћрЃўрЃДрЃЋрЃљрЃюрЃћ рЃарЃўрЃфрЃ«рЃЋрЃў, рЃљрЃарЃљ рЃбрЃћрЃЦрЃАрЃбрЃў!")
    
except ZeroDivisionError:
    print("РЮї рЃюрЃБрЃџрЃќрЃћ рЃњрЃљрЃДрЃЮрЃцрЃљ рЃерЃћрЃБрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ!")
    
except Exception as e:
    print(f"РЮї рЃБрЃфрЃюрЃЮрЃЉрЃў рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ: {e}")
```

### ­ЪЊІ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃўрЃА рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃўрЃА рЃЏрЃўрЃдрЃћрЃЉрЃљ

```python
try:
    x = 10 / 0
except ZeroDivisionError as error:
    print(f"рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ: {error}")
    print(f"рЃерЃћрЃфрЃЊрЃЮрЃЏрЃўрЃА рЃбрЃўрЃърЃў: {type(error).__name__}")
```

---

### ­ЪћЌ рЃарЃљрЃЏрЃЊрЃћрЃюрЃўрЃЏрЃћ Exception-рЃўрЃА рЃЊрЃљрЃГрЃћрЃарЃљ рЃћрЃарЃЌрЃЊрЃарЃЮрЃБрЃџрЃљрЃЊ

```python
# рЃЏрЃћрЃЌрЃЮрЃЊрЃў 1: рЃфрЃљрЃџ-рЃфрЃљрЃџрЃЎрЃћ except
try:
    # рЃарЃўрЃАрЃЎрЃўрЃљрЃюрЃў рЃЎрЃЮрЃЊрЃў
    pass
except ValueError:
    print("Value Error!")
except TypeError:
    print("Type Error!")
except KeyError:
    print("Key Error!")

# рЃЏрЃћрЃЌрЃЮрЃЊрЃў 2: рЃћрЃарЃЌ except-рЃерЃў (tuple)
try:
    # рЃарЃўрЃАрЃЎрЃўрЃљрЃюрЃў рЃЎрЃЮрЃЊрЃў
    pass
except (ValueError, TypeError, KeyError):
    print("рЃЏрЃЮрЃ«рЃЊрЃљ рЃћрЃарЃЌ-рЃћрЃарЃЌрЃў рЃљрЃЏ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃўрЃЊрЃљрЃю!")

# рЃЏрЃћрЃЌрЃЮрЃЊрЃў 3: рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃў + рЃќрЃЮрЃњрЃљрЃЊрЃў
try:
    # рЃарЃўрЃАрЃЎрЃўрЃљрЃюрЃў рЃЎрЃЮрЃЊрЃў
    pass
except ValueError:
    print("ValueError!")
except Exception as e:
    print(f"рЃАрЃ«рЃЋрЃљ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ: {e}")
```

## ­Ъћё Else рЃЊрЃљ Finally

### ­ЪЊї рЃАрЃарЃБрЃџрЃў рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ

```python
try:
    # рЃЎрЃЮрЃЊрЃў рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃгрЃЋрЃўрЃЮрЃА рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ
    risky_code()
    
except SomeException:
    # рЃЌрЃБ рЃЏрЃЮрЃ«рЃЊрЃљ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ
    handle_error()
    
else:
    # рЃЌрЃБ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ рЃљрЃа рЃЏрЃЮрЃ«рЃЊрЃљ
    success_code()
    
finally:
    # рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃА рЃерЃћрЃАрЃарЃБрЃџрЃЊрЃћрЃЉрЃљ (рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ рЃўрЃДрЃЮ рЃЌрЃБ рЃљрЃарЃљ)
    cleanup_code()
```

---

### ­ЪњА Else - рЃгрЃљрЃарЃЏрЃљрЃбрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў

`else` рЃЉрЃџрЃЮрЃЎрЃў рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃЏрЃљрЃерЃўрЃю рЃерЃћрЃАрЃарЃБрЃџрЃЊрЃћрЃЉрЃљ, рЃЌрЃБ `try` рЃЉрЃџрЃЮрЃЎрЃерЃў рЃљрЃарЃљрЃюрЃљрЃўрЃарЃў рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ рЃљрЃа рЃЏрЃЮрЃ«рЃЊрЃљ.

```python
try:
    number = int(input("рЃерЃћрЃўрЃДрЃЋрЃљрЃюрЃћ рЃарЃўрЃфрЃ«рЃЋрЃў: "))
    result = 100 / number
    
except ValueError:
    print("РЮї рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃў рЃцрЃЮрЃарЃЏрЃљрЃбрЃў!")
    
except ZeroDivisionError:
    print("РЮї рЃюрЃБрЃџрЃќрЃћ рЃњрЃљрЃДрЃЮрЃцрЃљ рЃљрЃа рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ!")
    
else:
    # рЃћрЃА рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃгрЃљрЃарЃЏрЃљрЃбрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў рЃерЃћрЃАрЃарЃБрЃџрЃЊрЃћрЃЉрЃљ
    print(f"РюЁ рЃерЃћрЃЊрЃћрЃњрЃў: {result}")
    print("РюЁ рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃљ рЃгрЃљрЃарЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌ рЃЊрЃљрЃАрЃарЃБрЃџрЃЊрЃљ!")
```

---

### ­Ъћњ Finally - рЃЊрЃљрЃАрЃБрЃцрЃЌрЃљрЃЋрЃћрЃЉрЃўрЃА рЃЎрЃЮрЃЊрЃў

`finally` рЃЉрЃџрЃЮрЃЎрЃў **рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃА** рЃерЃћрЃАрЃарЃБрЃџрЃЊрЃћрЃЉрЃљ, рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ рЃЏрЃЮрЃ«рЃЊрЃљ рЃЌрЃБ рЃљрЃарЃљ.

```python
try:
    file = open("data.txt", "r")
    content = file.read()
    print(content)
    
except FileNotFoundError:
    print("РЮї рЃцрЃљрЃўрЃџрЃў рЃЋрЃћрЃа рЃЏрЃЮрЃўрЃФрЃћрЃЉрЃюрЃљ!")
    
finally:
    # рЃћрЃА рЃЎрЃЮрЃЊрЃў рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃА рЃерЃћрЃАрЃарЃБрЃџрЃЊрЃћрЃЉрЃљ
    try:
        file.close()
        print("РюЁ рЃцрЃљрЃўрЃџрЃў рЃЊрЃљрЃўрЃ«рЃБрЃарЃљ")
    except:
        print("Рё╣№ИЈ рЃцрЃљрЃўрЃџрЃў рЃБрЃЎрЃЋрЃћ рЃЊрЃљрЃ«рЃБрЃарЃБрЃџрЃўрЃљ рЃљрЃю рЃљрЃа рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА")
```

---

### ­Ъј» рЃАрЃарЃБрЃџрЃў рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў - рЃДрЃЋрЃћрЃџрЃљ рЃЉрЃџрЃЮрЃЎрЃў

```python
def safe_divide(a, b):
    """рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮ рЃњрЃљрЃДрЃЮрЃцрЃљ рЃЊрЃћрЃбрЃљрЃџрЃБрЃарЃў рЃџрЃЮрЃњрЃўрЃарЃћрЃЉрЃўрЃЌ"""
    
    try:
        print(f"­Ъћё рЃЋрЃфрЃЊрЃўрЃџрЃЮрЃЉ рЃњрЃљрЃЋрЃДрЃЮрЃцрЃЮ {a} / {b}")
        result = a / b
        
    except ZeroDivisionError:
        print("РЮї рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ: рЃюрЃБрЃџрЃќрЃћ рЃњрЃљрЃДрЃЮрЃцрЃљ!")
        return None
        
    except TypeError:
        print("РЮї рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ: рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃў рЃбрЃўрЃърЃў!")
        return None
        
    else:
        print("РюЁ рЃњрЃљрЃДрЃЮрЃцрЃљ рЃгрЃљрЃарЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌ рЃерЃћрЃАрЃарЃБрЃџрЃЊрЃљ")
        return result
        
    finally:
        print("­ЪЈЂ рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃљ рЃЊрЃљрЃАрЃарЃБрЃџрЃЊрЃљ
")

# рЃбрЃћрЃАрЃбрЃћрЃЉрЃў
print(safe_divide(10, 2))      # РюЁ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА
print(safe_divide(10, 0))      # РЮї ZeroDivisionError
print(safe_divide("10", 2))    # РЮї TypeError
```

**рЃњрЃљрЃЏрЃЮрЃАрЃљрЃЋрЃљрЃџрЃў:**
```
­Ъћё рЃЋрЃфрЃЊрЃўрЃџрЃЮрЃЉ рЃњрЃљрЃЋрЃДрЃЮрЃцрЃЮ 10 / 2
РюЁ рЃњрЃљрЃДрЃЮрЃцрЃљ рЃгрЃљрЃарЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌ рЃерЃћрЃАрЃарЃБрЃџрЃЊрЃљ
­ЪЈЂ рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃљ рЃЊрЃљрЃАрЃарЃБрЃџрЃЊрЃљ

5.0

­Ъћё рЃЋрЃфрЃЊрЃўрЃџрЃЮрЃЉ рЃњрЃљрЃЋрЃДрЃЮрЃцрЃЮ 10 / 0
РЮї рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ: рЃюрЃБрЃџрЃќрЃћ рЃњрЃљрЃДрЃЮрЃцрЃљ!
­ЪЈЂ рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃљ рЃЊрЃљрЃАрЃарЃБрЃџрЃЊрЃљ

None
```

## ­ЪЊЂ With-As Statement

### ­ЪЊї рЃарЃљ рЃљрЃарЃўрЃА With-As?

`with` рЃАрЃбрЃћрЃўрЃбрЃЏрЃћрЃюрЃбрЃў рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА рЃарЃћрЃАрЃБрЃарЃАрЃћрЃЉрЃўрЃА рЃАрЃгрЃЮрЃа рЃЏрЃљрЃарЃЌрЃЋрЃљрЃА (рЃцрЃљрЃўрЃџрЃћрЃЉрЃў, рЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃў рЃЊрЃљ рЃљ.рЃе.).

### ­ЪћЉ рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃБрЃърЃўрЃарЃљрЃбрЃћрЃАрЃЮрЃЉрЃћрЃЉрЃў:

РюЁ рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃ«рЃБрЃарЃљрЃЋрЃА рЃцрЃљрЃўрЃџрЃА  
РюЁ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА рЃерЃћрЃфрЃЊрЃЮрЃЏрЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃўрЃф  
РюЁ рЃБрЃцрЃарЃЮ рЃАрЃБрЃцрЃЌрЃљ рЃЊрЃљ рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљрЃЊрЃў рЃЎрЃЮрЃЊрЃў  
РюЁ рЃљрЃа рЃАрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ `finally` рЃЉрЃџрЃЮрЃЎрЃў

---

### ­ЪЊќ рЃцрЃљрЃўрЃџрЃћрЃЉрЃЌрЃљрЃю рЃЏрЃБрЃерЃљрЃЮрЃЉрЃљ - рЃФрЃЋрЃћрЃџрЃў vs рЃљрЃ«рЃљрЃџрЃў

#### РЮї рЃФрЃЋрЃћрЃџрЃў рЃЏрЃћрЃЌрЃЮрЃЊрЃў (try-finally)

```python
# рЃарЃЌрЃБрЃџрЃў рЃЊрЃљ рЃњрЃарЃФрЃћрЃџрЃў
try:
    file = open("data.txt", "r")
    content = file.read()
    print(content)
except FileNotFoundError:
    print("рЃцрЃљрЃўрЃџрЃў рЃЋрЃћрЃа рЃЏрЃЮрЃўрЃФрЃћрЃЉрЃюрЃљ")
finally:
    try:
        file.close()
    except:
        pass
```

#### РюЁ рЃљрЃ«рЃљрЃџрЃў рЃЏрЃћрЃЌрЃЮрЃЊрЃў (with-as)

```python
# рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃЊрЃљ рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮ
try:
    with open("data.txt", "r") as file:
        content = file.read()
        print(content)
    # рЃцрЃљрЃўрЃџрЃў рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃЊрЃљрЃўрЃ«рЃБрЃарЃљ!
except FileNotFoundError:
    print("рЃцрЃљрЃўрЃџрЃў рЃЋрЃћрЃа рЃЏрЃЮрЃўрЃФрЃћрЃЉрЃюрЃљ")
```

---

### ­ЪњА With-As - рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃБрЃџрЃў рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃў

#### 1№ИЈРЃБ рЃцрЃљрЃўрЃџрЃўрЃА рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљ

```python
# рЃбрЃћрЃЦрЃАрЃбрЃБрЃарЃў рЃцрЃљрЃўрЃџрЃўрЃА рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљ
try:
    with open("students.txt", "r", encoding="utf-8") as file:
        for line in file:
            print(line.strip())
except FileNotFoundError:
    print("РЮї рЃцрЃљрЃўрЃџрЃў students.txt рЃЋрЃћрЃа рЃЏрЃЮрЃўрЃФрЃћрЃЉрЃюрЃљ")
```

#### 2№ИЈРЃБ рЃцрЃљрЃўрЃџрЃерЃў рЃЕрЃљрЃгрЃћрЃарЃљ

```python
# рЃљрЃ«рЃљрЃџрЃў рЃцрЃљрЃўрЃџрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ рЃЊрЃљ рЃЕрЃљрЃгрЃћрЃарЃљ
try:
    with open("output.txt", "w", encoding="utf-8") as file:
        file.write("рЃњрЃљрЃЏрЃљрЃарЃ»рЃЮрЃЉрЃљ, рЃЏрЃАрЃЮрЃцрЃџрЃўрЃЮ!
")
        file.write("Python-рЃўрЃЌ рЃЋрЃгрЃћрЃарЃЌ рЃцрЃљрЃўрЃџрЃерЃў
")
    print("РюЁ рЃцрЃљрЃўрЃџрЃў рЃгрЃљрЃарЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌ рЃЕрЃљрЃўрЃгрЃћрЃарЃљ")
except IOError as e:
    print(f"РЮї рЃцрЃљрЃўрЃџрЃерЃў рЃЕрЃљрЃгрЃћрЃарЃўрЃА рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ: {e}")
```

#### 3№ИЈРЃБ рЃцрЃљрЃўрЃџрЃерЃў рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ (append)

```python
# рЃљрЃарЃАрЃћрЃЉрЃБрЃџ рЃцрЃљрЃўрЃџрЃерЃў рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ
try:
    with open("log.txt", "a", encoding="utf-8") as file:
        file.write("рЃљрЃ«рЃљрЃџрЃў рЃџрЃЮрЃњрЃў: рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃљ рЃерЃћрЃАрЃарЃБрЃџрЃЊрЃљ
")
    print("РюЁ рЃџрЃЮрЃњрЃў рЃЊрЃљрЃћрЃЏрЃљрЃбрЃљ")
except Exception as e:
    print(f"РЮї рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ: {e}")
```

#### 4№ИЈРЃБ рЃЮрЃарЃў рЃцрЃљрЃўрЃџрЃў рЃћрЃарЃЌрЃЊрЃарЃЮрЃБрЃџрЃљрЃЊ

```python
# рЃћрЃарЃЌрЃў рЃцрЃљрЃўрЃџрЃўрЃЊрЃљрЃю рЃЏрЃћрЃЮрЃарЃћрЃерЃў рЃЎрЃЮрЃърЃўрЃарЃћрЃЉрЃљ
try:
    with open("source.txt", "r") as source, \
         open("destination.txt", "w") as dest:
        
        content = source.read()
        dest.write(content)
        
    print("РюЁ рЃцрЃљрЃўрЃџрЃў рЃњрЃљрЃЊрЃљрЃЎрЃЮрЃърЃўрЃарЃЊрЃљ")
    
except FileNotFoundError:
    print("РЮї рЃгрЃДрЃљрЃарЃЮ рЃцрЃљрЃўрЃџрЃў рЃЋрЃћрЃа рЃЏрЃЮрЃўрЃФрЃћрЃЉрЃюрЃљ")
except IOError as e:
    print(f"РЮї I/O рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ: {e}")
```

#### 5№ИЈРЃБ JSON рЃцрЃљрЃўрЃџрЃћрЃЉрЃЌрЃљрЃю рЃЏрЃБрЃерЃљрЃЮрЃЉрЃљ

```python
import json

# JSON-рЃўрЃА рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљ
try:
    with open("data.json", "r", encoding="utf-8") as file:
        data = json.load(file)
        print(f"рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў: {data}")
except FileNotFoundError:
    print("РЮї JSON рЃцрЃљрЃўрЃџрЃў рЃЋрЃћрЃа рЃЏрЃЮрЃўрЃФрЃћрЃЉрЃюрЃљ")
except json.JSONDecodeError:
    print("РЮї рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃў JSON рЃцрЃЮрЃарЃЏрЃљрЃбрЃў")

# JSON-рЃерЃў рЃЕрЃљрЃгрЃћрЃарЃљ
student = {
    "name": "рЃљрЃюрЃљ",
    "age": 20,
    "grades": [95, 88, 92]
}

try:
    with open("student.json", "w", encoding="utf-8") as file:
        json.dump(student, file, ensure_ascii=False, indent=4)
    print("РюЁ JSON рЃцрЃљрЃўрЃџрЃў рЃерЃћрЃўрЃЦрЃЏрЃюрЃљ")

    print(f"РЮї рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ: {e}")
```

## ­Ъџђ Raise - рЃерЃћрЃфрЃЊрЃЮрЃЏрЃўрЃА рЃњрЃћрЃюрЃћрЃарЃўрЃарЃћрЃЉрЃљ

### ­ЪЊї рЃарЃљрЃА рЃюрЃўрЃерЃюрЃљрЃЋрЃА Raise?

`raise` рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃњрЃЋрЃљрЃФрЃџрЃћрЃЋрЃА рЃЕрЃЋрЃћрЃю рЃЌрЃљрЃЋрЃљрЃЊ "рЃњрЃљрЃЋрЃљрЃњрЃЊрЃЮрЃЌ" рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ.

```python
def check_age(age):
    if age < 0:
        raise ValueError("РЮї рЃљрЃАрЃљрЃЎрЃў рЃљрЃа рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃўрЃДрЃЮрЃА рЃБрЃљрЃарЃДрЃЮрЃцрЃўрЃЌрЃў!")

    return f"РюЁ рЃљрЃАрЃљрЃЎрЃў рЃАрЃгрЃЮрЃарЃўрЃљ: {age}"

# рЃбрЃћрЃАрЃбрЃў
try:
    print(check_age(25))   # РюЁ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА

    print(e)
```

## ­Ъје Custom Exceptions - рЃАрЃљрЃЎрЃБрЃЌрЃљрЃарЃў рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃў

### ­ЪЊї рЃарЃљрЃбрЃЮрЃЏ рЃњрЃЋрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ?

рЃарЃЮрЃфрЃљ Python-рЃўрЃА рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃБрЃџрЃў exception-рЃћрЃЉрЃў рЃљрЃа рЃљрЃарЃўрЃА рЃАрЃљрЃЎрЃЏрЃљрЃарЃўрЃАрЃў.

```python
# рЃАрЃљрЃЎрЃБрЃЌрЃљрЃарЃў exception-рЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ
class NegativeNumberError(Exception):
    """рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ рЃБрЃљрЃарЃДрЃЮрЃцрЃўрЃЌрЃў рЃарЃўрЃфрЃ«рЃЋрЃўрЃАрЃЌрЃЋрЃўрЃА"""
    pass

class TooLargeNumberError(Exception):
    """рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ рЃФрЃљрЃџрЃўрЃљрЃю рЃЊрЃўрЃЊрЃў рЃарЃўрЃфрЃ«рЃЋрЃўрЃАрЃЌрЃЋрЃўрЃА"""
    def __init__(self, number, max_value):
        self.number = number
        self.max_value = max_value
        super().__init__(f"рЃарЃўрЃфрЃ«рЃЋрЃў {number} рЃљрЃдрЃћрЃЏрЃљрЃбрЃћрЃЉрЃљ рЃЏрЃљрЃЦрЃАрЃўрЃЏрЃБрЃЏрЃА {max_value}")

# рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ
def validate_score(score):
    if score < 0:
        raise NegativeNumberError("рЃЦрЃБрЃџрЃљ рЃљрЃа рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃўрЃДрЃЮрЃА рЃБрЃљрЃарЃДрЃЮрЃцрЃўрЃЌрЃў!")
    if score > 100:
        raise TooLargeNumberError(score, 100)
    return f"РюЁ рЃЦрЃБрЃџрЃљ: {score}"

# рЃбрЃћрЃАрЃбрЃў
try:
    print(validate_score(95))   # РюЁ
    print(validate_score(150))  # РЮї
except NegativeNumberError as e:
    print(f"РЮї {e}")
except TooLargeNumberError as e:
    print(f"РЮї {e}")
    print(f"   рЃЏрЃЮрЃфрЃћрЃЏрЃБрЃџрЃў: {e.number}, рЃЏрЃљрЃЦрЃАрЃўрЃЏрЃБрЃЏрЃў: {e.max_value}")
```

### ­ЪЈЌ№ИЈ Exception-рЃћрЃЉрЃўрЃА рЃўрЃћрЃарЃљрЃарЃЦрЃўрЃљ

```python
class PhoneValidationError(ValidationError):
    """рЃбрЃћрЃџрЃћрЃцрЃЮрЃюрЃўрЃА рЃЋрЃљрЃџрЃўрЃЊрЃљрЃфрЃўрЃўрЃА рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ"""

# рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ

# рЃЊрЃљрЃГрЃћрЃарЃљ
try:
    validate_email("invalid_email")
except ValidationError as e:  # рЃЊрЃљрЃўрЃГрЃћрЃарЃА рЃДрЃЋрЃћрЃџрЃљ ValidationError-рЃА
    print(f"РЮї рЃЋрЃљрЃџрЃўрЃЊрЃљрЃфрЃўрЃўрЃА рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ: {e}")
```

## ­ЪДЕ рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃБрЃџрЃў рЃљрЃЏрЃЮрЃфрЃљрЃюрЃћрЃЉрЃў

### ­ЪЊЮ рЃљрЃЏрЃЮрЃфрЃљрЃюрЃћрЃЉрЃў (1-30)

#### A. рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў Try-Except (1-10)

1. **рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃЊрЃљрЃфрЃЋрЃљ**: рЃерЃћрЃЦрЃЏрЃћрЃюрЃў рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃЌрЃ«рЃЮрЃЋрЃА рЃарЃўрЃфрЃ«рЃЋрЃА рЃЊрЃљ рЃўрЃГрЃћрЃарЃА ValueError-рЃА
2. **рЃњрЃљрЃДрЃЮрЃцрЃљ**: рЃЊрЃљрЃгрЃћрЃарЃћ рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЮрЃа рЃарЃўрЃфрЃ«рЃЋрЃА рЃДрЃЮрЃцрЃА рЃЊрЃљ рЃўрЃГрЃћрЃарЃА ZeroDivisionError-рЃА
3. **List Index**: рЃАрЃўрЃўрЃЊрЃљрЃю рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃљрЃЏрЃЮрЃдрЃћрЃЉрЃљ try-except-рЃўрЃЌ (IndexError)
4. **Dictionary Key**: dictionary-рЃЊрЃљрЃю value-рЃўрЃА рЃљрЃЏрЃЮрЃдрЃћрЃЉрЃљ try-except-рЃўрЃЌ (KeyError)
5. **Type Conversion**: str Рєњ int рЃЎрЃЮрЃюрЃЋрЃћрЃарЃбрЃљрЃфрЃўрЃљ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃўрЃА рЃЊрЃљрЃГрЃћрЃарЃўрЃЌ

```python
class Resource:

        self.name = name

        self.is_open = True
    
    def close(self):
        print(f"­Ъћњ {self.name} рЃЊрЃљрЃўрЃ«рЃБрЃарЃљ")
        self.is_open = False

    finally:

class WeakPasswordError(Exception):
    """рЃАрЃБрЃАрЃбрЃў рЃърЃљрЃарЃЮрЃџрЃўрЃА рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ"""
    def __init__(self, message, issues):
        super().__init__(message)
        self.issues = issues

def validate_password(password):
    issues = []
    
    if len(password) < 8:
        issues.append("рЃърЃљрЃарЃЮрЃџрЃў рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА рЃЏрЃўрЃюрЃўрЃЏрЃБрЃЏ 8 рЃАрЃўрЃЏрЃЉрЃЮрЃџрЃЮ")
    if not any(c.isupper() for c in password):
        issues.append("рЃърЃљрЃарЃЮрЃџрЃў рЃБрЃюрЃЊрЃљ рЃерЃћрЃўрЃфрЃљрЃЋрЃЊрЃћрЃА рЃЊрЃўрЃЊ рЃљрЃАрЃЮрЃА")
    if not any(c.isdigit() for c in password):
        issues.append("рЃърЃљрЃарЃЮрЃџрЃў рЃБрЃюрЃЊрЃљ рЃерЃћрЃўрЃфрЃљрЃЋрЃЊрЃћрЃА рЃфрЃўрЃцрЃарЃА")
    
    if issues:
        raise WeakPasswordError("рЃърЃљрЃарЃЮрЃџрЃў рЃФрЃљрЃџрЃўрЃљрЃю рЃАрЃБрЃАрЃбрЃўрЃљ!", issues)
    
    return "РюЁ рЃърЃљрЃарЃЮрЃџрЃў рЃФрЃџрЃўрЃћрЃарЃўрЃљ!"

try:
    print(validate_password("MyPass123"))  # РюЁ
    print(validate_password("weak"))       # РЮї
except WeakPasswordError as e:
    print(f"РЮї {e}")
    print("рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃћрЃЉрЃў:")
    for issue in e.issues:
        print(f"  - {issue}")
```
