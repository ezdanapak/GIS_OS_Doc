­ЪЊў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃАрЃ«рЃЋрЃљ рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃћрЃЉрЃў Python-рЃерЃў
рЃАрЃўрЃЏрЃарЃљрЃЋрЃџрЃћ (set) рЃЊрЃљ рЃџрЃћрЃЦрЃАрЃўрЃЎрЃЮрЃюрЃў (dictionary)
+ рЃАрЃљрЃЏрЃБрЃерЃљрЃЮ рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў рЃЊрЃљ рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃў
1) ­Ъћ░ рЃерЃћрЃАрЃљрЃЋрЃљрЃџрЃў
Python-рЃерЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃћрЃЉрЃў рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЮрЃарЃњрЃљрЃюрЃўрЃќрЃћрЃЉрЃБрЃџрЃљрЃЊ рЃерЃћрЃАрЃљрЃюрЃљрЃ«рЃљрЃЊ рЃЊрЃљ рЃЏрЃЮрЃАрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃџрЃљрЃЊ. рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃ«рЃерЃўрЃарЃљрЃЊ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ:

list Рђћ рЃАрЃўрЃљ

tuple Рђћ рЃбрЃБрЃърЃџрЃў

set Рђћ рЃАрЃўрЃЏрЃарЃљрЃЋрЃџрЃћ

dictionary Рђћ рЃџрЃћрЃЦрЃАрЃўрЃЎрЃЮрЃюрЃў


2) ­Ъћи рЃАрЃўрЃЏрЃарЃљрЃЋрЃџрЃћ (Set) <br>
2.1 рЃарЃљ рЃљрЃарЃўрЃА Set? <br>
Set (рЃАрЃўрЃЏрЃарЃљрЃЋрЃџрЃћ) Рђћ рЃљрЃарЃўрЃА рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃарЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃЎрЃЮрЃџрЃћрЃЦрЃфрЃўрЃљ. <br>

­ЪћЉ рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃџрЃћрЃЉрЃў <br>
рЃљрЃа рЃўрЃюрЃљрЃ«рЃљрЃЋрЃА рЃЊрЃБрЃЉрЃџрЃўрЃарЃћрЃЉрЃБрЃџ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃА <br>

рЃЊрЃљрЃџрЃљрЃњрЃћрЃЉрЃБрЃџрЃў рЃљрЃарЃљрЃљ <br>

рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃќрЃћ рЃўрЃюрЃЊрЃћрЃЦрЃАрЃўрЃЌ рЃгрЃЋрЃЊрЃЮрЃЏрЃљ рЃерЃћрЃБрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ <br>

рЃфрЃЋрЃљрЃџрЃћрЃЉрЃљрЃЊрЃў рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљрЃљ, рЃерЃћрЃњрЃЋрЃўрЃФрЃџрЃўрЃљ рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ/рЃљрЃЏрЃЮрЃдрЃћрЃЉрЃљ <br>

2.2 рЃАрЃўрЃЏрЃарЃљрЃЋрЃџрЃћрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ <br>
```python
fruits = {"apple", "banana", "orange"} 
numbers = {1, 2, 3, 4}
```

рЃфрЃљрЃарЃўрЃћрЃџрЃў set:

```python
empty_set = set()
# Рџа№ИЈ {} Рђћ рЃЦрЃЏрЃюрЃўрЃА рЃфрЃљрЃарЃўрЃћрЃџ dictionary-рЃА, рЃљрЃарЃљ set-рЃА!
```
2.3 рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ <br>
fruits.add("mango") <br>
2.4 рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃљрЃЏрЃЮрЃфрЃџрЃљ <br>
fruits.remove("banana")   # рЃЌрЃБ рЃљрЃа рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА Рєњ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ <br>
fruits.discard("banana")  # рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃљ, рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљрЃА рЃљрЃа рЃўрЃФрЃџрЃћрЃЋрЃљ <br>
2.5 рЃАрЃўрЃЏрЃарЃљрЃЋрЃџрЃћрЃћрЃЉрЃўрЃА рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃў <br>
рЃњрЃљрЃћрЃарЃЌрЃўрЃљрЃюрЃћрЃЉрЃљ: <br>
A | B <br>
A.union(B)  <br>
рЃњрЃљрЃЊрЃљрЃЎрЃЋрЃћрЃЌрЃљ: <br>
A & B <br>
A.intersection(B) <br>
рЃАрЃ«рЃЋрЃљрЃЮрЃЉрЃљ: <br>
A - B <br>
A.difference(B) <br>
рЃАрЃўрЃЏрЃћрЃбрЃарЃўрЃБрЃџрЃў рЃАрЃ«рЃЋрЃљрЃЮрЃЉрЃљ: <br>
A ^ B <br>
A.symmetric_difference(B) <br>
3) ­ЪћХ рЃџрЃћрЃЦрЃАрЃўрЃЎрЃЮрЃюрЃў (Dictionary) <br>
3.1 рЃарЃљ рЃљрЃарЃўрЃА Dictionary? <br>
Dictionary (рЃџрЃћрЃЦрЃАрЃўрЃЎрЃЮрЃюрЃў) Рђћ рЃгрЃљрЃарЃЏрЃЮрЃљрЃЊрЃњрЃћрЃюрЃА key:value рЃгрЃДрЃЋрЃўрЃџрЃћрЃЉрЃўрЃА рЃЎрЃЮрЃџрЃћрЃЦрЃфрЃўрЃљрЃА. <br>

­ЪћЉ рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃџрЃћрЃЉрЃў <br>
Key рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃарЃў <br>

Value рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃњрЃљрЃюрЃЏрЃћрЃЮрЃарЃЊрЃћрЃА <br>

Key-рЃћрЃЉрЃў рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЋрЃюрЃћрЃю рЃБрЃфрЃЋрЃџрЃљрЃЊрЃў (str, int, tuple, ...) <br>

3.2 рЃџрЃћрЃЦрЃАрЃўрЃЎрЃЮрЃюрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ <br>
```python
student = {
    "name": "Giorgi",
    "age": 21,
    "faculty": "Computer Science"
}
```
3.3 рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃќрЃћ рЃгрЃЋрЃЊрЃЮрЃЏрЃљ <br>
print(student["name"]) <br>
print(student.get("age")) <br>
3.4 рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃўрЃА рЃерЃћрЃфрЃЋрЃџрЃљ <br>
student["age"] = 22 <br>
3.5 рЃљрЃ«рЃљрЃџрЃў рЃгрЃДрЃЋрЃўрЃџрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ <br>
student["grade"] = 97 <br>
3.6 рЃгрЃДрЃЋрЃўрЃџрЃўрЃА рЃгрЃљрЃерЃџрЃљ <br>
del student["age"] <br>
student.pop("faculty") <br>
3.7 рЃџрЃћрЃЦрЃАрЃўрЃЎрЃЮрЃюрЃўрЃА рЃЏрЃћрЃЌрЃЮрЃЊрЃћрЃЉрЃў <br>
```python
student.keys()      # key-рЃћрЃЉрЃў
student.values()    # values
student.items()     # key-value рЃгрЃДрЃЋрЃўрЃџрЃћрЃЉрЃў
student.clear()     # рЃџрЃћрЃЦрЃАрЃўрЃЎрЃЮрЃюрЃўрЃА рЃњрЃљрЃАрЃБрЃцрЃЌрЃљрЃЋрЃћрЃЉрЃљ
```
4) ­Ъћх Set vs Dictionary Рђћ рЃерЃћрЃЊрЃљрЃарЃћрЃЉрЃљ <br>
рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃћрЃџрЃў	Set	Dictionary <br>
рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ	рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃЎрЃЮрЃџрЃћрЃЦрЃфрЃўрЃљ	key:value рЃгрЃДрЃЋрЃўрЃџрЃћрЃЉрЃў <br>
рЃЊрЃБрЃЉрЃџрЃўрЃарЃћрЃЉрЃљ	рЃљрЃа рЃљрЃЦрЃЋрЃА	key Рђћ рЃљрЃарЃљ ; value Рђћ рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ <br>
рЃгрЃЋрЃЊрЃЮрЃЏрЃљ	рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃќрЃћ рЃўрЃюрЃЊрЃћрЃЦрЃАрЃўрЃЌ рЃљрЃарЃљ	key-рЃўрЃЌ <br>
рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ	рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃў, рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃарЃЮрЃЉрЃљ	рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃерЃћрЃюрЃљрЃ«рЃЋрЃљ <br>
рЃАрЃўрЃюрЃбрЃљрЃЦрЃАрЃў	{1, 2, 3}	{"a":1, "b":2} <br>
5) ­ЪДЕ 30 рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃБрЃџрЃў рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ <br>

A. рЃљрЃЏрЃЮрЃфрЃљрЃюрЃћрЃЉрЃў Рђћ Set (1РђЊ15)
рЃерЃћрЃЦрЃЏрЃћрЃюрЃў set, рЃЊрЃљрЃљрЃЏрЃљрЃбрЃћ рЃЏрЃљрЃАрЃерЃў 2 рЃљрЃ«рЃљрЃџрЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў.

рЃгрЃљрЃерЃљрЃџрЃћ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў set-рЃўрЃЊрЃљрЃю remove() рЃЏрЃћрЃЌрЃЮрЃЊрЃўрЃЌ.

рЃгрЃљрЃерЃљрЃџрЃћ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў discard()-рЃўрЃЌ рЃўрЃАрЃћ, рЃарЃЮрЃЏ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ рЃљрЃа рЃЏрЃЮрЃ«рЃЊрЃћрЃА.

рЃерЃћрЃљрЃЊрЃљрЃарЃћ рЃЮрЃарЃў set: рЃњрЃљрЃЊрЃљрЃЎрЃЋрЃћрЃЌрЃљ.

рЃерЃћрЃљрЃЊрЃљрЃарЃћ рЃЮрЃарЃў set: рЃњрЃљрЃћрЃарЃЌрЃўрЃљрЃюрЃћрЃЉрЃљ.

рЃерЃћрЃљрЃЊрЃљрЃарЃћ рЃЮрЃарЃў set: рЃАрЃ«рЃЋрЃљрЃЮрЃЉрЃљ.

рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў рЃљрЃарЃўрЃА рЃЌрЃБ рЃљрЃарЃљ set-рЃерЃў.

set-рЃерЃў рЃЕрЃљрЃгрЃћрЃарЃћ 10 рЃарЃўрЃфрЃ«рЃЋрЃў, рЃгрЃљрЃерЃљрЃџрЃћ рЃДрЃЋрЃћрЃџрЃљ рЃџрЃБрЃгрЃў.

рЃЮрЃарЃў set-рЃўрЃА рЃАрЃўрЃЏрЃћрЃбрЃарЃўрЃБрЃџрЃў рЃАрЃ«рЃЋрЃљрЃЮрЃЉрЃўрЃА рЃърЃЮрЃЋрЃюрЃљ.

set-рЃўрЃА рЃљрЃАрЃџрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ copy().

list-рЃўрЃЊрЃљрЃю рЃњрЃљрЃљрЃЎрЃћрЃЌрЃћ set рЃЊрЃљ рЃљрЃЏрЃЮрЃўрЃдрЃћ рЃЊрЃБрЃЉрЃџрЃўрЃЎрЃљрЃбрЃћрЃЉрЃў.

рЃЮрЃарЃў set-рЃўрЃА рЃерЃћрЃЊрЃљрЃарЃћрЃЉрЃљ: рЃљрЃарЃўрЃА рЃЌрЃБ рЃљрЃарЃљ рЃћрЃарЃЌрЃў рЃЏрЃћрЃЮрЃарЃўрЃА рЃЦрЃЋрЃћрЃАрЃўрЃЏрЃарЃљрЃЋрЃџрЃћ.

set-рЃўрЃЊрЃљрЃю pop() рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃљрЃЏрЃЮрЃдрЃћрЃЉрЃљ.

set-рЃўрЃА рЃњрЃљрЃАрЃБрЃцрЃЌрЃљрЃЋрЃћрЃЉрЃљ clear().

рЃЏрЃљрЃАрЃўрЃЋрЃў Рєњ рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃарЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃЌ set Рєњ рЃўрЃАрЃћрЃЋ list.

B. рЃљрЃЏрЃЮрЃфрЃљрЃюрЃћрЃЉрЃў Рђћ Dictionary (16РђЊ30)
рЃерЃћрЃЦрЃЏрЃћрЃюрЃў dictionary рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃўрЃА рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ.

рЃЊрЃљрЃљрЃЏрЃљрЃбрЃћ рЃљрЃ«рЃљрЃџрЃў рЃЋрЃћрЃџрЃў: "grade".

рЃерЃћрЃфрЃЋрЃљрЃџрЃћ "age" рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ.

рЃгрЃљрЃерЃљрЃџрЃћ рЃћрЃарЃЌрЃў key:value рЃгрЃДрЃЋрЃўрЃџрЃў.

рЃЊрЃљрЃЉрЃћрЃГрЃЊрЃћ рЃДрЃЋрЃћрЃџрЃљ key().
