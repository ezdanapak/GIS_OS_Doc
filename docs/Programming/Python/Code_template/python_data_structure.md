­ЪЊў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃАрЃ«рЃЋрЃљ рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃћрЃЉрЃў Python-рЃерЃў
рЃАрЃўрЃЏрЃарЃљрЃЋрЃџрЃћ (set) рЃЊрЃљ рЃџрЃћрЃЦрЃАрЃўрЃЎрЃЮрЃюрЃў (dictionary)
+ рЃАрЃљрЃЏрЃБрЃерЃљрЃЮ рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў рЃЊрЃљ рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃў
1) ­Ъћ░ рЃерЃћрЃАрЃљрЃЋрЃљрЃџрЃў
Python-рЃерЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃћрЃЉрЃў рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЮрЃарЃњрЃљрЃюрЃўрЃќрЃћрЃЉрЃБрЃџрЃљрЃЊ рЃерЃћрЃАрЃљрЃюрЃљрЃ«рЃљрЃЊ рЃЊрЃљ рЃЏрЃЮрЃАрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃџрЃљрЃЊ. рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃ«рЃерЃўрЃарЃљрЃЊ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ:

list Рђћ рЃАрЃўрЃљ

tuple Рђћ рЃбрЃБрЃърЃџрЃў

set Рђћ рЃАрЃўрЃЏрЃарЃљрЃЋрЃџрЃћ

dictionary Рђћ рЃџрЃћрЃЦрЃАрЃўрЃЎрЃЮрЃюрЃў

рЃљрЃЏ рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃерЃў рЃЋрЃўрЃЏрЃАрЃ»рЃћрЃџрЃћрЃЉрЃЌ рЃАрЃўрЃЏрЃарЃљрЃЋрЃџрЃћрЃАрЃљ рЃЊрЃљ рЃџрЃћрЃЦрЃАрЃўрЃЎрЃЮрЃюрЃќрЃћ, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃБрЃцрЃарЃЮ рЃарЃЌрЃБрЃџ, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃФрЃљрЃџрЃќрЃћрЃЊ рЃАрЃљрЃАрЃљрЃарЃњрЃћрЃЉрЃџрЃЮ рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃћрЃЉрЃА рЃгрЃљрЃарЃЏрЃЮрЃљрЃЊрЃњрЃћрЃюрЃћрЃю.

2) ­Ъћи рЃАрЃўрЃЏрЃарЃљрЃЋрЃџрЃћ (Set)
2.1 рЃарЃљ рЃљрЃарЃўрЃА Set?
Set (рЃАрЃўрЃЏрЃарЃљрЃЋрЃџрЃћ) Рђћ рЃљрЃарЃўрЃА рЃљрЃарЃљрЃЕрЃљрЃюрЃБрЃАрЃбрЃўрЃџрЃў, рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃарЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃЎрЃЮрЃџрЃћрЃЦрЃфрЃўрЃљ.

­ЪћЉ рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃџрЃћрЃЉрЃў
рЃљрЃа рЃўрЃюрЃљрЃ«рЃљрЃЋрЃА рЃЊрЃБрЃЉрЃџрЃўрЃарЃћрЃЉрЃБрЃџ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃА

рЃЊрЃљрЃџрЃљрЃњрЃћрЃЉрЃБрЃџрЃў рЃљрЃарЃљрЃљ

рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃќрЃћ рЃўрЃюрЃЊрЃћрЃЦрЃАрЃўрЃЌ рЃгрЃЋрЃЊрЃЮрЃЏрЃљ рЃерЃћрЃБрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ

рЃфрЃЋрЃљрЃџрЃћрЃЉрЃљрЃЊрЃў рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљрЃљ, рЃерЃћрЃњрЃЋрЃўрЃФрЃџрЃўрЃљ рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ/рЃљрЃЏрЃЮрЃдрЃћрЃЉрЃљ

2.2 рЃАрЃўрЃЏрЃарЃљрЃЋрЃџрЃћрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ
fruits = {"apple", "banana", "orange"}
numbers = {1, 2, 3, 4}
рЃфрЃљрЃарЃўрЃћрЃџрЃў set:

empty_set = set()
Рџа№ИЈ {} Рђћ рЃЦрЃЏрЃюрЃўрЃА рЃфрЃљрЃарЃўрЃћрЃџ dictionary-рЃА, рЃљрЃарЃљ set-рЃА!

2.3 рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ
fruits.add("mango")
2.4 рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃљрЃЏрЃЮрЃфрЃџрЃљ
fruits.remove("banana")   # рЃЌрЃБ рЃљрЃа рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА Рєњ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ
fruits.discard("banana")  # рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃљ, рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљрЃА рЃљрЃа рЃўрЃФрЃџрЃћрЃЋрЃљ
2.5 рЃАрЃўрЃЏрЃарЃљрЃЋрЃџрЃћрЃћрЃЉрЃўрЃА рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃў
рЃњрЃљрЃћрЃарЃЌрЃўрЃљрЃюрЃћрЃЉрЃљ:
A | B
A.union(B)
рЃњрЃљрЃЊрЃљрЃЎрЃЋрЃћрЃЌрЃљ:
A & B
A.intersection(B)
рЃАрЃ«рЃЋрЃљрЃЮрЃЉрЃљ:
A - B
A.difference(B)
рЃАрЃўрЃЏрЃћрЃбрЃарЃўрЃБрЃџрЃў рЃАрЃ«рЃЋрЃљрЃЮрЃЉрЃљ:
A ^ B
A.symmetric_difference(B)
3) ­ЪћХ рЃџрЃћрЃЦрЃАрЃўрЃЎрЃЮрЃюрЃў (Dictionary)
3.1 рЃарЃљ рЃљрЃарЃўрЃА Dictionary?
Dictionary (рЃџрЃћрЃЦрЃАрЃўрЃЎрЃЮрЃюрЃў) Рђћ рЃгрЃљрЃарЃЏрЃЮрЃљрЃЊрЃњрЃћрЃюрЃА key:value рЃгрЃДрЃЋрЃўрЃџрЃћрЃЉрЃўрЃА рЃЎрЃЮрЃџрЃћрЃЦрЃфрЃўрЃљрЃА.

­ЪћЉ рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃџрЃћрЃЉрЃў
Key рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃарЃў

Value рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃњрЃљрЃюрЃЏрЃћрЃЮрЃарЃЊрЃћрЃА

Key-рЃћрЃЉрЃў рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЋрЃюрЃћрЃю рЃБрЃфрЃЋрЃџрЃљрЃЊрЃў (str, int, tuple, ...)

3.2 рЃџрЃћрЃЦрЃАрЃўрЃЎрЃЮрЃюрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ
student = {
    "name": "Giorgi",
    "age": 21,
    "faculty": "Computer Science"
}
3.3 рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃќрЃћ рЃгрЃЋрЃЊрЃЮрЃЏрЃљ
print(student["name"])
print(student.get("age"))
3.4 рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃўрЃА рЃерЃћрЃфрЃЋрЃџрЃљ
student["age"] = 22
3.5 рЃљрЃ«рЃљрЃџрЃў рЃгрЃДрЃЋрЃўрЃџрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ
student["grade"] = 97
3.6 рЃгрЃДрЃЋрЃўрЃџрЃўрЃА рЃгрЃљрЃерЃџрЃљ
del student["age"]
student.pop("faculty")
3.7 рЃџрЃћрЃЦрЃАрЃўрЃЎрЃЮрЃюрЃўрЃА рЃЏрЃћрЃЌрЃЮрЃЊрЃћрЃЉрЃў
student.keys()      # key-рЃћрЃЉрЃў
student.values()    # values
student.items()     # key-value рЃгрЃДрЃЋрЃўрЃџрЃћрЃЉрЃў
student.clear()     # рЃџрЃћрЃЦрЃАрЃўрЃЎрЃЮрЃюрЃўрЃА рЃњрЃљрЃАрЃБрЃцрЃЌрЃљрЃЋрЃћрЃЉрЃљ
4) ­Ъћх Set vs Dictionary Рђћ рЃерЃћрЃЊрЃљрЃарЃћрЃЉрЃљ
рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃћрЃџрЃў	Set	Dictionary
рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ	рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃЎрЃЮрЃџрЃћрЃЦрЃфрЃўрЃљ	key:value рЃгрЃДрЃЋрЃўрЃџрЃћрЃЉрЃў
рЃЊрЃБрЃЉрЃџрЃўрЃарЃћрЃЉрЃљ	рЃљрЃа рЃљрЃЦрЃЋрЃА	key Рђћ рЃљрЃарЃљ ; value Рђћ рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ
рЃгрЃЋрЃЊрЃЮрЃЏрЃљ	рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃќрЃћ рЃўрЃюрЃЊрЃћрЃЦрЃАрЃўрЃЌ рЃљрЃарЃљ	key-рЃўрЃЌ
рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ	рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃў, рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃарЃЮрЃЉрЃљ	рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃерЃћрЃюрЃљрЃ«рЃЋрЃљ
рЃАрЃўрЃюрЃбрЃљрЃЦрЃАрЃў	{1, 2, 3}	{"a":1, "b":2}
5) ­ЪДЕ 30 рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃБрЃџрЃў рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ
A. рЃљрЃЏрЃЮрЃфрЃљрЃюрЃћрЃЉрЃў Рђћ Set (1РђЊ15)
рЃерЃћрЃЦрЃЏрЃћрЃюрЃў set, рЃЊрЃљрЃљрЃЏрЃљрЃбрЃћ рЃЏрЃљрЃАрЃерЃў 2 рЃљрЃ«рЃљрЃџрЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў.

рЃгрЃљрЃерЃљрЃџрЃћ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў set-рЃўрЃЊрЃљрЃю remove() рЃЏрЃћрЃЌрЃЮрЃЊрЃўрЃЌ.

рЃгрЃљрЃерЃљрЃџрЃћ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў discard()-рЃўрЃЌ рЃўрЃАрЃћ, рЃарЃЮрЃЏ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ рЃљрЃа рЃЏрЃЮрЃ«рЃЊрЃћрЃА.

рЃерЃћрЃљрЃЊрЃљрЃарЃћ рЃЮрЃарЃў set: рЃњрЃљрЃЊрЃљрЃЎрЃЋрЃћрЃЌрЃљ.

рЃерЃћрЃљрЃЊрЃљрЃарЃћ рЃЮрЃарЃў set: рЃњрЃљрЃћрЃарЃЌрЃўрЃљрЃюрЃћрЃЉрЃљ.

рЃерЃћрЃљрЃЊрЃљрЃарЃћ рЃЮрЃарЃў set: рЃАрЃ«рЃЋрЃљрЃЮрЃЉрЃљ.

рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў рЃљрЃарЃўрЃА рЃЌрЃБ рЃљрЃарЃљ set-рЃерЃў.

set-рЃерЃў рЃЕрЃљрЃгрЃћрЃарЃћ 10 рЃарЃўрЃфрЃ«рЃЋрЃў, рЃгрЃљрЃерЃљрЃџрЃћ рЃДрЃЋрЃћрЃџрЃљ рЃџрЃБрЃгрЃў.

рЃЮрЃарЃў set-рЃўрЃА рЃАрЃўрЃЏрЃћрЃбрЃарЃўрЃБрЃџрЃў рЃАрЃ«рЃЋрЃљрЃЮрЃЉрЃўрЃА рЃърЃЮрЃЋрЃюрЃљ.

set-рЃўрЃА рЃљрЃАрЃџрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ copy().

list-рЃўрЃЊрЃљрЃю рЃњрЃљрЃљрЃЎрЃћрЃЌрЃћ set рЃЊрЃљ рЃљрЃЏрЃЮрЃўрЃдрЃћ рЃЊрЃБрЃЉрЃџрЃўрЃЎрЃљрЃбрЃћрЃЉрЃў.

рЃЮрЃарЃў set-рЃўрЃА рЃерЃћрЃЊрЃљрЃарЃћрЃЉрЃљ: рЃљрЃарЃўрЃА рЃЌрЃБ рЃљрЃарЃљ рЃћрЃарЃЌрЃў рЃЏрЃћрЃЮрЃарЃўрЃА рЃЦрЃЋрЃћрЃАрЃўрЃЏрЃарЃљрЃЋрЃџрЃћ.

set-рЃўрЃЊрЃљрЃю pop() рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃљрЃЏрЃЮрЃдрЃћрЃЉрЃљ.

set-рЃўрЃА рЃњрЃљрЃАрЃБрЃцрЃЌрЃљрЃЋрЃћрЃЉрЃљ clear().

рЃЏрЃљрЃАрЃўрЃЋрЃў Рєњ рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃарЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃЌ set Рєњ рЃўрЃАрЃћрЃЋ list.
