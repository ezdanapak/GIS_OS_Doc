# Python-рЃўрЃА рЃАрЃљрЃцрЃБрЃФрЃЋрЃџрЃћрЃЉрЃў: рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃў, рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃбрЃўрЃърЃћрЃЉрЃў рЃЊрЃљ рЃљрЃарЃўрЃЌрЃЏрЃћрЃбрЃўрЃЎрЃБрЃџрЃў рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃў

 рЃљрЃЦ рЃњрЃљрЃюрЃ«рЃўрЃџрЃБрЃџрЃўрЃљ рЃарЃћрЃќрЃћрЃарЃЋрЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃАрЃўрЃбрЃДрЃЋрЃћрЃЉрЃў, рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃўрЃА рЃЊрЃљрЃарЃЦрЃЏрЃћрЃЋрЃљ, рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃбрЃўрЃърЃћрЃЉрЃў рЃЊрЃљ рЃљрЃарЃўрЃЌрЃЏрЃћрЃбрЃўрЃЎрЃБрЃџрЃў рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃўрЃА рЃърЃарЃўрЃЮрЃарЃўрЃбрЃћрЃбрЃћрЃЉрЃў рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃўрЃЌ.

## рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃў рЃЊрЃљ рЃЏрЃљрЃЌрЃў рЃбрЃўрЃърЃћрЃЉрЃў
**рЃфрЃЋрЃџрЃљрЃЊрЃў** рЃљрЃарЃўрЃА рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃљрЃерЃў рЃљрЃЊрЃњрЃўрЃџрЃў, рЃАрЃљрЃЊрЃљрЃф рЃЋрЃўрЃюрЃљрЃ«рЃљрЃЋрЃЌ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃА.  

рЃфрЃЋрЃџрЃљрЃЊрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃћрЃЉрЃў рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА рЃљрЃдрЃгрЃћрЃарЃўрЃЌрЃў, рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃЊрЃљ рЃњрЃљрЃАрЃљрЃњрЃћрЃЉрЃў. рЃљрЃў рЃарЃљрЃЏрЃЊрЃћрЃюрЃўрЃЏрЃћ рЃгрЃћрЃАрЃў:
- рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃљрЃАрЃЮрЃћрЃЉрЃў (a-z, A-Z), рЃфрЃўрЃцрЃарЃћрЃЉрЃў (0-9) рЃЊрЃљ рЃЦрЃЋрЃћрЃЊрЃљ рЃ«рЃљрЃќрЃў `_`.
- рЃАрЃљрЃ«рЃћрЃџрЃў рЃљрЃа рЃБрЃюрЃЊрЃљ рЃўрЃгрЃДрЃћрЃЉрЃЮрЃЊрЃћрЃА рЃфрЃўрЃцрЃарЃўрЃЌ.
- рЃљрЃа рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮрЃЌ рЃарЃћрЃќрЃћрЃарЃЋрЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃАрЃўрЃбрЃДрЃЋрЃћрЃЉрЃў.
- рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ `snake_case` рЃАрЃбрЃўрЃџрЃў (рЃАрЃўрЃбрЃДрЃЋрЃћрЃЉрЃў рЃњрЃљрЃЏрЃЮрЃДрЃЮрЃцрЃўрЃџрЃўрЃљ `_`-рЃўрЃЌ) Python-рЃерЃў.

1. рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃфрЃ«рЃљрЃЊрЃћрЃЉрЃљ

рЃљрЃдрЃгрЃћрЃарЃљ: рЃфрЃЋрЃџрЃљрЃЊрЃў рЃљрЃарЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃюрЃљрЃ«рЃљрЃЋрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃА. рЃърЃўрЃЌрЃЮрЃюрЃерЃў рЃфрЃЋрЃџрЃљрЃЊрЃўрЃА рЃњрЃљрЃЏрЃЮрЃфрЃ«рЃљрЃЊрЃћрЃЉрЃљ рЃ«рЃЊрЃћрЃЉрЃљ рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ, рЃБрЃЉрЃарЃљрЃџрЃЮрЃЊ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃўрЃА рЃЏрЃўрЃюрЃўрЃГрЃћрЃЉрЃўрЃЌ (=).
рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:# рЃфрЃЋрЃџрЃљрЃЊрЃўрЃА рЃњрЃљрЃЏрЃЮрЃфрЃ«рЃљрЃЊрЃћрЃЉрЃљ
name = "рЃњрЃўрЃЮрЃарЃњрЃў"  # рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃў
age = 20         # рЃЏрЃЌрЃћрЃџрЃў рЃарЃўрЃфрЃ«рЃЋрЃў (int)
height = 1.75    # рЃгрЃўрЃџрЃљрЃЊрЃў рЃарЃўрЃфрЃ«рЃЋрЃў (float)
print(name, age, height)  # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: рЃњрЃўрЃЮрЃарЃњрЃў 20 1.75


рЃњрЃљрЃюрЃЏрЃљрЃарЃбрЃћрЃЉрЃљ: 
рЃфрЃЋрЃџрЃљрЃЊрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃў рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА рЃљрЃдрЃгрЃћрЃарЃўрЃЌрЃў, рЃЏрЃљрЃњ.: age рЃљрЃАрЃљрЃЎрЃўрЃАрЃЌрЃЋрЃўрЃА.
рЃфрЃЋрЃџрЃљрЃЊрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃў рЃљрЃа рЃБрЃюрЃЊрЃљ рЃўрЃгрЃДрЃћрЃЉрЃЮрЃЊрЃћрЃА рЃфрЃўрЃцрЃарЃўрЃЌ рЃЊрЃљ рЃљрЃа рЃБрЃюрЃЊрЃљ рЃерЃћрЃўрЃфрЃљрЃЋрЃЊрЃћрЃА рЃАрЃърЃћрЃфрЃўрЃљрЃџрЃБрЃа рЃАрЃўрЃЏрЃЉрЃЮрЃџрЃЮрЃћрЃЉрЃА (рЃњрЃљрЃарЃЊрЃљ _).
рЃърЃўрЃЌрЃЮрЃюрЃў рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљрЃЋрЃА рЃфрЃЋрЃџрЃљрЃЊрЃўрЃА рЃбрЃўрЃърЃА.



2. рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃбрЃўрЃърЃћрЃЉрЃў

рЃљрЃдрЃгрЃћрЃарЃљ: рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃА рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃ░рЃЦрЃЮрЃюрЃЊрЃћрЃЌ рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃбрЃўрЃърЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў. рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃбрЃўрЃърЃћрЃЉрЃўрЃљ:
int: рЃЏрЃЌрЃћрЃџрЃў рЃарЃўрЃфрЃ«рЃЋрЃћрЃЉрЃў (рЃЏрЃљрЃњ.: 5, -10)
float: рЃгрЃўрЃџрЃљрЃЊрЃў рЃарЃўрЃфрЃ«рЃЋрЃћрЃЉрЃў (рЃЏрЃљрЃњ.: 3.14, -0.5)
str: рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃћрЃЉрЃў (рЃЏрЃљрЃњ.: "рЃњрЃљрЃЏрЃљрЃарЃ»рЃЮрЃЉрЃљ")
bool: рЃџрЃЮрЃњрЃўрЃЎрЃБрЃарЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃў (True, False)
list: рЃАрЃўрЃћрЃЉрЃў (рЃЏрЃљрЃњ.: [1, 2, 3])
tuple: рЃЎрЃЮрЃарЃбрЃћрЃЪрЃћрЃЉрЃў (рЃЏрЃљрЃњ.: (1, 2))
set: рЃАрЃўрЃЏрЃарЃљрЃЋрЃџрЃћрЃћрЃЉрЃў (рЃЏрЃљрЃњ.: {1, 2, 3})
dict: рЃџрЃћрЃЦрЃАрЃўрЃЎрЃЮрЃюрЃћрЃЉрЃў (рЃЏрЃљрЃњ.: {"name": "рЃљрЃюрЃљ", "age": 25})


рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:x = 10          # int
y = 2.5         # float
text = "рЃАрЃљрЃџрЃљрЃЏрЃў" # str
is_valid = True # bool
print(type(x), type(y), type(text), type(is_valid))  # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: <class 'int'> <class 'float'> <class 'str'> <class 'bool'>


рЃњрЃљрЃюрЃЏрЃљрЃарЃбрЃћрЃЉрЃљ: 
type() рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ рЃњрЃЋрЃўрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА рЃфрЃЋрЃџрЃљрЃЊрЃўрЃА рЃбрЃўрЃърЃА.
рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃбрЃўрЃърЃў рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљрЃЋрЃА, рЃЌрЃБ рЃарЃљ рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃўрЃА рЃњрЃљрЃЎрЃћрЃЌрЃћрЃЉрЃљ рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃфрЃЋрЃџрЃљрЃЊрЃќрЃћ.



3. рЃарЃћрЃќрЃћрЃарЃЋрЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃАрЃўрЃбрЃДрЃЋрЃћрЃЉрЃў

рЃљрЃдрЃгрЃћрЃарЃљ: рЃърЃўрЃЌрЃЮрЃюрЃА рЃљрЃЦрЃЋрЃА рЃАрЃўрЃбрЃДрЃЋрЃћрЃЉрЃў, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃЊрЃљрЃфрЃБрЃџрЃўрЃљ рЃћрЃюрЃўрЃАрЃЌрЃЋрЃўрЃА рЃЊрЃљ рЃљрЃа рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃЮрЃЌ рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃћрЃЉрЃљрЃЊ. рЃЏрЃљрЃњ.: if, else, for, while, True, False, None, and, or, not, def, class, return рЃЊрЃљ рЃАрЃ«рЃЋ.
рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:# рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃў: рЃарЃћрЃќрЃћрЃарЃЋрЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃАрЃўрЃбрЃДрЃЋрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ
if = 10  # рЃћрЃА рЃњрЃљрЃЏрЃЮрЃўрЃгрЃЋрЃћрЃЋрЃА рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљрЃА!

рЃњрЃљрЃюрЃЏрЃљрЃарЃбрЃћрЃЉрЃљ:
рЃарЃћрЃќрЃћрЃарЃЋрЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃАрЃўрЃбрЃДрЃЋрЃћрЃЉрЃўрЃА рЃАрЃарЃБрЃџрЃў рЃАрЃўрЃљ: import keyword; print(keyword.kwlist).
рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃА рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ рЃљрЃдрЃгрЃћрЃарЃўрЃЌрЃў рЃАрЃљрЃ«рЃћрЃџрЃћрЃЉрЃў, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃљрЃа рЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљ рЃљрЃЏ рЃАрЃўрЃбрЃДрЃЋрЃћрЃЉрЃА.



## рЃљрЃарЃўрЃЌрЃЏрЃћрЃбрЃўрЃЎрЃБрЃџрЃў рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃў

рЃљрЃдрЃгрЃћрЃарЃљ: рЃърЃўрЃЌрЃЮрЃюрЃў рЃњрЃЋрЃљрЃФрЃџрЃћрЃЋрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА, рЃерЃћрЃЋрЃљрЃАрЃарЃБрЃџрЃЮрЃЌ рЃЏрЃљрЃЌрЃћрЃЏрЃљрЃбрЃўрЃЎрЃБрЃарЃў рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃў:  
+ (рЃЏрЃўрЃЏрЃљрЃбрЃћрЃЉрЃљ)
- (рЃњрЃљрЃЏрЃЮрЃЎрЃџрЃћрЃЉрЃљ)

## рЃерЃћрЃЏрЃЮрЃЎрЃџрЃћрЃЉрЃБрЃџрЃў рЃљрЃарЃўрЃЌрЃЏрЃћрЃбрЃўрЃЎрЃБрЃџрЃў рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃў

*= (рЃњрЃљрЃЏрЃарЃљрЃЋрЃџрЃћрЃЉрЃљ рЃЊрЃљ рЃЏрЃўрЃюрЃўрЃГрЃћрЃЉрЃљ)
/= (рЃњрЃљрЃДрЃЮрЃцрЃљ рЃЊрЃљ рЃЏрЃўрЃюрЃўрЃГрЃћрЃЉрЃљ)
//= (рЃЏрЃЌрЃћрЃџрЃў рЃњрЃљрЃДрЃЮрЃцрЃљ рЃЊрЃљ рЃЏрЃўрЃюрЃўрЃГрЃћрЃЉрЃљ)
%= (рЃюрЃљрЃерЃЌрЃў рЃЊрЃљ рЃЏрЃўрЃюрЃўрЃГрЃћрЃЉрЃљ)
**= (рЃ«рЃљрЃарЃўрЃАрЃ«рЃў рЃЊрЃљ рЃЏрЃўрЃюрЃўрЃГрЃћрЃЉрЃљ)

## рЃљрЃарЃўрЃЌрЃЏрЃћрЃбрЃўрЃЎрЃБрЃџрЃў рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃўрЃА рЃърЃарЃўрЃЮрЃарЃўрЃбрЃћрЃбрЃћрЃЉрЃў

рЃљрЃдрЃгрЃћрЃарЃљ: рЃърЃўрЃЌрЃЮрЃюрЃў рЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃА рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃА рЃњрЃљрЃарЃЎрЃЋрЃћрЃБрЃџрЃў рЃЌрЃљрЃюрЃЏрЃўрЃЏрЃЊрЃћрЃЋрЃарЃЮрЃЉрЃўрЃЌ:
() РђЊ рЃцрЃарЃЕрЃ«рЃўрЃџрЃћрЃЉрЃў
** РђЊ рЃ«рЃљрЃарЃўрЃАрЃ«рЃў

## рЃљрЃЏрЃЮрЃфрЃљрЃюрЃћрЃЉрЃў

total /= 3   # total = 10.0
print("рЃАрЃљрЃЉрЃЮрЃџрЃЮрЃЮ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ:", total)  # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: рЃАрЃљрЃЉрЃЮрЃџрЃЮрЃЮ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ: 10.0



рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ : рЃърЃарЃўрЃЮрЃарЃўрЃбрЃћрЃбрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ

рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃљ: рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃљрЃџрЃћрЃЌ рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃљ 5 + 2 * 3 ** 2 рЃЮрЃарЃ»рЃћрЃа: рЃћрЃарЃЌрЃ«рЃћрЃџ рЃцрЃарЃЕрЃ«рЃўрЃџрЃћрЃЉрЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ рЃЊрЃљ рЃЏрЃћрЃЮрЃарЃћрЃЊ рЃцрЃарЃЕрЃ«рЃўрЃџрЃћрЃЉрЃўрЃЌ (5 + 2) * 3 ** 2. рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃћрЃЌ рЃЮрЃарЃўрЃЋрЃћ рЃерЃћрЃЊрЃћрЃњрЃў.
рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:result1 = 5 + 2 * 3 ** 2  # 3 ** 2 = 9, 2 * 9 = 18, 5 + 18 = 23
result2 = (5 + 2) * 3 ** 2  # 5 + 2 = 7, 3 ** 2 = 9, 7 * 9 = 63
print("рЃцрЃарЃЕрЃ«рЃўрЃџрЃћрЃЉрЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ:", result1)  # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: 23
print("рЃцрЃарЃЕрЃ«рЃўрЃџрЃћрЃЉрЃўрЃЌ:", result2)       # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: 63



рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ : рЃфрЃЋрЃџрЃљрЃЊрЃўрЃА рЃбрЃўрЃърЃўрЃА рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃљ

рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃљ: рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃарЃљрЃЏрЃЊрЃћрЃюрЃўрЃЏрЃћ рЃфрЃЋрЃџрЃљрЃЊрЃў (рЃЏрЃљрЃњ.: num, text, flag) рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃбрЃўрЃърЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃЌ (int, str, bool). рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ type() рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ рЃЏрЃљрЃЌрЃў рЃбрЃўрЃърЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃАрЃљрЃбрЃљрЃюрЃљрЃЊ.
рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:num = 42
text = "рЃърЃўрЃЌрЃЮрЃюрЃў"
flag = True
print("num-рЃўрЃА рЃбрЃўрЃърЃў:", type(num))   # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: <class 'int'>
print("text-рЃўрЃА рЃбрЃўрЃърЃў:", type(text)) # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: <class 'str'>
print("flag-рЃўрЃА рЃбрЃўрЃърЃў:", type(flag)) # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: <class 'bool'>



!!!tip "рЃарЃЕрЃћрЃЋрЃћрЃЉрЃў:"
    рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃћрЃЉрЃў рЃњрЃљрЃљрЃЎрЃћрЃЌрЃћрЃЌ рЃљрЃдрЃгрЃћрЃарЃўрЃЌрЃў, рЃЏрЃљрЃњ.: student_name рЃБрЃЎрЃћрЃЌрЃћрЃАрЃўрЃљ, рЃЋрЃўрЃЊрЃарЃћ x. <br>
    рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ print() рЃ«рЃерЃўрЃарЃљрЃЊ, рЃарЃљрЃЌрЃљ рЃюрЃљрЃ«рЃЮрЃЌ, рЃарЃљ рЃ«рЃЊрЃћрЃЉрЃљ рЃЌрЃЦрЃЋрЃћрЃюрЃА рЃЎрЃЮрЃЊрЃерЃў. <br>
    рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃўрЃА рЃърЃарЃўрЃЮрЃарЃўрЃбрЃћрЃбрЃћрЃЉрЃў рЃарЃЌрЃБрЃџрЃў рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃћрЃЉрЃўрЃАрЃљрЃА рЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ рЃцрЃарЃЕрЃ«рЃўрЃџрЃћрЃЉрЃў рЃАрЃўрЃфрЃ«рЃљрЃЊрЃўрЃАрЃЌрЃЋрЃўрЃА. <br>
    рЃЌрЃБ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ рЃњрЃљрЃЦрЃЋрЃЌ, рЃгрЃљрЃўрЃЎрЃўрЃЌрЃ«рЃћрЃЌ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃўрЃА рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃљ РђЊ рЃўрЃА рЃ«рЃерЃўрЃарЃљрЃЊ рЃЏрЃўрЃњрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃЌ рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃўрЃА рЃЏрЃўрЃќрЃћрЃќрЃќрЃћ. <br>



рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ : рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃфрЃ«рЃљрЃЊрЃћрЃЉрЃљ рЃЊрЃљ рЃбрЃўрЃърЃўрЃА рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃљ

рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃЮрЃЌрЃ«рЃў рЃфрЃЋрЃџрЃљрЃЊрЃў: name (рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃў), age (рЃЏрЃЌрЃћрЃџрЃў рЃарЃўрЃфрЃ«рЃЋрЃў), height (рЃгрЃўрЃџрЃљрЃЊрЃў рЃарЃўрЃфрЃ«рЃЋрЃў), is_student (рЃЉрЃБрЃџрЃћрЃљрЃюрЃў). рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃћрЃЌ рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃфрЃЋрЃџрЃљрЃЊрЃўрЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ рЃЊрЃљ рЃЏрЃўрЃАрЃў рЃбрЃўрЃърЃў type() рЃцрЃБрЃюрЃЦрЃфрЃўрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ.
рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:name = "рЃЏрЃљрЃарЃўрЃљрЃЏрЃў"
age = 22
height = 1.65
is_student = True
print("рЃАрЃљрЃ«рЃћрЃџрЃў:", name, "рЃбрЃўрЃърЃў:", type(name))      # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: рЃАрЃљрЃ«рЃћрЃџрЃў: рЃЏрЃљрЃарЃўрЃљрЃЏрЃў рЃбрЃўрЃърЃў: <class 'str'>
print("рЃљрЃАрЃљрЃЎрЃў:", age, "рЃбрЃўрЃърЃў:", type(age))         # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: рЃљрЃАрЃљрЃЎрЃў: 22 рЃбрЃўрЃърЃў: <class 'int'>
print("рЃАрЃўрЃЏрЃљрЃдрЃџрЃћ:", height, "рЃбрЃўрЃърЃў:", type(height)) # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: рЃАрЃўрЃЏрЃљрЃдрЃџрЃћ: 1.65 рЃбрЃўрЃърЃў: <class 'float'>
print("рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃўрЃљ:", is_student, "рЃбрЃўрЃърЃў:", type(is_student)) # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃўрЃљ: True рЃбрЃўрЃърЃў: <class 'bool'>


рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃљ: рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ рЃљрЃдрЃгрЃћрЃарЃўрЃЌрЃў рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃћрЃЉрЃў. <br>
рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ, рЃарЃЮрЃЏ рЃфрЃЋрЃџрЃљрЃЊрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃў рЃљрЃа рЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљ рЃарЃћрЃќрЃћрЃарЃЋрЃўрЃарЃћрЃЉрЃБрЃџ рЃАрЃўрЃбрЃДрЃЋрЃћрЃЉрЃА рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ: <br>

- [x] True
- [x] False

рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ : рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃљрЃарЃўрЃЌрЃЏрЃћрЃбрЃўрЃЎрЃБрЃџрЃў рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃў

рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃЮрЃарЃў рЃфрЃЋрЃџрЃљрЃЊрЃў 
рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ рЃДрЃЋрЃћрЃџрЃљ рЃљрЃарЃўрЃЌрЃЏрЃћрЃбрЃўрЃЎрЃБрЃџрЃў рЃЮрЃърЃћрЃарЃљрЃбрЃЮрЃарЃў (+, -, *, /, //, %, **) рЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃћрЃЌ рЃерЃћрЃЊрЃћрЃњрЃћрЃЉрЃў.
рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:

```py title="Simple Arithmetic Operations.py" linenums="1"
a = 15
b = 4
print("рЃЏрЃўрЃЏрЃљрЃбрЃћрЃЉрЃљ:", a + b)      # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: 19
print("рЃњрЃљрЃЏрЃЮрЃЎрЃџрЃћрЃЉрЃљ:", a - b)     # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: 11
print("рЃњрЃљрЃЏрЃарЃљрЃЋрЃџрЃћрЃЉрЃљ:", a * b)    # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: 60
print("рЃњрЃљрЃДрЃЮрЃцрЃљ:", a / b)        # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: 3.75
print("рЃЏрЃЌрЃћрЃџрЃў рЃњрЃљрЃДрЃЮрЃцрЃљ:", a // b) # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: 3
print("рЃюрЃљрЃерЃЌрЃў:", a % b)          # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: 3
print("рЃ«рЃљрЃарЃўрЃАрЃ«рЃў:", a ** b)       # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: 50625

a = 10 
b = 3
print("Addition:", a + b) # 13 
print("Subtraction:", a - b) # 7 
print("Multiplication:", a * b) # 30 
print("Division:", a / b) # 3.3333333333333335 
print("Floor division:", a // b) # 3 
print("Modulus:", a % b) # 1 
print("Exponentiation:", a ** b) # 1000
```

!!! note "рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃљ:" 
    рЃДрЃБрЃарЃљрЃЊрЃдрЃћрЃЉрЃљ рЃЏрЃўрЃљрЃЦрЃфрЃўрЃћрЃЌ, рЃарЃЮрЃЏ / рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА float-рЃА, рЃ«рЃЮрЃџрЃЮ // рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА int-рЃА.

рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ : рЃерЃћрЃЏрЃЮрЃЎрЃџрЃћрЃЉрЃБрЃџрЃў рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ

рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃфрЃЋрЃџрЃљрЃЊрЃў score = 100. рЃњрЃљрЃюрЃљрЃљрЃ«рЃџрЃћрЃЌ рЃЏрЃўрЃАрЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃўрЃЌ: рЃњрЃљрЃќрЃљрЃарЃЊрЃћрЃЌ 20-рЃўрЃЌ, рЃњрЃљрЃљрЃЏрЃарЃљрЃЋрЃџрЃћрЃЌ 2-рЃќрЃћ, рЃњрЃљрЃДрЃљрЃЋрЃўрЃЌ 5-рЃќрЃћ, рЃерЃћрЃљрЃЏрЃфрЃўрЃарЃћрЃЌ 10-рЃўрЃЌ. рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ рЃерЃћрЃЏрЃЮрЃЎрЃџрЃћрЃЉрЃБрЃџрЃў рЃЮрЃърЃћрЃарЃљрЃбрЃЮрЃарЃћрЃЉрЃў (+=, *=, /=, -=).
рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:score = 100
score += 20   # score = 120
score *= 2    # score = 240
score /= 5    # score = 48.0
score -= 10   # score = 38.0
print("рЃАрЃљрЃЉрЃЮрЃџрЃЮрЃЮ рЃЦрЃБрЃџрЃљ:", score)  # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: рЃАрЃљрЃЉрЃЮрЃџрЃЮрЃЮ рЃЦрЃБрЃџрЃљ: 38.0


рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃљ: рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ, рЃарЃЮрЃЏ /= рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА float-рЃА, рЃЌрЃБ рЃњрЃљрЃДрЃЮрЃцрЃўрЃА рЃерЃћрЃЊрЃћрЃњрЃў рЃгрЃўрЃџрЃљрЃЊрЃўрЃљ.

рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ : рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃўрЃА рЃърЃарЃўрЃЮрЃарЃўрЃбрЃћрЃбрЃћрЃЉрЃў

рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃљрЃџрЃћрЃЌ рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃћрЃЉрЃў:  
10 + 5 * 2  
(10 + 5) * 2  
2 ** 3 * 4 + 1  
2 ** (3 * 4) + 1рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃћрЃЌ рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃерЃћрЃЊрЃћрЃњрЃў рЃЊрЃљ рЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ, рЃарЃљрЃбрЃЮрЃЏ рЃЏрЃўрЃўрЃдрЃћрЃЌ рЃћрЃА рЃерЃћрЃЊрЃћрЃњрЃў.


рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:result1 = 10 + 5 * 2          # 5 * 2 = 10, 10 + 10 = 20
result2 = (10 + 5) * 2        # 10 + 5 = 15, 15 * 2 = 30
result3 = 2 ** 3 * 4 + 1      # 2 ** 3 = 8, 8 * 4 = 32, 32 + 1 = 33
result4 = 2 ** (3 * 4) + 1    # 3 * 4 = 12, 2 ** 12 = 4096, 4096 + 1 = 4097
print("рЃерЃћрЃЊрЃћрЃњрЃў 1:", result1)   # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: 20
print("рЃерЃћрЃЊрЃћрЃњрЃў 2:", result2)   # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: 30
print("рЃерЃћрЃЊрЃћрЃњрЃў 3:", result3)   # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: 33
print("рЃерЃћрЃЊрЃћрЃњрЃў 4:", result4)   # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: 4097


рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃљ: рЃњрЃљрЃўрЃЌрЃЋрЃљрЃџрЃўрЃАрЃгрЃўрЃюрЃћрЃЌ, рЃарЃЮрЃЏ ** рЃљрЃЦрЃЋрЃА рЃБрЃЏрЃљрЃдрЃџрЃћрЃАрЃў рЃърЃарЃўрЃЮрЃарЃўрЃбрЃћрЃбрЃў, рЃерЃћрЃЏрЃЊрЃћрЃњ *, /, //, %, рЃЉрЃЮрЃџрЃЮрЃА +, -. рЃцрЃарЃЕрЃ«рЃўрЃџрЃћрЃЉрЃўрЃЌ рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃерЃћрЃфрЃЋрЃљрЃџрЃЮрЃЌ рЃЌрЃљрЃюрЃЏрЃўрЃЏрЃЊрЃћрЃЋрЃарЃЮрЃЉрЃљ.

рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ : рЃАрЃљрЃерЃБрЃљрЃџрЃЮ рЃљрЃарЃўрЃЌрЃЏрЃћрЃбрЃўрЃЎрЃБрЃџрЃў

рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃЌрЃЋрЃџрЃўрЃА рЃАрЃљрЃЏрЃў рЃарЃўрЃфрЃ«рЃЋрЃўрЃА (num1, num2, num3) рЃАрЃљрЃерЃБрЃљрЃџрЃЮ рЃљрЃарЃўрЃЌрЃЏрЃћрЃбрЃўрЃЎрЃБрЃџрЃА. рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃў рЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃћрЃЌ рЃерЃћрЃЊрЃћрЃњрЃў.
рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:num1 = 10
num2 = 20
num3 = 30
average = (num1 + num2 + num3) / 3
print("рЃАрЃљрЃерЃБрЃљрЃџрЃЮ:", average)  # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: рЃАрЃљрЃерЃБрЃљрЃџрЃЮ: 20.0


рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃљ: рЃњрЃљрЃДрЃЮрЃцрЃљ (/) рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃА рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА float-рЃА.

рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ: рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃўрЃА GPA-рЃА рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃљ

рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃљ: рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃљрЃџрЃћрЃЌ рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃўрЃА GPA (Grade Point Average - рЃАрЃљрЃерЃБрЃљрЃџрЃЮ рЃЦрЃБрЃџрЃљ) рЃАрЃљрЃЏрЃў рЃАрЃљрЃњрЃюрЃўрЃА рЃЦрЃБрЃџрЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ. рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃў grade1, grade2, grade3 рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃўрЃЌ: grade1 = 85, grade2 = 90, grade3 = 78. GPA рЃњрЃљрЃЏрЃЮрЃўрЃЌрЃЋрЃџрЃћрЃЉрЃљ рЃцрЃЮрЃарЃЏрЃБрЃџрЃўрЃЌ: gpa = (grade1 + grade2 + grade3) / 3. рЃњрЃљрЃЏрЃЮрЃљрЃЕрЃўрЃюрЃћрЃЌ GPA-рЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ.

рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:

grade1 = 85
grade2 = 90
grade3 = 78
gpa = (grade1 + grade2 + grade3) / 3  # (85 + 90 + 78) / 3 = 253 / 3 = 84.333...
print("рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃўрЃА GPA:", gpa)  # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃўрЃА GPA: 84.33333333333333


рЃњрЃљрЃюрЃЏрЃљрЃарЃбрЃћрЃЉрЃљ:

рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃў grade1, grade2, grade3 рЃљрЃарЃўрЃА int рЃбрЃўрЃърЃўрЃА, рЃарЃљрЃЊрЃњрЃљрЃю рЃЦрЃБрЃџрЃћрЃЉрЃў рЃЏрЃЌрЃћрЃџрЃў рЃарЃўрЃфрЃ«рЃЋрЃћрЃЉрЃўрЃљ.
рЃцрЃарЃЕрЃ«рЃўрЃџрЃћрЃЉрЃў (grade1 + grade2 + grade3) рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА, рЃарЃЮрЃЏ рЃЏрЃўрЃЏрЃљрЃбрЃћрЃЉрЃљ рЃерЃћрЃАрЃарЃБрЃџрЃЊрЃћрЃА рЃњрЃљрЃДрЃЮрЃцрЃљрЃЏрЃЊрЃћ.
рЃњрЃљрЃДрЃЮрЃцрЃљ (/) рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА float рЃбрЃўрЃърЃА, рЃљрЃЏрЃўрЃбрЃЮрЃЏ рЃерЃћрЃЊрЃћрЃњрЃў рЃгрЃўрЃџрЃљрЃЊрЃў рЃарЃўрЃфрЃ«рЃЋрЃўрЃљ.
GPA-рЃА рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃўрЃАрЃљрЃА рЃАрЃљрЃЏрЃў рЃЦрЃБрЃџрЃўрЃА рЃ»рЃљрЃЏрЃў рЃўрЃДрЃЮрЃцрЃљ рЃАрЃљрЃњрЃюрЃћрЃЉрЃўрЃА рЃарЃљрЃЮрЃЊрЃћрЃюрЃЮрЃЉрЃљрЃќрЃћ (3).

рЃарЃЕрЃћрЃЋрЃћрЃЉрЃў 

рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ рЃљрЃдрЃгрЃћрЃарЃўрЃЌрЃў рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃћрЃЉрЃў, рЃЏрЃљрЃњ.: grade1, gpa, рЃарЃљрЃЌрЃљ рЃЎрЃЮрЃЊрЃў рЃўрЃДрЃЮрЃА рЃњрЃљрЃАрЃљрЃњрЃћрЃЉрЃў.
рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ print() рЃерЃћрЃЊрЃћрЃњрЃўрЃА рЃерЃћрЃАрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃџрЃљрЃЊ.
рЃЌрЃБ рЃњрЃАрЃБрЃарЃЌ, рЃАрЃфрЃљрЃЊрЃћрЃЌ рЃАрЃ«рЃЋрЃљ рЃЦрЃБрЃџрЃћрЃЉрЃў (рЃЏрЃљрЃњ.: 92, 76, 88) рЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃљрЃџрЃћрЃЌ рЃљрЃ«рЃљрЃџрЃў GPA.
рЃњрЃљрЃўрЃЌрЃЋрЃљрЃџрЃўрЃАрЃгрЃўрЃюрЃћрЃЌ, рЃарЃЮрЃЏ рЃњрЃљрЃДрЃЮрЃцрЃљ (/) рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃА рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА float-рЃА, рЃЏрЃљрЃерЃўрЃюрЃљрЃф рЃЎрЃў, рЃЌрЃБ рЃерЃћрЃЊрЃћрЃњрЃў рЃЏрЃЌрЃћрЃџрЃў рЃарЃўрЃфрЃ«рЃЋрЃўрЃљ.

рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ : рЃ«рЃљрЃарЃўрЃАрЃ«рЃерЃў рЃљрЃДрЃЋрЃљрЃюрЃљ

рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃЌрЃЋрЃџрЃўрЃА рЃарЃўрЃфрЃ«рЃЋрЃўрЃА base power-рЃћ рЃ«рЃљрЃарЃўрЃАрЃ«рЃА ** рЃЮрЃърЃћрЃарЃљрЃбрЃЮрЃарЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ. рЃАрЃфрЃљрЃЊрЃћрЃЌ base = 2 рЃЊрЃљ power = 3, рЃерЃћрЃЏрЃЊрЃћрЃњ base = 5 рЃЊрЃљ power = 4.
рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:base = 2
power = 3
result = base ** power
print(base, "рЃ«рЃљрЃарЃўрЃАрЃ«рЃерЃў", power, ":", result)  # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: 2 рЃ«рЃљрЃарЃўрЃАрЃ«рЃерЃў 3 : 8

base = 5
power = 4
result = base ** power
print(base, "рЃ«рЃљрЃарЃўрЃАрЃ«рЃерЃў", power, ":", result)  # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: 5 рЃ«рЃљрЃарЃўрЃАрЃ«рЃерЃў 4 : 625


рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃљ: ** рЃЮрЃърЃћрЃарЃљрЃбрЃЮрЃарЃў рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃ«рЃљрЃарЃўрЃАрЃ«рЃерЃў рЃљрЃАрЃљрЃДрЃЋрЃљрЃюрЃљрЃЊ. рЃАрЃфрЃљрЃЊрЃћрЃЌ рЃБрЃљрЃарЃДрЃЮрЃцрЃўрЃЌрЃў power-рЃўрЃф.

рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ : рЃфрЃЋрЃџрЃљрЃЊрЃўрЃА рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃљ

рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃфрЃЋрЃџрЃљрЃЊрЃў counter = 0. рЃњрЃљрЃюрЃљрЃљрЃ«рЃџрЃћрЃЌ рЃўрЃА 5-рЃ»рЃћрЃа рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃўрЃЌ: += 2, *= 3, -= 4, /= 2, += 10. рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃћрЃЌ рЃДрЃЮрЃЋрЃћрЃџрЃў рЃюрЃљрЃЉрЃўрЃ»рЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ.
рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:counter = 0
counter += 2
print("рЃюрЃљрЃЉрЃўрЃ»рЃў 1:", counter)  # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: 2
counter *= 3
print("рЃюрЃљрЃЉрЃўрЃ»рЃў 2:", counter)  # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: 6
counter -= 4
print("рЃюрЃљрЃЉрЃўрЃ»рЃў 3:", counter)  # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: 2
counter /= 2
print("рЃюрЃљрЃЉрЃўрЃ»рЃў 4:", counter)  # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: 1.0
counter += 10
print("рЃюрЃљрЃЉрЃўрЃ»рЃў 5:", counter)  # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: 11.0


рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃљ: рЃЌрЃЋрЃљрЃџрЃў рЃљрЃЊрЃћрЃЋрЃюрЃћрЃЌ, рЃарЃЮрЃњрЃЮрЃа рЃўрЃфрЃЋрЃџрЃћрЃЉрЃљ рЃфрЃЋрЃџрЃљрЃЊрЃўрЃА рЃбрЃўрЃърЃў (int Рєњ float) /= рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ.


рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ : рЃарЃћрЃќрЃћрЃарЃЋрЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃАрЃўрЃбрЃДрЃЋрЃћрЃЉрЃўрЃА рЃЌрЃљрЃЋрЃўрЃЊрЃљрЃю рЃљрЃфрЃўрЃџрЃћрЃЉрЃљ

рЃАрЃфрЃљрЃЊрЃћрЃЌ рЃерЃћрЃЦрЃЏрЃюрЃљрЃЌ рЃфрЃЋрЃџрЃљрЃЊрЃў рЃАрЃљрЃ«рЃћрЃџрЃўрЃЌ for рЃљрЃю while рЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃћрЃЌ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ. рЃерЃћрЃЏрЃЊрЃћрЃњ рЃерЃћрЃфрЃЋрЃљрЃџрЃћрЃЌ рЃАрЃљрЃ«рЃћрЃџрЃў рЃљрЃдрЃгрЃћрЃарЃўрЃЌрЃў рЃАрЃљрЃ«рЃћрЃџрЃўрЃЌ (рЃЏрЃљрЃњ.: loop_count) рЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ рЃўрЃА рЃарЃўрЃфрЃ«рЃЋрЃўрЃА рЃњрЃљрЃАрЃљрЃЮрЃарЃЏрЃљрЃњрЃћрЃЉрЃџрЃљрЃЊ.
рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:# рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃў:
# for = 5  # рЃњрЃљрЃЏрЃЮрЃўрЃгрЃЋрЃћрЃЋрЃА SyntaxError

# рЃАрЃгрЃЮрЃарЃў:
loop_count = 5
loop_count *= 2
print("рЃњрЃљрЃЮрЃарЃЏрЃљрЃњрЃћрЃЉрЃБрЃџрЃў:", loop_count)  # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: рЃњрЃљрЃЮрЃарЃЏрЃљрЃњрЃћрЃЉрЃБрЃџрЃў: 10


рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃљ: рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ import keyword; print(keyword.kwlist) рЃарЃћрЃќрЃћрЃарЃЋрЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃАрЃўрЃбрЃДрЃЋрЃћрЃЉрЃўрЃА рЃАрЃљрЃюрЃљрЃ«рЃљрЃЋрЃљрЃЊ.


рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ : рЃюрЃљрЃерЃЌрЃўрЃА рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃљ

рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃљрЃџрЃћрЃЌ, рЃарЃљ рЃюрЃљрЃерЃЌрЃў рЃарЃЕрЃћрЃЉрЃљ, рЃарЃЮрЃфрЃљ рЃарЃўрЃфрЃ«рЃЋрЃў n = 25 рЃўрЃДрЃЮрЃцрЃљ m = 7-рЃќрЃћ. рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ % рЃЮрЃърЃћрЃарЃљрЃбрЃЮрЃарЃў. рЃљрЃАрЃћрЃЋрЃћ рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃљрЃџрЃћрЃЌ рЃЏрЃЌрЃћрЃџрЃў рЃњрЃљрЃДрЃЮрЃцрЃўрЃА рЃерЃћрЃЊрЃћрЃњрЃў (//).


рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:

n = 25
m = 7
remainder = n % m  # 25 ├и 7 = 3 (рЃЏрЃЌрЃћрЃџрЃў) рЃЊрЃљ рЃюрЃљрЃерЃЌрЃў 4
quotient = n // m  # 25 ├и 7 = 3 (рЃЏрЃЌрЃћрЃџрЃў рЃюрЃљрЃгрЃўрЃџрЃў)
print("рЃюрЃљрЃерЃЌрЃў:", remainder)  # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: рЃюрЃљрЃерЃЌрЃў: 4
print("рЃЏрЃЌрЃћрЃџрЃў рЃњрЃљрЃДрЃЮрЃцрЃљ:", quotient)  # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: рЃЏрЃЌрЃћрЃџрЃў рЃњрЃљрЃДрЃЮрЃцрЃљ: 3

рЃњрЃљрЃюрЃЏрЃљрЃарЃбрЃћрЃЉрЃљ:


% рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА рЃюрЃљрЃерЃЌрЃА, рЃ«рЃЮрЃџрЃЮ // рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА рЃњрЃљрЃДрЃЮрЃцрЃўрЃА рЃЏрЃЌрЃћрЃџ рЃюрЃљрЃгрЃўрЃџрЃА.
рЃЮрЃарЃўрЃЋрЃћ рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃўрЃА рЃерЃћрЃЊрЃћрЃњрЃў int рЃбрЃўрЃърЃўрЃАрЃљрЃљ.

рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ : рЃцрЃљрЃарЃћрЃюрЃ░рЃљрЃўрЃбрЃўрЃЊрЃљрЃю рЃфрЃћрЃџрЃАрЃўрЃБрЃАрЃерЃў рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃюрЃљ

рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃбрЃћрЃЏрЃърЃћрЃарЃљрЃбрЃБрЃарЃљ рЃцрЃљрЃарЃћрЃюрЃ░рЃљрЃўрЃбрЃўрЃЊрЃљрЃю рЃфрЃћрЃџрЃАрЃўрЃБрЃАрЃерЃў рЃцрЃЮрЃарЃЏрЃБрЃџрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ: C = (F - 32) * 5 / 9. рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ рЃфрЃЋрЃџрЃљрЃЊрЃў fahrenheit = 68. рЃњрЃљрЃЏрЃЮрЃљрЃЕрЃўрЃюрЃћрЃЌ рЃфрЃћрЃџрЃАрЃўрЃБрЃАрЃўрЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ.

рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:

fahrenheit = 68
celsius = (fahrenheit - 32) * 5 / 9  # (68 - 32) * 5 / 9 = 36 * 5 / 9 = 20
print("рЃфрЃћрЃџрЃАрЃўрЃБрЃАрЃў:", celsius)  # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: рЃфрЃћрЃџрЃАрЃўрЃБрЃАрЃў: 20.0

рЃњрЃљрЃюрЃЏрЃљрЃарЃбрЃћрЃЉрЃљ:

рЃцрЃарЃЕрЃ«рЃўрЃџрЃћрЃЉрЃў (fahrenheit - 32) рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА, рЃарЃЮрЃЏ рЃњрЃљрЃЏрЃЮрЃЎрЃџрЃћрЃЉрЃљ рЃерЃћрЃАрЃарЃБрЃџрЃЊрЃћрЃА рЃњрЃљрЃЏрЃарЃљрЃЋрЃџрЃћрЃЉрЃљрЃЏрЃЊрЃћ.
рЃњрЃљрЃДрЃЮрЃцрЃљ (/) рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА float-рЃА.

рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ : рЃфрЃћрЃџрЃАрЃўрЃБрЃАрЃўрЃЊрЃљрЃю рЃцрЃљрЃарЃћрЃюрЃ░рЃљрЃўрЃбрЃерЃў рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃюрЃљ

рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃбрЃћрЃЏрЃърЃћрЃарЃљрЃбрЃБрЃарЃљ рЃфрЃћрЃџрЃАрЃўрЃБрЃАрЃўрЃЊрЃљрЃю рЃцрЃљрЃарЃћрЃюрЃ░рЃљрЃўрЃбрЃерЃў рЃцрЃЮрЃарЃЏрЃБрЃџрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ: F = C * 9 / 5 + 32. рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ рЃфрЃЋрЃџрЃљрЃЊрЃў celsius = 25. рЃњрЃљрЃЏрЃЮрЃљрЃЕрЃўрЃюрЃћрЃЌ рЃцрЃљрЃарЃћрЃюрЃ░рЃљрЃўрЃбрЃўрЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ.

рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:

celsius = 25
fahrenheit = celsius * 9 / 5 + 32  # 25 * 9 / 5 + 32 = 45 + 32 = 77
print("рЃцрЃљрЃарЃћрЃюрЃ░рЃљрЃўрЃбрЃў:", fahrenheit)  # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: рЃцрЃљрЃарЃћрЃюрЃ░рЃљрЃўрЃбрЃў: 77.0



рЃњрЃљрЃюрЃЏрЃљрЃарЃбрЃћрЃЉрЃљ:

рЃњрЃљрЃЏрЃарЃљрЃЋрЃџрЃћрЃЉрЃљ рЃЊрЃљ рЃњрЃљрЃДрЃЮрЃцрЃљ (*, /) рЃерЃћрЃАрЃарЃБрЃџрЃЊрЃћрЃЉрЃљ рЃЏрЃљрЃюрЃљрЃЏ, рЃАрЃљрЃюрЃљрЃЏ рЃЏрЃўрЃЏрЃљрЃбрЃћрЃЉрЃљ (+) рЃЏрЃЮрЃ«рЃЊрЃћрЃЉрЃљ.
рЃерЃћрЃЊрЃћрЃњрЃў float рЃбрЃўрЃърЃўрЃАрЃљрЃљ рЃњрЃљрЃДрЃЮрЃцрЃўрЃА рЃњрЃљрЃЏрЃЮ.

рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ : рЃЏрЃћрЃбрЃарЃћрЃЉрЃўрЃЊрЃљрЃю рЃЎрЃўрЃџрЃЮрЃЏрЃћрЃбрЃарЃћрЃЉрЃерЃў рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃюрЃљ

рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃЏрЃљрЃюрЃФрЃўрЃџрЃў рЃЏрЃћрЃбрЃарЃћрЃЉрЃўрЃЊрЃљрЃю рЃЎрЃўрЃџрЃЮрЃЏрЃћрЃбрЃарЃћрЃЉрЃерЃў. рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ рЃфрЃЋрЃџрЃљрЃЊрЃў meters = 2500. рЃцрЃЮрЃарЃЏрЃБрЃџрЃљ: kilometers = meters / 1000. рЃњрЃљрЃЏрЃЮрЃљрЃЕрЃўрЃюрЃћрЃЌ рЃерЃћрЃЊрЃћрЃњрЃў.

рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:

meters = 2500
kilometers = meters / 1000  # 2500 / 1000 = 2.5
print("рЃЎрЃўрЃџрЃЮрЃЏрЃћрЃбрЃарЃћрЃЉрЃў:", kilometers)  # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: рЃЎрЃўрЃџрЃЮрЃЏрЃћрЃбрЃарЃћрЃЉрЃў: 2.5

рЃњрЃљрЃюрЃЏрЃљрЃарЃбрЃћрЃЉрЃљ:

рЃњрЃљрЃДрЃЮрЃцрЃљ (/) рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА float-рЃА.
1 рЃЎрЃўрЃџрЃЮрЃЏрЃћрЃбрЃарЃў = 1000 рЃЏрЃћрЃбрЃарЃў.

рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ 6: рЃЎрЃўрЃџрЃЮрЃЏрЃћрЃбрЃарЃћрЃЉрЃўрЃЊрЃљрЃю рЃЏрЃћрЃбрЃарЃћрЃЉрЃерЃў рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃюрЃљ

рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃЏрЃљрЃюрЃФрЃўрЃџрЃў рЃЎрЃўрЃџрЃЮрЃЏрЃћрЃбрЃарЃћрЃЉрЃўрЃЊрЃљрЃю рЃЏрЃћрЃбрЃарЃћрЃЉрЃерЃў. рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ рЃфрЃЋрЃџрЃљрЃЊрЃў kilometers = 3. рЃцрЃЮрЃарЃЏрЃБрЃџрЃљ: meters = kilometers * 1000. рЃњрЃљрЃЏрЃЮрЃљрЃЕрЃўрЃюрЃћрЃЌ рЃерЃћрЃЊрЃћрЃњрЃў.

рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:

kilometers = 3
meters = kilometers * 1000  # 3 * 1000 = 3000
print("рЃЏрЃћрЃбрЃарЃћрЃЉрЃў:", meters)  # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: рЃЏрЃћрЃбрЃарЃћрЃЉрЃў: 3000.0


рЃњрЃљрЃюрЃЏрЃљрЃарЃбрЃћрЃЉрЃљ:

рЃњрЃљрЃЏрЃарЃљрЃЋрЃџрЃћрЃЉрЃљ рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃА рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА рЃўрЃЏрЃљрЃЋрЃћ рЃбрЃўрЃърЃА, рЃарЃљрЃф рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃА рЃљрЃЦрЃЋрЃА, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃърЃўрЃЌрЃЮрЃюрЃў рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃњрЃљрЃЊрЃљрЃЋрЃўрЃЊрЃћрЃА float-рЃќрЃћ.

рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ 9: рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃюрЃљ (рЃЏрЃћрЃбрЃарЃћрЃЉрЃў рЃцрЃБрЃбрЃћрЃЉрЃерЃў)

рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃЏрЃљрЃюрЃФрЃўрЃџрЃў рЃЏрЃћрЃбрЃарЃћрЃЉрЃўрЃЊрЃљрЃю рЃцрЃБрЃбрЃћрЃЉрЃерЃў. рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ рЃфрЃЋрЃџрЃљрЃЊрЃў meters = 10. рЃцрЃЮрЃарЃЏрЃБрЃџрЃљ: feet = meters * 3.28084. рЃњрЃљрЃЏрЃЮрЃљрЃЕрЃўрЃюрЃћрЃЌ рЃерЃћрЃЊрЃћрЃњрЃў.

рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:

meters = 10
feet = meters * 3.28084  # 10 * 3.28084 = 32.8084
print("рЃцрЃБрЃбрЃћрЃЉрЃў:", feet)  # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: рЃцрЃБрЃбрЃћрЃЉрЃў: 32.8084

рЃњрЃљрЃюрЃЏрЃљрЃарЃбрЃћрЃЉрЃљ:

1 рЃЏрЃћрЃбрЃарЃў РЅѕ 3.28084 рЃцрЃБрЃбрЃў.
рЃерЃћрЃЊрЃћрЃњрЃў float рЃбрЃўрЃърЃўрЃАрЃљрЃљ.

рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ : рЃерЃћрЃЏрЃЮрЃЎрЃџрЃћрЃЉрЃБрЃџрЃў рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃў рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃюрЃўрЃАрЃЌрЃЋрЃўрЃА

рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃфрЃЋрЃџрЃљрЃЊрЃў distance = 500 (рЃЏрЃћрЃбрЃарЃћрЃЉрЃерЃў). рЃњрЃљрЃарЃЊрЃљрЃљрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃўрЃА рЃЎрЃўрЃџрЃЮрЃЏрЃћрЃбрЃарЃћрЃЉрЃерЃў (/= 1000), рЃерЃћрЃЏрЃЊрЃћрЃњ рЃњрЃљрЃќрЃљрЃарЃЊрЃћрЃЌ 2-рЃ»рЃћрЃа (*= 2), рЃЊрЃљ рЃЉрЃЮрЃџрЃЮрЃА рЃерЃћрЃљрЃЏрЃфрЃўрЃарЃћрЃЌ 1-рЃўрЃЌ (-= 1). рЃњрЃљрЃЏрЃЮрЃљрЃЕрЃўрЃюрЃћрЃЌ рЃДрЃЮрЃЋрЃћрЃџрЃў рЃюрЃљрЃЉрЃўрЃ»рЃўрЃА рЃерЃћрЃЊрЃћрЃњрЃў.

рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:

distance = 500
print("рЃАрЃљрЃгрЃДрЃўрЃАрЃў рЃЏрЃљрЃюрЃФрЃўрЃџрЃў (рЃЏрЃћрЃбрЃарЃћрЃЉрЃерЃў):", distance)  # 500
distance /= 1000  # 500 / 1000 = 0.5
print("рЃЎрЃўрЃџрЃЮрЃЏрЃћрЃбрЃарЃћрЃЉрЃерЃў:", distance)  # 0.5
distance *= 2     # 0.5 * 2 = 1.0
print("рЃњрЃљрЃЮрЃарЃЏрЃљрЃњрЃћрЃЉрЃБрЃџрЃў:", distance)  # 1.0
distance -= 1     # 1.0 - 1 = 0.0
print("рЃАрЃљрЃЉрЃЮрЃџрЃЮрЃЮ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ:", distance)  # 0.0

рЃњрЃљрЃюрЃЏрЃљрЃарЃбрЃћрЃЉрЃљ:

рЃерЃћрЃЏрЃЮрЃЎрЃџрЃћрЃЉрЃБрЃџрЃў рЃЮрЃърЃћрЃарЃљрЃбрЃЮрЃарЃћрЃЉрЃў рЃљрЃЏрЃљрЃарЃбрЃўрЃЋрЃћрЃЉрЃА рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃћрЃЉрЃА.

/= рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА float-рЃА.


рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ : рЃЏрЃљрЃарЃЌрЃЎрЃБрЃЌрЃ«рЃћрЃЊрЃўрЃА рЃцрЃљрЃарЃЌрЃЮрЃЉрЃў рЃЊрЃљ рЃърЃћрЃарЃўрЃЏрЃћрЃбрЃарЃў

рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃљрЃџрЃћрЃЌ рЃЏрЃљрЃарЃЌрЃЎрЃБрЃЌрЃ«рЃћрЃЊрЃўрЃА рЃцрЃљрЃарЃЌрЃЮрЃЉрЃў (area = length * width) рЃЊрЃљ рЃърЃћрЃарЃўрЃЏрЃћрЃбрЃарЃў (perimeter = 2 * (length + width)) рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ. рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ length = 6 рЃЊрЃљ width = 4. рЃњрЃљрЃЏрЃЮрЃљрЃЕрЃўрЃюрЃћрЃЌ рЃЮрЃарЃўрЃЋрЃћ рЃерЃћрЃЊрЃћрЃњрЃў.

рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:

length = 6
width = 4
area = length * width  # 6 * 4 = 24
perimeter = 2 * (length + width)  # 2 * (6 + 4) = 2 * 10 = 20
print("рЃЏрЃљрЃарЃЌрЃЎрЃБрЃЌрЃ«рЃћрЃЊрЃўрЃА рЃцрЃљрЃарЃЌрЃЮрЃЉрЃў:", area)  # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: рЃЏрЃљрЃарЃЌрЃЎрЃБрЃЌрЃ«рЃћрЃЊрЃўрЃА рЃцрЃљрЃарЃЌрЃЮрЃЉрЃў: 24
print("рЃЏрЃљрЃарЃЌрЃЎрЃБрЃЌрЃ«рЃћрЃЊрЃўрЃА рЃърЃћрЃарЃўрЃЏрЃћрЃбрЃарЃў:", perimeter)  # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: рЃЏрЃљрЃарЃЌрЃЎрЃБрЃЌрЃ«рЃћрЃЊрЃўрЃА рЃърЃћрЃарЃўрЃЏрЃћрЃбрЃарЃў: 20

рЃњрЃљрЃюрЃЏрЃљрЃарЃбрЃћрЃЉрЃљ:

length рЃЊрЃљ width рЃљрЃарЃўрЃА int рЃбрЃўрЃърЃўрЃА рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃў.
рЃцрЃарЃЕрЃ«рЃўрЃџрЃћрЃЉрЃў (length + width) рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА, рЃарЃЮрЃЏ рЃЏрЃўрЃЏрЃљрЃбрЃћрЃЉрЃљ рЃерЃћрЃАрЃарЃБрЃџрЃЊрЃћрЃА рЃњрЃљрЃЏрЃарЃљрЃЋрЃџрЃћрЃЉрЃљрЃЏрЃЊрЃћ.
рЃерЃћрЃЊрЃћрЃњрЃћрЃЉрЃў int рЃбрЃўрЃърЃўрЃАрЃљрЃљ, рЃарЃљрЃЊрЃњрЃљрЃю рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃЏрЃЌрЃћрЃџрЃў рЃарЃўрЃфрЃ«рЃЋрЃћрЃЉрЃў.

рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ : рЃЮрЃЌрЃ«рЃЎрЃБрЃЌрЃ«рЃћрЃЊрЃўрЃА (рЃЎрЃЋрЃљрЃЊрЃарЃљрЃбрЃўрЃА) рЃцрЃљрЃарЃЌрЃЮрЃЉрЃў рЃЊрЃљ рЃърЃћрЃарЃўрЃЏрЃћрЃбрЃарЃў

рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃљрЃџрЃћрЃЌ рЃЎрЃЋрЃљрЃЊрЃарЃљрЃбрЃўрЃА рЃцрЃљрЃарЃЌрЃЮрЃЉрЃў (area = side * side) рЃЊрЃљ рЃърЃћрЃарЃўрЃЏрЃћрЃбрЃарЃў (perimeter = 4 * side) рЃфрЃЋрЃџрЃљрЃЊрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ. рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ side = 5. рЃњрЃљрЃЏрЃЮрЃљрЃЕрЃўрЃюрЃћрЃЌ рЃЮрЃарЃўрЃЋрЃћ рЃерЃћрЃЊрЃћрЃњрЃў.

рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:

side = 5
area = side * side  # 5 * 5 = 25
perimeter = 4 * side  # 4 * 5 = 20
print("рЃЎрЃЋрЃљрЃЊрЃарЃљрЃбрЃўрЃА рЃцрЃљрЃарЃЌрЃЮрЃЉрЃў:", area)  # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: рЃЎрЃЋрЃљрЃЊрЃарЃљрЃбрЃўрЃА рЃцрЃљрЃарЃЌрЃЮрЃЉрЃў: 25
print("рЃЎрЃЋрЃљрЃЊрЃарЃљрЃбрЃўрЃА рЃърЃћрЃарЃўрЃЏрЃћрЃбрЃарЃў:", perimeter)  # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: рЃЎрЃЋрЃљрЃЊрЃарЃљрЃбрЃўрЃА рЃърЃћрЃарЃўрЃЏрЃћрЃбрЃарЃў: 20

рЃњрЃљрЃюрЃЏрЃљрЃарЃбрЃћрЃЉрЃљ:

side рЃљрЃарЃўрЃА int рЃбрЃўрЃърЃўрЃА рЃфрЃЋрЃџрЃљрЃЊрЃў.
рЃцрЃљрЃарЃЌрЃЮрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃњрЃљрЃЏрЃарЃљрЃЋрЃџрЃћрЃЉрЃљ, рЃ«рЃЮрЃџрЃЮ рЃърЃћрЃарЃўрЃЏрЃћрЃбрЃарЃўрЃАрЃЌрЃЋрЃўрЃА РђЊ рЃњрЃљрЃЏрЃарЃљрЃЋрЃџрЃћрЃЉрЃљ 4-рЃќрЃћ.
рЃерЃћрЃЊрЃћрЃњрЃћрЃЉрЃў int рЃбрЃўрЃърЃўрЃАрЃљрЃљ.

рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ : рЃърЃљрЃарЃљрЃџрЃћрЃџрЃћрЃърЃўрЃърЃћрЃЊрЃўрЃА рЃЏрЃЮрЃфрЃБрЃџрЃЮрЃЉрЃљ

рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃљрЃџрЃћрЃЌ рЃърЃљрЃарЃљрЃџрЃћрЃџрЃћрЃърЃўрЃърЃћрЃЊрЃўрЃА рЃЏрЃЮрЃфрЃБрЃџрЃЮрЃЉрЃљ (volume = length * width * height) рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ. рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ length = 3, width = 4, height = 5. рЃњрЃљрЃЏрЃЮрЃљрЃЕрЃўрЃюрЃћрЃЌ рЃерЃћрЃЊрЃћрЃњрЃў.

рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:

length = 3
width = 4
height = 5
volume = length * width * height  # 3 * 4 * 5 = 60
print("рЃърЃљрЃарЃљрЃџрЃћрЃџрЃћрЃърЃўрЃърЃћрЃЊрЃўрЃА рЃЏрЃЮрЃфрЃБрЃџрЃЮрЃЉрЃљ:", volume)  # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: рЃърЃљрЃарЃљрЃџрЃћрЃџрЃћрЃърЃўрЃърЃћрЃЊрЃўрЃА рЃЏрЃЮрЃфрЃБрЃџрЃЮрЃЉрЃљ: 60

рЃњрЃљрЃюрЃЏрЃљрЃарЃбрЃћрЃЉрЃљ:

рЃДрЃЋрЃћрЃџрЃљ рЃфрЃЋрЃџрЃљрЃЊрЃў (length, width, height) рЃљрЃарЃўрЃА int рЃбрЃўрЃърЃўрЃА.
рЃњрЃљрЃЏрЃарЃљрЃЋрЃџрЃћрЃЉрЃљ рЃАрЃарЃБрЃџрЃЊрЃћрЃЉрЃљ рЃЏрЃљрЃарЃфрЃ«рЃюрЃўрЃЊрЃљрЃю рЃЏрЃљрЃарЃ»рЃЋрЃюрЃўрЃЋ, рЃљрЃЏрЃўрЃбрЃЮрЃЏ рЃцрЃарЃЕрЃ«рЃўрЃџрЃћрЃЉрЃў рЃљрЃарЃљрЃљ рЃАрЃљрЃГрЃўрЃарЃЮ.
рЃерЃћрЃЊрЃћрЃњрЃў int рЃбрЃўрЃърЃўрЃАрЃљрЃљ.

рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ : рЃЎрЃБрЃЉрЃўрЃА рЃЏрЃЮрЃфрЃБрЃџрЃЮрЃЉрЃљ рЃЊрЃљ рЃцрЃљрЃарЃЌрЃЮрЃЉрЃў

рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃљрЃџрЃћрЃЌ рЃЎрЃБрЃЉрЃўрЃА рЃЏрЃЮрЃфрЃБрЃџрЃЮрЃЉрЃљ (volume = side * side * side) рЃЊрЃљ рЃќрЃћрЃЊрЃљрЃърЃўрЃарЃўрЃА рЃцрЃљрЃарЃЌрЃЮрЃЉрЃў (surface_area = 6 * side * side) рЃфрЃЋрЃџрЃљрЃЊрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ. рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ side = 2. рЃњрЃљрЃЏрЃЮрЃљрЃЕрЃўрЃюрЃћрЃЌ рЃЮрЃарЃўрЃЋрЃћ рЃерЃћрЃЊрЃћрЃњрЃў.

рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:

side = 2
volume = side * side * side  # 2 * 2 * 2 = 8
surface_area = 6 * side * side  # 6 * 2 * 2 = 24
print("рЃЎрЃБрЃЉрЃўрЃА рЃЏрЃЮрЃфрЃБрЃџрЃЮрЃЉрЃљ:", volume)  # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: рЃЎрЃБрЃЉрЃўрЃА рЃЏрЃЮрЃфрЃБрЃџрЃЮрЃЉрЃљ: 8
print("рЃЎрЃБрЃЉрЃўрЃА рЃќрЃћрЃЊрЃљрЃърЃўрЃарЃўрЃА рЃцрЃљрЃарЃЌрЃЮрЃЉрЃў:", surface_area)  # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: рЃЎрЃБрЃЉрЃўрЃА рЃќрЃћрЃЊрЃљрЃърЃўрЃарЃўрЃА рЃцрЃљрЃарЃЌрЃЮрЃЉрЃў: 24

рЃњрЃљрЃюрЃЏрЃљрЃарЃбрЃћрЃЉрЃљ:

side рЃљрЃарЃўрЃА int рЃбрЃўрЃърЃўрЃА рЃфрЃЋрЃџрЃљрЃЊрЃў.
рЃЏрЃЮрЃфрЃБрЃџрЃЮрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃАрЃљрЃЏрЃ»рЃћрЃарЃљрЃЊрЃў рЃњрЃљрЃЏрЃарЃљрЃЋрЃџрЃћрЃЉрЃљ, рЃ«рЃЮрЃџрЃЮ рЃќрЃћрЃЊрЃљрЃърЃўрЃарЃўрЃА рЃцрЃљрЃарЃЌрЃЮрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА РђЊ 6 рЃњрЃЋрЃћрЃарЃЊрЃўрЃА рЃцрЃљрЃарЃЌрЃЮрЃЉрЃў.
рЃерЃћрЃЊрЃћрЃњрЃћрЃЉрЃў int рЃбрЃўрЃърЃўрЃАрЃљрЃљ.

рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ : рЃЏрЃљрЃарЃЌрЃЎрЃБрЃЌрЃ«рЃћрЃЊрЃўрЃА рЃцрЃљрЃарЃЌрЃЮрЃЉрЃўрЃА рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃљ

рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃљ: рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃЮрЃарЃў рЃфрЃЋрЃџрЃљрЃЊрЃў length = 10 рЃЊрЃљ width = 5. рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃљрЃџрЃћрЃЌ рЃЏрЃљрЃарЃЌрЃЎрЃБрЃЌрЃ«рЃћрЃЊрЃўрЃА рЃцрЃљрЃарЃЌрЃЮрЃЉрЃў (area = length * width). рЃерЃћрЃЏрЃЊрЃћрЃњ рЃњрЃљрЃќрЃљрЃарЃЊрЃћрЃЌ length 2-рЃўрЃЌ рЃЊрЃљ width 3-рЃўрЃЌ рЃерЃћрЃЏрЃЮрЃЎрЃџрЃћрЃЉрЃБрЃџрЃў рЃЮрЃърЃћрЃарЃљрЃбрЃЮрЃарЃћрЃЉрЃўрЃЌ (+=) рЃЊрЃљ рЃ«рЃћрЃџрЃљрЃ«рЃџрЃљ рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃљрЃџрЃћрЃЌ рЃцрЃљрЃарЃЌрЃЮрЃЉрЃў. рЃњрЃљрЃЏрЃЮрЃљрЃЕрЃўрЃюрЃћрЃЌ рЃЮрЃарЃўрЃЋрЃћ рЃцрЃљрЃарЃЌрЃЮрЃЉрЃў.

рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:

length = 10
width = 5
area = length * width  # 10 * 5 = 50
print("рЃАрЃљрЃгрЃДрЃўрЃАрЃў рЃцрЃљрЃарЃЌрЃЮрЃЉрЃў:", area)  # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: рЃАрЃљрЃгрЃДрЃўрЃАрЃў рЃцрЃљрЃарЃЌрЃЮрЃЉрЃў: 50

length += 2  # 10 + 2 = 12
width += 3   # 5 + 3 = 8
new_area = length * width  # 12 * 8 = 96
print("рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃБрЃџрЃў рЃцрЃљрЃарЃЌрЃЮрЃЉрЃў:", new_area)  # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃБрЃџрЃў рЃцрЃљрЃарЃЌрЃЮрЃЉрЃў: 96


рЃњрЃљрЃюрЃЏрЃљрЃарЃбрЃћрЃЉрЃљ:

+= рЃљрЃЏрЃљрЃарЃбрЃўрЃЋрЃћрЃЉрЃА рЃфрЃЋрЃџрЃљрЃЊрЃўрЃА рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃљрЃА.

рЃЮрЃарЃўрЃЋрЃћ рЃцрЃљрЃарЃЌрЃЮрЃЉрЃў int рЃбрЃўрЃърЃўрЃАрЃљрЃљ, рЃарЃљрЃЊрЃњрЃљрЃю рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ рЃЏрЃЌрЃћрЃџрЃў рЃарЃўрЃфрЃ«рЃЋрЃћрЃЉрЃў.

рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ : рЃърЃћрЃарЃўрЃЏрЃћрЃбрЃарЃўрЃАрЃљ рЃЊрЃљ рЃцрЃљрЃарЃЌрЃЮрЃЉрЃўрЃА рЃерЃћрЃЊрЃљрЃарЃћрЃЉрЃљ

рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃљрЃџрЃћрЃЌ рЃЏрЃљрЃарЃЌрЃЎрЃБрЃЌрЃ«рЃћрЃЊрЃўрЃА рЃцрЃљрЃарЃЌрЃЮрЃЉрЃў (area = length * width) рЃЊрЃљ рЃърЃћрЃарЃўрЃЏрЃћрЃбрЃарЃў (perimeter = 2 * (length + width)) рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃўрЃЌ length = 7 рЃЊрЃљ width = 3. рЃерЃћрЃЏрЃЊрЃћрЃњ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ рЃерЃћрЃЏрЃЮрЃЎрЃџрЃћрЃЉрЃБрЃџрЃў рЃЮрЃърЃћрЃарЃљрЃбрЃЮрЃарЃў, рЃарЃЮрЃЏ рЃњрЃљрЃљрЃЮрЃарЃЏрЃљрЃњрЃЮрЃЌ length (*= 2) рЃЊрЃљ рЃ«рЃћрЃџрЃљрЃ«рЃџрЃљ рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃљрЃџрЃћрЃЌ рЃцрЃљрЃарЃЌрЃЮрЃЉрЃў рЃЊрЃљ рЃърЃћрЃарЃўрЃЏрЃћрЃбрЃарЃў. рЃњрЃљрЃЏрЃЮрЃљрЃЕрЃўрЃюрЃћрЃЌ рЃДрЃЋрЃћрЃџрЃљ рЃерЃћрЃЊрЃћрЃњрЃў.

рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:

length = 7
width = 3
area = length * width  # 7 * 3 = 21
perimeter = 2 * (length + width)  # 2 * (7 + 3) = 20
print("рЃАрЃљрЃгрЃДрЃўрЃАрЃў рЃцрЃљрЃарЃЌрЃЮрЃЉрЃў:", area)  # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: рЃАрЃљрЃгрЃДрЃўрЃАрЃў рЃцрЃљрЃарЃЌрЃЮрЃЉрЃў: 21
print("рЃАрЃљрЃгрЃДрЃўрЃАрЃў рЃърЃћрЃарЃўрЃЏрЃћрЃбрЃарЃў:", perimeter)  # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: рЃАрЃљрЃгрЃДрЃўрЃАрЃў рЃърЃћрЃарЃўрЃЏрЃћрЃбрЃарЃў: 20

length *= 2  # 7 * 2 = 14
new_area = length * width  # 14 * 3 = 42
new_perimeter = 2 * (length + width)  # 2 * (14 + 3) = 34
print("рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃБрЃџрЃў рЃцрЃљрЃарЃЌрЃЮрЃЉрЃў:", new_area)  # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃБрЃџрЃў рЃцрЃљрЃарЃЌрЃЮрЃЉрЃў: 42
print("рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃБрЃџрЃў рЃърЃћрЃарЃўрЃЏрЃћрЃбрЃарЃў:", new_perimeter)  # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃБрЃџрЃў рЃърЃћрЃарЃўрЃЏрЃћрЃбрЃарЃў: 34


рЃњрЃљрЃюрЃЏрЃљрЃарЃбрЃћрЃЉрЃљ:

*= 2 рЃЮрЃарЃЏрЃљрЃњрЃЊрЃћрЃЉрЃљ length-рЃўрЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ.
рЃцрЃарЃЕрЃ«рЃўрЃџрЃћрЃЉрЃў (length + width) рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА рЃАрЃгрЃЮрЃа рЃърЃарЃўрЃЮрЃарЃўрЃбрЃћрЃбрЃА.
рЃерЃћрЃЊрЃћрЃњрЃћрЃЉрЃў int рЃбрЃўрЃърЃўрЃАрЃљрЃљ.

рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ : рЃърЃљрЃарЃљрЃџрЃћрЃџрЃћрЃърЃўрЃърЃћрЃЊрЃўрЃА рЃЏрЃЮрЃфрЃБрЃџрЃЮрЃЉрЃўрЃА рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃљ

рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃљрЃџрЃћрЃЌ рЃърЃљрЃарЃљрЃџрЃћрЃџрЃћрЃърЃўрЃърЃћрЃЊрЃўрЃА рЃЏрЃЮрЃфрЃБрЃџрЃЮрЃЉрЃљ (volume = length * width * height) рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃўрЃЌ length = 2, width = 3, height = 4. рЃерЃћрЃЏрЃЊрЃћрЃњ рЃњрЃљрЃљрЃАрЃљрЃЏрЃЏрЃљрЃњрЃћрЃЌ height рЃерЃћрЃЏрЃЮрЃЎрЃџрЃћрЃЉрЃБрЃџрЃў рЃЮрЃърЃћрЃарЃљрЃбрЃЮрЃарЃўрЃЌ (*= 3) рЃЊрЃљ рЃ«рЃћрЃџрЃљрЃ«рЃџрЃљ рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃљрЃџрЃћрЃЌ рЃЏрЃЮрЃфрЃБрЃџрЃЮрЃЉрЃљ. рЃњрЃљрЃЏрЃЮрЃљрЃЕрЃўрЃюрЃћрЃЌ рЃЮрЃарЃўрЃЋрЃћ рЃЏрЃЮрЃфрЃБрЃџрЃЮрЃЉрЃљ.

рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:

length = 2
width = 3
height = 4
volume = length * width * height  # 2 * 3 * 4 = 24
print("рЃАрЃљрЃгрЃДрЃўрЃАрЃў рЃЏрЃЮрЃфрЃБрЃџрЃЮрЃЉрЃљ:", volume)  # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: рЃАрЃљрЃгрЃДрЃўрЃАрЃў рЃЏрЃЮрЃфрЃБрЃџрЃЮрЃЉрЃљ: 24

height *= 3  # 4 * 3 = 12
new_volume = length * width * height  # 2 * 3 * 12 = 72
print("рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃБрЃџрЃў рЃЏрЃЮрЃфрЃБрЃџрЃЮрЃЉрЃљ:", new_volume)  # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃБрЃџрЃў рЃЏрЃЮрЃфрЃБрЃџрЃЮрЃЉрЃљ: 72



рЃњрЃљрЃюрЃЏрЃљрЃарЃбрЃћрЃЉрЃљ:

*= 3 рЃќрЃарЃЊрЃўрЃА height-рЃА рЃАрЃљрЃЏрЃ»рЃћрЃа.
рЃЏрЃЮрЃфрЃБрЃџрЃЮрЃЉрЃљ рЃарЃЕрЃћрЃЉрЃљ int рЃбрЃўрЃърЃўрЃА, рЃарЃљрЃЊрЃњрЃљрЃю рЃДрЃЋрЃћрЃџрЃљ рЃфрЃЋрЃџрЃљрЃЊрЃў рЃЏрЃЌрЃћрЃџрЃў рЃарЃўрЃфрЃ«рЃЋрЃўрЃљ.

## рЃарЃЕрЃћрЃЋрЃћрЃЉрЃў 

рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃА рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ print() рЃерЃБрЃљрЃџрЃћрЃЊрЃБрЃарЃў рЃерЃћрЃЊрЃћрЃњрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃџрЃљрЃЊ.
рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ рЃљрЃдрЃгрЃћрЃарЃўрЃЌрЃў рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃћрЃЉрЃў (рЃЏрЃљрЃњ.: length, width, area), рЃарЃљрЃЌрЃљ рЃЎрЃЮрЃЊрЃў рЃўрЃДрЃЮрЃА рЃњрЃљрЃАрЃљрЃњрЃћрЃЉрЃў.
рЃЌрЃБ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ рЃњрЃљрЃЦрЃЋрЃЌ, рЃДрЃБрЃарЃљрЃЊрЃдрЃћрЃЉрЃўрЃЌ рЃгрЃљрЃўрЃЎрЃўрЃЌрЃ«рЃћрЃЌ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃўрЃА рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃљ.
рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ рЃцрЃарЃЕрЃ«рЃўрЃџрЃћрЃЉрЃў рЃарЃЌрЃБрЃџрЃў рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃњрЃљрЃАрЃљрЃарЃЎрЃЋрЃћрЃЋрЃљрЃЊ.
рЃДрЃБрЃарЃљрЃЊрЃдрЃћрЃЉрЃљ рЃЏрЃўрЃљрЃЦрЃфрЃўрЃћрЃЌ рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃўрЃА рЃърЃарЃўрЃЮрЃарЃўрЃбрЃћрЃбрЃА; рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ рЃцрЃарЃЕрЃ«рЃўрЃџрЃћрЃЉрЃў, рЃЌрЃБ рЃљрЃа рЃ«рЃљрЃарЃЌ рЃЊрЃљрЃарЃгрЃЏрЃБрЃюрЃћрЃЉрЃБрЃџрЃў рЃЌрЃљрЃюрЃЏрЃўрЃЏрЃЊрЃћрЃЋрЃарЃЮрЃЉрЃљрЃерЃў.
рЃАрЃфрЃљрЃЊрЃћрЃЌ рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃў, рЃарЃљрЃЌрЃљ рЃюрЃљрЃ«рЃЮрЃЌ, рЃарЃЮрЃњрЃЮрЃа рЃўрЃфрЃЋрЃџрЃћрЃЉрЃљ рЃерЃћрЃЊрЃћрЃњрЃў.
рЃЌрЃБ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃЏрЃЌрЃћрЃџрЃў рЃарЃўрЃфрЃ«рЃЋрЃћрЃЉрЃў, рЃерЃћрЃЊрЃћрЃњрЃў рЃўрЃЦрЃюрЃћрЃЉрЃљ int. рЃЌрЃБ рЃњрЃљрЃДрЃЮрЃцрЃљ (/) рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ, рЃерЃћрЃЊрЃћрЃњрЃў float рЃўрЃЦрЃюрЃћрЃЉрЃљ.


## рЃарЃЌрЃБрЃџрЃўрЃљ

рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ : рЃърЃўрЃЌрЃљрЃњрЃЮрЃарЃљрЃА рЃЌрЃћрЃЮрЃарЃћрЃЏрЃљ

рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃЌрЃЋрЃџрЃўрЃА рЃАрЃљрЃЏрЃЎрЃБрЃЌрЃ«рЃћрЃЊрЃўрЃА рЃ░рЃўрЃърЃЮрЃбрЃћрЃюрЃБрЃќрЃўрЃА рЃАрЃўрЃњрЃарЃФрЃћрЃА рЃърЃўрЃЌрЃљрЃњрЃЮрЃарЃљрЃА рЃЌрЃћрЃЮрЃарЃћрЃЏрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ (c = Рѕџ(a┬▓ + b┬▓)). рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ рЃЮрЃарЃў рЃфрЃЋрЃџрЃљрЃЊрЃў a рЃЊрЃљ b (рЃЎрЃљрЃЌрЃћрЃбрЃћрЃЉрЃў) рЃЊрЃљ math.sqrt() рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ рЃцрЃћрЃАрЃЋрЃўрЃА рЃњрЃљрЃЏрЃЮрЃАрЃљрЃЌрЃЋрЃџрЃћрЃџрЃљрЃЊ. рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃћрЃЌ рЃерЃћрЃЊрЃћрЃњрЃў.
рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:import math

a = 3
b = 4
c = math.sqrt(a ** 2 + b ** 2)  # 3┬▓ + 4┬▓ = 9 + 16 = 25, Рѕџ25 = 5
print("рЃ░рЃўрЃърЃЮрЃбрЃћрЃюрЃБрЃќрЃљ:", c)  # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: рЃ░рЃўрЃърЃЮрЃбрЃћрЃюрЃБрЃќрЃљ: 5.0


!!! note "рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃљ:"  
    math.sqrt()-рЃўрЃА рЃњрЃљрЃЏрЃЮрЃАрЃљрЃДрЃћрЃюрЃћрЃЉрЃџрЃљрЃЊ рЃАрЃљрЃГрЃўрЃарЃЮрЃљ import math. <br>
    рЃЊрЃљрЃарЃгрЃЏрЃБрЃюрЃЊрЃўрЃЌ, рЃарЃЮрЃЏ a рЃЊрЃљ b рЃЊрЃљрЃЊрЃћрЃЉрЃўрЃЌрЃў рЃарЃўрЃфрЃ«рЃЋрЃћрЃЉрЃўрЃљ. <br>
    рЃерЃћрЃЊрЃћрЃњрЃў рЃўрЃЦрЃюрЃћрЃЉрЃљ float. <br>



рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ : рЃюрЃљрЃарЃЕрЃћрЃюрЃўрЃА рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃљ

рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃА, рЃљрЃарЃўрЃА рЃЌрЃБ рЃљрЃарЃљ рЃарЃўрЃфрЃ«рЃЋрЃў n 3-рЃўрЃА рЃ»рЃћрЃарЃљрЃЊрЃў, % рЃЮрЃърЃћрЃарЃљрЃбрЃЮрЃарЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ. рЃЌрЃБ рЃюрЃљрЃерЃЌрЃў 0-рЃўрЃљ, рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃћрЃЌ Рђъ3-рЃўрЃА рЃ»рЃћрЃарЃљрЃЊрЃўрЃљРђю.
рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:n = 9
if n % 3 == 0:
    print(n, "рЃљрЃарЃўрЃА 3-рЃўрЃА рЃ»рЃћрЃарЃљрЃЊрЃў")  # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: 9 рЃљрЃарЃўрЃА 3-рЃўрЃА рЃ»рЃћрЃарЃљрЃЊрЃў
else:
    print(n, "рЃљрЃа рЃљрЃарЃўрЃА 3-рЃўрЃА рЃ»рЃћрЃарЃљрЃЊрЃў")


!!! note "рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃљ:" 
    рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ % рЃюрЃљрЃерЃЌрЃўрЃА рЃњрЃљрЃЏрЃЮрЃАрЃљрЃЌрЃЋрЃџрЃћрЃџрЃљрЃЊ. <br>
    рЃАрЃфрЃљрЃЊрЃћрЃЌ рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ n-рЃўрЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃў. <br>


рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ : рЃърЃўрЃЌрЃљрЃњрЃЮрЃарЃљрЃА рЃЌрЃћрЃЮрЃарЃћрЃЏрЃљ (рЃњрЃљрЃцрЃљрЃарЃЌрЃЮрЃћрЃЉрЃБрЃџрЃў)

рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃЌрЃЋрЃџрЃўрЃА рЃАрЃљрЃЏрЃЎрЃБрЃЌрЃ«рЃћрЃЊрЃўрЃА рЃ░рЃўрЃърЃЮрЃбрЃћрЃюрЃБрЃќрЃљрЃА a рЃЊрЃљ b рЃЎрЃљрЃЌрЃћрЃбрЃћрЃЉрЃўрЃЊрЃљрЃю. рЃАрЃЌрЃ«рЃЮрЃЋрЃћрЃЌ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃА, рЃерЃћрЃўрЃДрЃЋрЃљрЃюрЃЮрЃА a рЃЊрЃљ b (input() рЃцрЃБрЃюрЃЦрЃфрЃўрЃўрЃЌ), рЃњрЃљрЃЊрЃљрЃљрЃЦрЃфрЃўрЃћрЃЌ рЃўрЃАрЃўрЃюрЃў float-рЃљрЃЊ, рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃљрЃџрЃћрЃЌ рЃ░рЃўрЃърЃЮрЃбрЃћрЃюрЃБрЃќрЃљ рЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃћрЃЌ рЃерЃћрЃЊрЃћрЃњрЃў рЃЮрЃарЃў рЃљрЃЌрЃЮрЃЉрЃўрЃЌрЃў рЃюрЃўрЃерЃюрЃўрЃА рЃАрЃўрЃќрЃБрЃАрЃбрЃўрЃЌ (:.2f).
рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:import math

a = float(input("рЃерЃћрЃўрЃДрЃЋрЃљрЃюрЃћрЃЌ рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃЎрЃљрЃЌрЃћрЃбрЃў: "))  # рЃЏрЃљрЃњ.: 6
b = float(input("рЃерЃћрЃўрЃДрЃЋрЃљрЃюрЃћрЃЌ рЃЏрЃћрЃЮрЃарЃћ рЃЎрЃљрЃЌрЃћрЃбрЃў: "))   # рЃЏрЃљрЃњ.: 8
c = math.sqrt(a ** 2 + b ** 2)
print(f"рЃ░рЃўрЃърЃЮрЃбрЃћрЃюрЃБрЃќрЃљ: {c:.2f}")  # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: рЃ░рЃўрЃърЃЮрЃбрЃћрЃюрЃБрЃќрЃљ: 10.00


!!! note "рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃљ:" 
    input() рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃА, рЃљрЃЏрЃўрЃбрЃЮрЃЏ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ float() рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃюрЃўрЃАрЃЌрЃЋрЃўрЃА. <br>
    {c:.2f} рЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃарЃћрЃЉрЃљ рЃњрЃљрЃЏрЃЮрЃљрЃЦрЃЋрЃА рЃарЃўрЃфрЃ«рЃЋрЃў рЃЮрЃарЃў рЃљрЃЌрЃЮрЃЉрЃўрЃЌрЃў рЃюрЃўрЃерЃюрЃўрЃЌ. <br>
    рЃЊрЃљрЃарЃгрЃЏрЃБрЃюрЃЊрЃўрЃЌ, рЃарЃЮрЃЏ рЃерЃћрЃДрЃЋрЃљрЃюрЃўрЃџрЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃў рЃЊрЃљрЃЊрЃћрЃЉрЃўрЃЌрЃўрЃљ. <br>


рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ: рЃцрЃљрЃарЃћрЃюрЃ░рЃљрЃўрЃбрЃўрЃЊрЃљрЃю рЃфрЃћрЃџрЃАрЃўрЃБрЃАрЃерЃў рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃюрЃљ

print(f"рЃбрЃћрЃЏрЃърЃћрЃарЃљрЃбрЃБрЃарЃљ рЃфрЃћрЃџрЃАрЃўрЃБрЃАрЃерЃў: {celsius:.2f}")  # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: рЃбрЃћрЃЏрЃърЃћрЃарЃљрЃбрЃБрЃарЃљ рЃфрЃћрЃџрЃАрЃўрЃБрЃАрЃерЃў: 37.00


рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃљ: рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ рЃцрЃарЃЕрЃ«рЃўрЃџрЃћрЃЉрЃў (fahrenheit - 32) рЃърЃарЃўрЃЮрЃарЃўрЃбрЃћрЃбрЃўрЃА рЃњрЃљрЃАрЃљрЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃћрЃЉрЃџрЃљрЃЊ.


рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ : рЃЏрЃљрЃюрЃФрЃўрЃџрЃўрЃА рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃљ

рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃЌрЃЋрЃџрЃўрЃА рЃЮрЃа рЃгрЃћрЃарЃбрЃўрЃџрЃА рЃерЃЮрЃарЃўрЃА рЃЏрЃљрЃюрЃФрЃўрЃџрЃА рЃЎрЃЮрЃЮрЃарЃЊрЃўрЃюрЃљрЃбрЃЌрЃљ рЃАрЃўрЃЉрЃарЃбрЃДрЃћрЃќрЃћ (distance = Рѕџ((x2 - x1)┬▓ + (y2 - y1)┬▓)). рЃАрЃЌрЃ«рЃЮрЃЋрЃћрЃЌ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃА, рЃерЃћрЃўрЃДрЃЋрЃљрЃюрЃЮрЃА x1, y1, x2, y2 рЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃљрЃЕрЃўрЃюрЃћрЃЌ рЃЏрЃљрЃюрЃФрЃўрЃџрЃў :.2f рЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃЌ.
рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:import math

x1 = float(input("рЃерЃћрЃўрЃДрЃЋрЃљрЃюрЃћрЃЌ x1: "))  # рЃЏрЃљрЃњ.: 0
y1 = float(input("рЃерЃћрЃўрЃДрЃЋрЃљрЃюрЃћрЃЌ y1: "))  # рЃЏрЃљрЃњ.: 0
x2 = float(input("рЃерЃћрЃўрЃДрЃЋрЃљрЃюрЃћрЃЌ x2: "))  # рЃЏрЃљрЃњ.: 3
y2 = float(input("рЃерЃћрЃўрЃДрЃЋрЃљрЃюрЃћрЃЌ y2: "))  # рЃЏрЃљрЃњ.: 4
distance = math.sqrt((x2 - x1) ** 2 + (y2 - y1) ** 2)
print(f"рЃЏрЃљрЃюрЃФрЃўрЃџрЃў: {distance:.2f}")  # рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА: рЃЏрЃљрЃюрЃФрЃўрЃџрЃў: 5.00

рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃљ: рЃћрЃА рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ рЃўрЃДрЃћрЃюрЃћрЃЉрЃА рЃърЃўрЃЌрЃљрЃњрЃЮрЃарЃљрЃА рЃЌрЃћрЃЮрЃарЃћрЃЏрЃљрЃА. рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ math.sqrt() рЃЊрЃљ рЃцрЃарЃЕрЃ«рЃўрЃџрЃћрЃЉрЃў рЃАрЃгрЃЮрЃарЃў рЃърЃарЃўрЃЮрЃарЃўрЃбрЃћрЃбрЃўрЃАрЃЌрЃЋрЃўрЃА.
