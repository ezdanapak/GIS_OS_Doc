# ­ЪљЇ VS Code рЃЊрЃљрЃюрЃљрЃЏрЃљрЃбрЃћрЃЉрЃў Python-рЃўрЃАрЃЌрЃЋрЃўрЃА

рЃЦрЃЋрЃћрЃЏрЃЮрЃЌ рЃЏрЃЮрЃфрЃћрЃЏрЃБрЃџрЃўрЃљ рЃўрЃА рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃЊрЃљрЃюрЃљрЃЏрЃљрЃбрЃћрЃЉрЃў, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃЊрЃљ рЃЏрЃћрЃбрЃљрЃЊ рЃАрЃљрЃАрЃљрЃарЃњрЃћрЃЉрЃџрЃЮрЃљ Python-рЃЌрЃљрЃю рЃЏрЃБрЃерЃљрЃЮрЃЉрЃўрЃАрЃљрЃА VS Code рЃњрЃљрЃарЃћрЃЏрЃЮрЃерЃў.

---

## ­Ъћ╣ рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃћрЃџрЃў рЃЊрЃљрЃюрЃљрЃЏрЃљрЃбрЃћрЃЉрЃў

### 1. **Python (by Microsoft)**
- Python рЃЎрЃЮрЃЊрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ рЃЊрЃљ рЃЊрЃћрЃЉрЃљрЃњрЃў.
- рЃўрЃюрЃбрЃћрЃарЃърЃарЃћрЃбрЃљрЃбрЃЮрЃарЃўрЃА рЃљрЃарЃЕрЃћрЃЋрЃљ.
- IntelliSense (рЃЎрЃЮрЃЊрЃўрЃА рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў рЃерЃћрЃЌрЃљрЃЋрЃљрЃќрЃћрЃЉрЃћрЃЉрЃў).

---

### 2. **Pylance**
- рЃАрЃгрЃарЃљрЃцрЃў рЃЊрЃљ рЃФрЃџрЃўрЃћрЃарЃў IntelliSense.
- рЃбрЃўрЃърЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃљ (type checking).
- рЃЎрЃЮрЃЊрЃўрЃА рЃБрЃцрЃарЃЮ рЃћрЃцрЃћрЃЦрЃбрЃБрЃарЃў рЃљрЃюрЃљрЃџрЃўрЃќрЃў.

---

### 3. **Jupyter**
- Notebook (.ipynb) рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃњрЃљрЃ«рЃАрЃюрЃљ рЃЊрЃљ рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ.
- рЃЎрЃЮрЃЊрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ рЃфрЃљрЃџрЃЎрЃћ рЃБрЃ»рЃарЃћрЃЊрЃћрЃЉрЃерЃў.
- рЃАрЃљрЃАрЃљрЃарЃњрЃћрЃЉрЃџрЃЮ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЏрЃћрЃфрЃюрЃўрЃћрЃарЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃљрЃюрЃљрЃџрЃўрЃбрЃўрЃЎрЃўрЃАрЃЌрЃЋрЃўрЃА.

---

## ­Ъћ╣ рЃЊрЃљрЃЏрЃ«рЃЏрЃљрЃарЃћ рЃЊрЃљ рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃћрЃЉрЃБрЃџрЃў рЃЊрЃљрЃюрЃљрЃЏрЃљрЃбрЃћрЃЉрЃў

### 4. **Python Debugger**
- рЃЎрЃЮрЃЊрЃўрЃА рЃћрЃбрЃљрЃърЃЮрЃЉрЃарЃўрЃЋрЃљрЃЊ рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ (step-by-step).
- рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃўрЃА рЃЊрЃљ рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃўрЃА рЃЏрЃБрЃерЃљрЃЮрЃЉрЃўрЃА рЃБрЃЎрЃћрЃЌ рЃњрЃљрЃњрЃћрЃЉрЃљ.

---

### 5. **Code Runner**
- рЃАрЃгрЃарЃљрЃцрЃљрЃЊ рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ рЃърЃљрЃбрЃљрЃарЃљ рЃЎрЃЮрЃЊрЃўрЃА рЃцрЃарЃљрЃњрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃА.
- рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљ рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃћрЃюрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃАрЃљрЃф.

---

### 6. **Jupyter Keymap**
- Notebook-рЃерЃў рЃАрЃгрЃарЃљрЃцрЃў рЃЎрЃџрЃљрЃЋрЃўрЃљрЃбрЃБрЃарЃБрЃџрЃў рЃЏрЃљрЃџрЃАрЃљрЃ«рЃЏрЃЮрЃЉрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ.

---

### 7. **Jupyter Cell Tags**
- Notebook-рЃўрЃА рЃЮрЃарЃњрЃљрЃюрЃўрЃќрЃћрЃЉрЃљ рЃЊрЃљ рЃфрЃљрЃџрЃЎрЃћрЃБрЃџрЃў рЃБрЃ»рЃарЃћрЃЊрЃћрЃЉрЃўрЃА рЃЏрЃЮрЃюрЃўрЃерЃЋрЃюрЃљ.

---

### 8. **autoDocstring**
- рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃўрЃА рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў рЃњрЃћрЃюрЃћрЃарЃљрЃфрЃўрЃљ рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃЊрЃљ рЃЎрЃџрЃљрЃАрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.
- рЃ«рЃћрЃџрЃА рЃБрЃгрЃДрЃЮрЃЉрЃА рЃЎрЃЮрЃЊрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃўрЃарЃћрЃЉрЃБрЃџ рЃЊрЃљ рЃњрЃљрЃАрЃљрЃњрЃћрЃЉ рЃгрЃћрЃарЃљрЃА.

---

### 9. **Black Formatter** рЃљрЃю **autopep8**
- рЃЎрЃЮрЃЊрЃўрЃА рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў рЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃарЃћрЃЉрЃљ PEP8 рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ.
- рЃЎрЃЮрЃЊрЃў рЃ«рЃЊрЃћрЃЉрЃљ рЃћрЃарЃЌрЃўрЃљрЃюрЃў рЃАрЃбрЃўрЃџрЃўрЃА.

---

### 10. **isort**
- `import`-рЃћрЃЉрЃўрЃА рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў рЃЊрЃљрЃџрЃљрЃњрЃћрЃЉрЃљ рЃЊрЃљ рЃАрЃўрЃАрЃБрЃцрЃЌрЃљрЃЋрЃўрЃА рЃерЃћрЃюрЃљрЃарЃЕрЃБрЃюрЃћрЃЉрЃљ.

---

### 11. **Error Lens**
- рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃўрЃА рЃЊрЃљ рЃњрЃљрЃцрЃарЃЌрЃ«рЃўрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃърЃўрЃарЃЊрЃљрЃърЃўрЃа рЃ«рЃљрЃќрЃќрЃћ рЃњрЃљрЃЏрЃЮрЃЕрЃћрЃюрЃљ.
- рЃљрЃЏрЃфрЃўрЃарЃћрЃЉрЃА рЃЎрЃЮрЃЊрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃљрЃЏрЃЊрЃћ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃўрЃА рЃЊрЃљрЃбрЃЮрЃЋрЃћрЃЉрЃўрЃА рЃљрЃџрЃЉрЃљрЃЌрЃЮрЃЉрЃљрЃА.

---

## ­Ъћ╣ рЃАрЃљрЃАрЃБрЃарЃЋрЃћрЃџрЃў рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌрЃў рЃЊрЃљрЃюрЃљрЃЏрЃљрЃбрЃћрЃЉрЃў

- **GitLens** Рєњ Git-рЃўрЃА рЃўрЃюрЃбрЃћрЃњрЃарЃљрЃфрЃўрЃљ VS Code-рЃерЃў.
- **Visual Studio IntelliCode** Рєњ AI-рЃўрЃА рЃЏрЃўрЃћрЃа рЃерЃћрЃЌрЃљрЃЋрЃљрЃќрЃћрЃЉрЃБрЃџрЃў рЃГрЃЎрЃЋрЃўрЃљрЃюрЃў рЃЎрЃЮрЃЊрЃўрЃА рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў рЃЊрЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃљ.
- **Path Intellisense** Рєњ рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃћрЃЉрЃўрЃА рЃњрЃќрЃћрЃЉрЃўрЃА рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў рЃерЃћрЃЌрЃљрЃЋрЃљрЃќрЃћрЃЉрЃљ.
- **Bracket Pair Colorizer 2** Рєњ рЃцрЃарЃЕрЃ«рЃўрЃџрЃћрЃЉрЃўрЃА рЃцрЃћрЃарЃљрЃЊрЃў рЃњрЃљрЃЏрЃЮрЃЎрЃЋрЃћрЃЌрЃљ.
- **Material Icon Theme** Рєњ рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃџрЃљрЃЏрЃљрЃќрЃў рЃљрЃўрЃЎрЃЮрЃюрЃћрЃЉрЃў.
- **Rainbow CSV** Рєњ CSV рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃцрЃћрЃарЃљрЃЊрЃў рЃњрЃљрЃЏрЃЮрЃбрЃљрЃюрЃљ.

---

## ­ЪЊї рЃЊрЃљрЃАрЃЎрЃЋрЃюрЃљ
рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА **рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃЏрЃўрЃюрЃўрЃЏрЃБрЃЏ**:  
- **Python**,  
- **Pylance**,  
- **Jupyter**.  

рЃЊрЃљрЃюрЃљрЃарЃЕрЃћрЃюрЃў рЃЊрЃљрЃюрЃљрЃЏрЃљрЃбрЃћрЃЉрЃў рЃЎрЃљрЃарЃњрЃўрЃљ рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌрЃў рЃЎрЃЮрЃЏрЃцрЃЮрЃарЃбрЃўрЃАрЃљ рЃЊрЃљ рЃЎрЃЮрЃЊрЃўрЃА рЃ«рЃљрЃарЃўрЃАрЃ«рЃўрЃА рЃњрЃљрЃАрЃљрЃБрЃЏрЃ»рЃЮрЃЉрЃћрЃАрЃћрЃЉрЃџрЃљрЃЊ.
