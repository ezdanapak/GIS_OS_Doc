# РџА VS Code Hotkeys РђЊ Python-рЃАрЃЌрЃЋрЃўрЃА (рЃАрЃљрЃЉрЃљрЃќрЃўрЃАрЃЮ, рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃБрЃџрЃў)

рЃћрЃА рЃцрЃљрЃўрЃџрЃў рЃерЃћрЃўрЃфрЃљрЃЋрЃА рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃю shortcut-рЃћрЃЉрЃА Python рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃўрЃА рЃЊрЃарЃЮрЃА VS Code-рЃерЃў, рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃљрЃю рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃБрЃџрЃў рЃАрЃгрЃљрЃЋрЃџрЃўрЃАрЃЌрЃЋрЃўрЃА.

---

## ­ЪДГ рЃцрЃљрЃўрЃџрЃўрЃАрЃљ рЃЊрЃљ рЃЎрЃЮрЃЊрЃўрЃА рЃюрЃљрЃЋрЃўрЃњрЃљрЃфрЃўрЃљ
| Shortcut | рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃљ |
|-----------|------------|
| Ctrl + P | рЃАрЃгрЃарЃљрЃцрЃљрЃЊ рЃњрЃљрЃ«рЃАрЃћрЃюрЃў рЃцрЃљрЃўрЃџрЃў рЃАрЃљрЃ«рЃћрЃџрЃўрЃЌ |
| Ctrl + Shift + O | рЃњрЃљрЃЊрЃљрЃЊрЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃљрЃќрЃћ рЃљрЃю рЃфрЃЋрЃџрЃљрЃЊрЃќрЃћ рЃЏрЃўрЃЏрЃЊрЃўрЃюрЃљрЃарЃћ рЃцрЃљрЃўрЃџрЃерЃў |
| Ctrl + Tab | рЃњрЃљрЃЊрЃљрЃарЃЌрЃћ рЃЉрЃЮрЃџрЃЮ рЃњрЃљрЃ«рЃАрЃюрЃўрЃџ рЃцрЃљрЃўрЃџрЃћрЃЉрЃА рЃерЃЮрЃарЃўрЃА |
| Alt + Рєљ / Рєњ | рЃБрЃЎрЃљрЃю рЃљрЃю рЃгрЃўрЃю рЃњрЃљрЃЊрЃљрЃ«рЃбрЃЮрЃЏрЃљ (рЃюрЃљрЃЋрЃўрЃњрЃљрЃфрЃўрЃљ) |
| Ctrl + G | рЃњрЃљрЃЊрЃљрЃЊрЃў рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџ рЃ«рЃљрЃќрЃќрЃћ |

---

## ­ЪД░ рЃЎрЃЮрЃЊрЃўрЃА рЃарЃћрЃЊрЃљрЃЦрЃбрЃўрЃарЃћрЃЉрЃљ
| Shortcut | рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃљ |
|-----------|------------|
| Alt + РєЉ / РєЊ | рЃњрЃљрЃЊрЃљрЃљрЃЊрЃњрЃўрЃџрЃћ рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃў рЃќрЃћрЃЏрЃЮрЃЌ рЃљрЃю рЃЦрЃЋрЃћрЃЏрЃЮрЃЌ |
| Shift + Alt + РєЊ | рЃЊрЃљрЃљрЃЎрЃЮрЃърЃўрЃарЃћ рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃў рЃЦрЃЋрЃћрЃЏрЃЮрЃЌ |
| Ctrl + / | рЃЎрЃЮрЃЏрЃћрЃюрЃбрЃљрЃарЃў / uncomment |
| Shift + Alt + A | рЃЉрЃџрЃЮрЃЎрЃўрЃА рЃЎрЃЮрЃЏрЃћрЃюрЃбрЃљрЃарЃў ('""" ... """') |
| Ctrl + D | рЃЏрЃЮрЃюрЃўрЃерЃюрЃћ рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃўрЃњрЃўрЃЋрЃћ рЃАрЃўрЃбрЃДрЃЋрЃљ |
| Ctrl + L | рЃЏрЃЮрЃюрЃўрЃерЃюрЃћ рЃЏрЃЌрЃћрЃџрЃў рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃў |
| Ctrl + Shift + K | рЃгрЃљрЃерЃљрЃџрЃћ рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃў |
| Alt + Click | рЃЊрЃљрЃљрЃЏрЃљрЃбрЃћ рЃЏрЃарЃљрЃЋрЃљрЃџрЃЎрЃБрЃарЃАрЃЮрЃарЃўрЃљрЃюрЃў рЃЏрЃЮрЃюрЃўрЃерЃЋрЃюрЃљ |
| Ctrl + Space | рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў рЃерЃћрЃЏрЃЮрЃЌрЃљрЃЋрЃљрЃќрЃћрЃЉрЃљ (intellisense) |

---

## ­ЪљЇ Python рЃАрЃърЃћрЃфрЃўрЃцрЃўрЃЎрЃБрЃарЃў (VS Code Python Extension)
| Shortcut | рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃљ |
|-----------|------------|
| Ctrl + F5 | рЃњрЃљрЃБрЃерЃЋрЃў Python рЃцрЃљрЃўрЃџрЃў Debugger-рЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ |
| F5 | рЃњрЃљрЃБрЃерЃЋрЃў Python Debug рЃарЃћрЃЪрЃўрЃЏрЃерЃў |
| Shift + Enter | рЃњрЃљрЃБрЃерЃЋрЃў рЃЏрЃЮрЃюрЃўрЃерЃюрЃБрЃџрЃў рЃЎрЃЮрЃЊрЃў рЃўрЃюрЃбрЃћрЃњрЃарЃўрЃарЃћрЃЉрЃБрЃџ рЃбрЃћрЃарЃЏрЃўрЃюрЃљрЃџрЃерЃў |
| Ctrl + Shift + Enter | рЃњрЃљрЃБрЃерЃЋрЃў рЃЏрЃЌрЃћрЃџрЃў рЃцрЃљрЃўрЃџрЃў рЃбрЃћрЃарЃЏрЃўрЃюрЃљрЃџрЃерЃў |
| Ctrl + '`' | рЃњрЃљрЃ«рЃАрЃћрЃюрЃў / рЃЊрЃљрЃ«рЃБрЃарЃћ рЃбрЃћрЃарЃЏрЃўрЃюрЃљрЃџрЃў |
| Ctrl + Shift + B | рЃњрЃљрЃБрЃерЃЋрЃў 'build' (custom task) |

---

## ­Ъћј рЃФрЃћрЃЉрЃюрЃљ рЃЊрЃљ рЃЕрЃљрЃюрЃљрЃфрЃЋрЃџрЃћрЃЉрЃљ
| Shortcut | рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃљ |
|-----------|------------|
| Ctrl + F | рЃЏрЃЮрЃФрЃћрЃЉрЃюрЃћ рЃЏрЃўрЃЏрЃЊрЃўрЃюрЃљрЃарЃћ рЃцрЃљрЃўрЃџрЃерЃў |
| Ctrl + H | рЃЕрЃљрЃљрЃюрЃљрЃфрЃЋрЃџрЃћ |
| Ctrl + Shift + F | рЃФрЃћрЃЉрЃюрЃљ рЃЏрЃЌрЃћрЃџ рЃърЃарЃЮрЃћрЃЦрЃбрЃерЃў |
| F3 / Shift + F3 | рЃерЃћрЃЏрЃЊрЃћрЃњрЃў / рЃгрЃўрЃюрЃљ рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃЮрЃЉрЃљ |

---

## ­ЪДЕ рЃњрЃљрЃарЃћрЃЏрЃЮ рЃЊрЃљ рЃњрЃљрЃцрЃљрЃарЃЌрЃЮрЃћрЃЉрЃћрЃЉрЃў
| Shortcut | рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃљ |
|-----------|------------|
| Ctrl + Shift + X | рЃњрЃљрЃ«рЃАрЃћрЃюрЃў Extensions |
| Ctrl + Shift + E | рЃњрЃљрЃ«рЃАрЃћрЃюрЃў Explorer (рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃърЃљрЃюрЃћрЃџрЃў) |
| Ctrl + ` | рЃњрЃљрЃ«рЃАрЃћрЃюрЃў рЃбрЃћрЃарЃЏрЃўрЃюрЃљрЃџрЃў |
| Ctrl + B | рЃЊрЃљрЃЏрЃљрЃџрЃћ / рЃљрЃЕрЃЋрЃћрЃюрЃћ рЃњрЃЋрЃћрЃарЃЊрЃўрЃЌрЃў рЃърЃљрЃюрЃћрЃџрЃў |
| Ctrl + J | рЃЊрЃљрЃЏрЃљрЃџрЃћ / рЃљрЃЕрЃЋрЃћрЃюрЃћ рЃЦрЃЋрЃћрЃЊрЃљ рЃърЃљрЃюрЃћрЃџрЃћрЃЉрЃў (terminal/output) |

---

## ­Ъје рЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃарЃћрЃЉрЃљ рЃЊрЃљ рЃЎрЃЮрЃЊрЃўрЃА рЃАрЃбрЃўрЃџрЃў
| Shortcut | рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃљ |
|-----------|------------|
| Shift + Alt + F | рЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃарЃћрЃЉрЃљ ('black' рЃљрЃю 'autopep8') |
| Ctrl + K, Ctrl + F | рЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃарЃћрЃЉрЃљ рЃЏрЃЮрЃюрЃўрЃерЃюрЃБрЃџрЃў рЃюрЃљрЃгрЃўрЃџрЃўрЃА |
| Alt + Shift + F | рЃўрЃњрЃўрЃЋрЃћ рЃќрЃЮрЃњрЃ»рЃћрЃа OS-рЃќрЃћ |
| Ctrl + K, Ctrl + S | рЃњрЃљрЃ«рЃАрЃћрЃюрЃў рЃДрЃЋрЃћрЃџрЃљ shortcut-рЃўрЃА рЃАрЃўрЃљ |

---

## ­ЪД« рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃўрЃБрЃџрЃЮрЃЉрЃўрЃА рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃў
- Ctrl + / Рђћ рЃАрЃгрЃарЃљрЃцрЃљрЃЊ рЃЊрЃљрЃљрЃЎрЃЮрЃЏрЃћрЃюрЃбрЃљрЃарЃћ рЃЎрЃЮрЃЊрЃў рЃљрЃю рЃбрЃћрЃАрЃбрЃћрЃЉрЃў  
- Alt + РєЉ / РєЊ Рђћ рЃњрЃљрЃЊрЃљрЃљрЃЊрЃњрЃўрЃџрЃћ рЃЎрЃЮрЃЊрЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃљрЃерЃў рЃЮрЃърЃбрЃўрЃЏрЃљрЃџрЃБрЃарЃљрЃЊ  
- Shift + Alt + РєЊ Рђћ рЃњрЃљрЃљрЃЏрЃарЃљрЃЋрЃџрЃћ рЃћрЃарЃЌрЃў рЃ«рЃљрЃќрЃўрЃЌ рЃЏрЃАрЃњрЃљрЃЋрЃАрЃў рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ (рЃЏрЃљрЃњ., print)  
- Ctrl + Shift + Enter Рђћ рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ рЃЏрЃЌрЃџрЃўрЃљрЃюрЃў рЃцрЃљрЃўрЃџрЃўрЃА рЃбрЃћрЃарЃЏрЃўрЃюрЃљрЃџрЃерЃў Python-рЃўрЃАрЃЌрЃЋрЃўрЃА  
- Ctrl + K Рєњ Z Рђћ 'Zen Mode' Рђћ distraction-free рЃЎрЃЮрЃЊрЃўрЃарЃћрЃЉрЃљ Рюе

---

**рЃерЃћрЃюрЃўрЃерЃЋрЃюрЃљ:** рЃћрЃА рЃДрЃЋрЃћрЃџрЃљ shortcut рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃБрЃџ Windows VS Code рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃљрЃерЃў. Mac OS-рЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў Ctrl рЃ«рЃерЃўрЃарЃљрЃЊ рЃБрЃюрЃЊрЃљ рЃерЃћрЃўрЃфрЃЋрЃљрЃџрЃЮрЃА Cmd-рЃўрЃЌ.
