# рЃерЃБрЃљрЃџрЃћрЃЊрЃБрЃарЃў рЃњрЃљрЃЏрЃЮрЃфрЃЊрЃљ (IX рЃЌрЃћрЃЏрЃљрЃЏрЃЊрЃћ)

## рЃњрЃљрЃЏрЃЮрЃфрЃЊрЃўрЃА рЃЏрЃЮрЃфрЃЋрЃљ
рЃњрЃљрЃЏрЃЮрЃфрЃЊрЃљ рЃЏрЃЮрЃўрЃфрЃљрЃЋрЃА IX рЃЌрЃћрЃЏрЃљрЃЏрЃЊрЃћ рЃњрЃљрЃюрЃ«рЃўрЃџрЃБрЃџ рЃЏрЃљрЃАрЃљрЃџрЃљрЃА:

1. Python рЃћрЃюрЃўрЃА рЃўрЃАрЃбрЃЮрЃарЃўрЃљ рЃЊрЃљ рЃњрЃљрЃарЃћрЃЏрЃЮ  
2. рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃў рЃЊрЃљ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃбрЃўрЃърЃћрЃЉрЃў  
3. рЃљрЃарЃўрЃЌрЃЏрЃћрЃбрЃўрЃЎрЃБрЃџрЃў рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃў  
4. рЃџрЃЮрЃњрЃўрЃЎрЃБрЃарЃў рЃЊрЃљ рЃерЃћрЃЊрЃљрЃарЃћрЃЉрЃўрЃА рЃЮрЃърЃћрЃарЃљрЃбрЃЮрЃарЃћрЃЉрЃў  
5. рЃфрЃўрЃЎрЃџрЃћрЃЉрЃў (`for`, `while`, break, continue, else)  
6. рЃЕрЃљрЃерЃћрЃюрЃћрЃЉрЃБрЃџрЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў рЃЊрЃљ `math`, `random` рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃў  
7. рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃЏрЃўрЃћрЃа рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў, default рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў, `lambda`  
8. рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃћрЃЉрЃў рЃЊрЃљ рЃЏрЃљрЃЌрЃќрЃћ рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃћрЃЉрЃў  
9. рЃерЃБрЃљрЃџрЃћрЃЊрЃБрЃарЃў рЃњрЃљрЃЏрЃЮрЃфрЃЊрЃўрЃА рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃљ  

---

## рЃерЃћрЃцрЃљрЃАрЃћрЃЉрЃўрЃА рЃЎрЃарЃўрЃбрЃћрЃарЃўрЃБрЃЏрЃћрЃЉрЃў
- РюЁ **рЃЎрЃЮрЃЊрЃў рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА** РђЊ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ рЃўрЃФрЃџрЃћрЃЋрЃљ рЃАрЃгрЃЮрЃарЃў рЃерЃћрЃЊрЃћрЃњрЃА  
- РюЁ **рЃАрЃўрЃюрЃбрЃљрЃЦрЃАрЃўрЃА рЃАрЃўрЃАрЃгрЃЮрЃарЃћ** РђЊ рЃљрЃа рЃљрЃарЃўрЃА рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃў Python-рЃўрЃА рЃгрЃћрЃАрЃћрЃЉрЃерЃў  
- РюЁ **рЃЎрЃЮрЃЏрЃћрЃюрЃбрЃљрЃарЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ** РђЊ рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃЏрЃљ рЃљрЃ«рЃАрЃюрЃљ, рЃарЃљрЃА рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА рЃЎрЃЮрЃЊрЃў  
- РюЁ **рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃўрЃА рЃАрЃарЃБрЃџрЃў рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃљ** РђЊ рЃДрЃЋрЃћрЃџрЃљ рЃърЃБрЃюрЃЦрЃбрЃў рЃњрЃљрЃЌрЃЋрЃљрЃџрЃўрЃАрЃгрЃўрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ  

---

## рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃњрЃљрЃЋрЃарЃфрЃћрЃџрЃћрЃЉрЃБрЃџрЃў рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃў
- РЮї рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃўрЃА рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃў рЃАрЃљрЃ«рЃћрЃџрЃћрЃЉрЃў (рЃЏрЃљрЃњ.: рЃарЃўрЃфрЃ«рЃЋрЃўрЃЌ рЃЊрЃљрЃгрЃДрЃћрЃЉрЃљ)  
- РЮї рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃћрЃЉрЃўрЃА рЃЊрЃљрЃерЃџрЃўрЃАрЃљрЃА (`split`) рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃў, рЃарЃЮрЃфрЃљ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃў рЃћрЃарЃЌ рЃАрЃўрЃбрЃДрЃЋрЃљрЃА рЃерЃћрЃўрЃДрЃЋрЃљрЃюрЃА  
- РЮї `input()` рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃА рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА `str`, рЃарЃўрЃфрЃ«рЃЋрЃћрЃЉрЃќрЃћ рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃюрЃљ рЃљрЃЋрЃўрЃгрЃДрЃЊрЃћрЃЉрЃљрЃЌ (`int()`, `float()`)  
- РЮї рЃфрЃўрЃЎрЃџрЃћрЃЉрЃерЃў `break`/`continue`-рЃўрЃА рЃБрЃДрЃБрЃарЃљрЃЊрЃдрЃћрЃЉрЃЮрЃЊ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ  

---

## рЃЊрЃљрЃЊрЃћрЃЉрЃўрЃЌрЃў рЃЏрЃ«рЃљрЃарЃћрЃћрЃЉрЃў
- ­ЪњА рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃћрЃЉрЃЏрЃљ рЃБрЃЎрЃЋрЃћ рЃЎрЃљрЃарЃњрЃљрЃЊ рЃљрЃўрЃЌрЃЋрЃўрЃАрЃћрЃА `for` рЃЊрЃљ `while` рЃфрЃўрЃЎрЃџрЃћрЃЉрЃў  
- ­ЪњА рЃЉрЃћрЃЋрЃарЃў рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃў рЃўрЃДрЃћрЃюрЃћрЃЉрЃА `f-string` рЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃарЃћрЃЉрЃљрЃА рЃбрЃћрЃЦрЃАрЃбрЃўрЃА рЃњрЃљрЃЏрЃЮрЃбрЃљрЃюрЃўрЃАрЃљрЃА  
- ­ЪњА `math` рЃЊрЃљ `random` рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃќрЃћ рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃћрЃЉрЃў рЃерЃћрЃАрЃарЃБрЃџрЃЊрЃљ рЃАрЃгрЃЮрЃарЃљрЃЊ  

---

## рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃљрЃфрЃўрЃћрЃЉрЃў
- ­ЪћЂ рЃЏрЃћрЃбрЃў рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃљ **рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃќрЃћ** РђЊ рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃўрЃљрЃюрЃў рЃЊрЃљ рЃЊрЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃБрЃџрЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃўрЃЌ  
- ­ЪћЂ рЃбрЃћрЃЦрЃАрЃбрЃБрЃа рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃќрЃћ **рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃБрЃџрЃў рЃЏрЃћрЃЌрЃЮрЃЊрЃћрЃЉрЃўрЃА** рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃљ  
- ­ЪћЂ рЃЏрЃфрЃўрЃарЃћ рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃњрЃљрЃЎрЃћрЃЌрЃћрЃЉрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃћрЃарЃЌрЃўрЃљрЃюрЃћрЃЉрЃА рЃарЃљрЃЏрЃЊрЃћрЃюрЃўрЃЏрЃћ рЃЌрЃћрЃЏрЃљрЃА (рЃЏрЃљрЃњ.: рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃўрЃА рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ)  

---

## рЃЊрЃљрЃАрЃЎрЃЋрЃюрЃљ
рЃерЃБрЃљрЃџрЃћрЃЊрЃБрЃарЃў рЃњрЃљрЃЏрЃЮрЃфрЃЊрЃўрЃА рЃерЃћрЃЊрЃћрЃњрЃћрЃЉрЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ:
- рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃБрЃЏрЃарЃљрЃЋрЃџрЃћрЃАрЃЮрЃЉрЃљрЃЏ рЃгрЃљрЃарЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌ рЃњрЃљрЃўрЃљрЃарЃљ IX рЃЌрЃћрЃЏрЃљрЃЏрЃЊрЃћ рЃљрЃарЃАрЃћрЃЉрЃБрЃџрЃў рЃАрЃљрЃЎрЃўрЃЌрЃ«рЃћрЃЉрЃў  
- рЃњрЃљрЃарЃЎрЃЋрЃћрЃБрЃџрЃў рЃАрЃўрЃарЃЌрЃБрЃџрЃћрЃћрЃЉрЃў рЃўрЃДрЃЮ рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃћрЃЉрЃќрЃћ рЃЊрЃљ рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃўрЃА рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃќрЃћ  
- рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃўрЃарЃћрЃЉрЃБрЃџрЃўрЃљ рЃЏрЃћрЃбрЃў рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃБрЃџрЃў рЃАрЃљрЃЋрЃљрЃарЃ»рЃўрЃерЃЮрЃћрЃЉрЃў рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃњрЃљрЃЎрЃЋрЃћрЃЌрЃўрЃџрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА
