# Google Earth Elevation Profile

Google Earth-рЃўрЃА **Elevation Profile** рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃњрЃЋрЃљрЃФрЃџрЃћрЃЋрЃА
рЃњрЃљрЃЋрЃўрЃњрЃЮрЃЌ, рЃарЃЮрЃњрЃЮрЃа рЃўрЃфрЃЋрЃџрЃћрЃЉрЃљ рЃарЃћрЃџрЃўрЃћрЃцрЃўрЃА рЃАрЃўрЃЏрЃљрЃдрЃџрЃћ рЃЏрЃЮрЃюрЃўрЃерЃюрЃБрЃџрЃў рЃЏрЃљрЃарЃерЃарЃБрЃбрЃўрЃА рЃљрЃю рЃ«рЃљрЃќрЃўрЃА
рЃњрЃљрЃАрЃгрЃЋрЃарЃўрЃЋ. рЃћрЃА рЃ«рЃћрЃџрЃАрЃљрЃДрЃарЃћрЃџрЃўрЃљ рЃњрЃћрЃЮрЃњрЃарЃљрЃцрЃўрЃўрЃА, рЃбрЃБрЃарЃўрЃќрЃЏрЃўрЃА, рЃћрЃЎрЃЮрЃџрЃЮрЃњрЃўрЃўрЃАрЃљ рЃЊрЃљ
рЃўрЃюрЃцрЃарЃљрЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃБрЃџрЃў рЃЊрЃљрЃњрЃћрЃњрЃЏрЃЋрЃўрЃА рЃАрЃцрЃћрЃарЃЮрЃћрЃЉрЃерЃў.

## рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў

1.  **рЃњрЃарЃљрЃцрЃўрЃЎрЃў (Profile Chart)**
    -   рЃ░рЃЮрЃарЃўрЃќрЃЮрЃюрЃбрЃљрЃџрЃБрЃа рЃдрЃћрЃарЃФрЃќрЃћ рЃЕрЃљрЃюрЃА рЃЏрЃљрЃарЃерЃарЃБрЃбрЃўрЃА рЃАрЃўрЃњрЃарЃФрЃћ (рЃЏрЃћрЃбрЃарЃћрЃЉрЃерЃў рЃљрЃю
        рЃЎрЃўрЃџрЃЮрЃЏрЃћрЃбрЃарЃћрЃЉрЃерЃў).
    -   рЃЋрЃћрЃарЃбрЃўрЃЎрЃљрЃџрЃБрЃа рЃдрЃћрЃарЃФрЃќрЃћ рЃгрЃљрЃарЃЏрЃЮрЃЊрЃњрЃћрЃюрЃўрЃџрЃўрЃљ рЃАрЃўрЃЏрЃљрЃдрЃџрЃћ рЃќрЃдрЃЋрЃўрЃА рЃЊрЃЮрЃюрЃўрЃЊрЃљрЃю (рЃЏрЃћрЃбрЃарЃћрЃЉрЃерЃў
        рЃљрЃю рЃцрЃБрЃбрЃћрЃЉрЃерЃў).
2.  **рЃАрЃўрЃЏрЃљрЃдрЃџрЃўрЃА рЃфрЃЋрЃљрЃџрЃћрЃЉрЃљрЃЊрЃЮрЃЉрЃљ**
    -   рЃњрЃарЃљрЃцрЃўрЃЎрЃў рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА, рЃАрЃљрЃЊ рЃљрЃарЃўрЃА рЃљрЃдрЃЏрЃљрЃарЃЌрЃў рЃЊрЃљ рЃАрЃљрЃЊ -- рЃЊрЃљрЃдрЃЏрЃљрЃарЃЌрЃў.
    -   рЃ«рЃљрЃќрЃќрЃћ рЃЏрЃўрЃњрЃарЃљрЃфрЃўрЃўрЃАрЃљрЃА рЃ«рЃћрЃЊрЃљрЃЋрЃЌ рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџ рЃгрЃћрЃарЃбрЃўрЃџрЃўрЃА рЃАрЃўрЃЏрЃљрЃдрЃџрЃћрЃАрЃљ рЃЊрЃљ
        рЃЏрЃљрЃюрЃФрЃўрЃџрЃА.
3.  **рЃЏрЃљрЃЦрЃАрЃўрЃЏрЃљрЃџрЃБрЃарЃў рЃЊрЃљ рЃЏрЃўрЃюрЃўрЃЏрЃљрЃџрЃБрЃарЃў рЃАрЃўрЃЏрЃљрЃдрЃџрЃћ**
    -   рЃърЃарЃЮрЃцрЃўрЃџрЃў рЃњрЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃЌ рЃЏрЃЌрЃћрЃџ рЃ«рЃљрЃќрЃќрЃћ рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃЏрЃљрЃдрЃљрЃџ рЃЊрЃљ рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃЊрЃљрЃЉрЃљрЃџ
        рЃгрЃћрЃарЃбрЃўрЃџрЃА.
4.  **рЃЏрЃЌрЃџрЃўрЃљрЃюрЃў рЃАрЃўрЃњрЃарЃФрЃћ рЃЊрЃљ рЃцрЃћрЃарЃЊрЃЮрЃЉрЃўрЃА рЃЎрЃБрЃЌрЃ«рЃћ**
    -   рЃњрЃарЃљрЃцрЃўрЃЎрЃў рЃўрЃЌрЃЋрЃџрЃўрЃА рЃЏрЃљрЃарЃерЃарЃБрЃбрЃўрЃА рЃАрЃљрЃћрЃарЃЌрЃЮ рЃАрЃўрЃњрЃарЃФрЃћрЃА.
    -   рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃцрЃћрЃарЃЊрЃЮрЃЉрЃўрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃЮ рЃЎрЃБрЃЌрЃ«рЃўрЃА рЃюрЃљрЃ«рЃЋрЃљрЃф, рЃарЃљрЃф рЃњрЃљрЃюрЃАрЃљрЃЎрЃБрЃЌрЃарЃћрЃЉрЃўрЃЌ
        рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃўрЃљ рЃџрЃљрЃерЃЦрЃарЃЮрЃЉрЃўрЃА рЃљрЃю рЃњрЃќрЃћрЃЉрЃўрЃА рЃЊрЃљрЃњрЃћрЃњрЃЏрЃЋрЃўрЃАрЃљрЃА.
5.  **рЃўрЃюрЃбрЃћрЃарЃљрЃЦрЃбрЃўрЃБрЃџрЃЮрЃЉрЃљ**
    -   рЃЏрЃљрЃБрЃАрЃўрЃА рЃњрЃљрЃЊрЃљрЃбрЃљрЃюрЃўрЃАрЃљрЃА рЃњрЃарЃљрЃцрЃўрЃЎрЃќрЃћ, рЃарЃБрЃЎрЃљрЃќрЃћ рЃќрЃБрЃАрЃбрЃљрЃЊ рЃњрЃљрЃЏрЃЮрЃЕрЃюрЃЊрЃћрЃЉрЃљ рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃў
        рЃгрЃћрЃарЃбрЃўрЃџрЃў.
    -   рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃЏрЃљрЃарЃерЃарЃБрЃбрЃўрЃА рЃарЃЌрЃБрЃџрЃў рЃЏрЃЮрЃюрЃљрЃЎрЃЋрЃћрЃЌрЃћрЃЉрЃўрЃА рЃЊрЃћрЃбрЃљрЃџрЃБрЃарЃў рЃљрЃюрЃљрЃџрЃўрЃќрЃў.

## рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃАрЃцрЃћрЃарЃЮрЃћрЃЉрЃў

-   **рЃџрЃљрЃерЃЦрЃарЃЮрЃЉрЃљ рЃЊрЃљ рЃбрЃБрЃарЃўрЃќрЃЏрЃў** -- рЃњрЃќрЃўрЃА рЃАрЃўрЃарЃЌрЃБрЃџрЃўрЃА рЃгрЃўрЃюрЃљрЃАрЃгрЃљрЃа рЃерЃћрЃцрЃљрЃАрЃћрЃЉрЃљ.
-   **рЃўрЃюрЃцрЃарЃљрЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ** -- рЃњрЃќрЃћрЃЉрЃўрЃА, рЃАрЃљрЃарЃЎрЃўрЃюрЃўрЃњрЃќрЃЮ рЃ«рЃљрЃќрЃћрЃЉрЃўрЃА рЃљрЃю рЃЏрЃўрЃџрЃАрЃљрЃЊрЃћрЃюрЃћрЃЉрЃўрЃА
    рЃЊрЃљрЃњрЃћрЃњрЃЏрЃЋрЃљ.
-   **рЃњрЃћрЃЮрЃњрЃарЃљрЃцрЃўрЃљ рЃЊрЃљ рЃћрЃЎрЃЮрЃџрЃЮрЃњрЃўрЃљ** -- рЃарЃћрЃџрЃўрЃћрЃцрЃўрЃА рЃљрЃюрЃљрЃџрЃўрЃќрЃў рЃЊрЃљ рЃЎрЃЋрЃџрЃћрЃЋрЃћрЃЉрЃў.
-   **рЃАрЃърЃЮрЃарЃбрЃў** -- рЃЋрЃћрЃџрЃЮрЃАрЃўрЃърЃћрЃЊрЃўрЃАрЃбрЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃЏрЃЮрЃарЃЉрЃћрЃюрЃљрЃџрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃЏрЃљрЃарЃерЃарЃБрЃбрЃўрЃА
    рЃЊрЃљрЃњрЃћрЃњрЃЏрЃЋрЃљ.

------------------------------------------------------------------------

­ЪЉЅ Elevation Profile рЃљрЃарЃўрЃА рЃАрЃљрЃАрЃљрЃарЃњрЃћрЃЉрЃџрЃЮ рЃўрЃюрЃАрЃбрЃарЃБрЃЏрЃћрЃюрЃбрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃарЃЌрЃБрЃџ рЃарЃћрЃџрЃўрЃћрЃцрЃА
рЃБрЃцрЃарЃЮ рЃњрЃљрЃАрЃљрЃњрЃћрЃЉрЃА рЃЊрЃљ рЃЋрЃўрЃќрЃБрЃљрЃџрЃБрЃарЃљрЃЊ рЃ«рЃћрЃџрЃЏрЃўрЃАрЃљрЃгрЃЋрЃЊрЃЮрЃЏрЃА рЃ«рЃЊрЃўрЃА.
