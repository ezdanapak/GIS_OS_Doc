    მონიშვნის ოპერატორები¶

    'OPERATOR':0

    0 — =

    1 — ≠

    2 — >

    3 — >=

    4 — <

    5 — <=

    6 — begins with

    7 — contains

    8 — is null

    9 — is not null

    10 — does not contain

    ინტერაქტიული მონიშვნის მეთოდები¶

    'METHOD':0

    0 — creating new selection

    1 — adding to current selection

    2 — removing from current selection

    3 — selecting within current selection
